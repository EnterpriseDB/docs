---
title: "EDB Postgres for Kubernetes"
---

**EDB Postgres for Kubernetes** is a stack of enterprise grade products
designed, developed, and supported by EDB to fully exploit Postgres workloads
in Kubernetes.

It currently comprises:

- [EDB Cloud Native Postgres](../edb_cnp/): an operator that is based on
  CloudNativePG for Postgres and EDB Postgres Advanced server workloads with a
  primary/standby architecture, also available on Red Hat OpenShift.
