This release of EDB Postgres for Kubernetes includes the following:

|      Type      |                                                                                  Description                                                                                   |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Upstream merge | Merged with community CloudNativePG 1.19.0. See the community [Release Notes](https://cloudnative-pg.io/documentation/1.19/release_notes/v1.19/).                                    |
| Feature | New PostgreSQL 15 image containing [PostgreSQL 15.1](https://www.postgresql.org/docs/15/release-15.html).  |
| Feature  | Support for [Transparent Data Encryption (TDE)](/tde/latest). TDE encrypts, transparently to the user, any user data stored in the database system. |
| Feature | New external backup adaptor to provide a generic way to integrate EDB Postgres for Kubernetes in a third-party tool for backups. See [External Backup Adapter](/postgres_for_kubernetes/latest/addons/#external_backup_adapter) for more information.|
