---
title: "Supported platforms"
---

MongoDB Foreign Data Wrapper is supported on the same platforms as EDB Postgres Advanced Server. To determine the platform support for the MongoDB Foreign Data Wrapper, see the [Platform Compatibility page](https://www.enterprisedb.com/platform-compatibility#epas) on the EDB website or [Installing MongoDB Foreign Data Wrapper](./installing).
