---
title: "Limitations"
---

The following limitations apply to TDS Foreign Data Wrapper:

- TDS Foreign Data Wrapper doesn't currently support JOIN push-down.
- TDS Foreign Data Wrapper doesn't currently support write operations (INSERT, UPDATE, DELETE).
- WHERE clause and column push downs require `match_column_names` to be enabled on the foreign table. Without it, push downs don't work.
- The `row_estimate_method` option `showplan_all` is supported only with Microsoft SQL Server. It isn't available for Sybase databases.
