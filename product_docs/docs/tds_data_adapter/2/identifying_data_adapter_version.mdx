---
title: "Identifying the version"
---

<div id="identifying_data_adapter_version" class="registered_link"></div>

The TDS Foreign Data Wrapper allows you to identify the currently installed version of the extension by querying a system catalog. 
To check the version, connect to your Postgres server and enter:

```sql
SELECT extversion FROM pg_extension WHERE extname = 'tds_fdw';
__OUTPUT__
 extversion
------------
 2.0.5
(1 row)
```
