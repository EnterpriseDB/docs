The TDS Foreign Data Wrapper (`tds_fdw`) lets you connect Postgres to databases that use the [Tabular Data Stream (TDS) protocol](https://en.wikipedia.org/wiki/Tabular_Data_Stream), such as Sybase databases and Microsoft SQL Server. It uses the [FreeTDS](https://www.freetds.org) library to implement the DB-Library interface.

Install the TDS Foreign Data Wrapper with an RPM package. You can download an installer from the [EDB website](https://www.enterprisedb.com/software-downloads-postgres/).
