The EDB Postgres Backup and Recovery Installation and Upgrade Guide is a comprehensive guide to install and upgrade EDB Postgres Backup and Recovery. This guide provides detailed information about BART installation:

-   The [Requirements Overview](#requirements_overview) section provides information about BART installation requirements and limitations.
-   The [Using an RPM Package to Install BART](#installing_bart_using_an_rpm_package) section provides instructions for installing and upgrading BART.
-   The [Configuring BART](#configuration) section provides instructions for configuring BART and the database servers.
-   The [Troubleshooting](#troubleshooting) section provides information about installation troubleshooting.
-   The [Performing a BART Upgrade](#upgrade) section provides information about upgrading BART.
-   The [Uninstallation](#uninstallation) section provides information about uninstalling BART.
