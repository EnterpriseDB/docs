title: "SHOW-SERVERS"
---

The `SHOW-SERVERS` subcommand displays information for the managed database servers listed in the BART configuration file.

**Syntax:**

```text
bart SHOW-SERVERS [ â€“s { <server_name> | all } ]
```

The following table describes the `SHOW-SERVERS` option:

<table><thead><tr class="header"><th>Option</th><th>Description</th></tr></thead><tbody><tr class="odd"><td><p><code>-s &#123; &lt;server_name&gt; | all</code> &#125;<br />
<code>--server &#123; &lt;server_name&gt; | all</code> &#125;</p></td><td><p><code>&lt;server_name&gt;</code> is the name of the database server to which the <code>SHOW-SERVERS</code> actions are to be applied.<br />
If <code>all</code> is specified or if the <code>-s</code> option is omitted, the actions are applied to all database servers.</p></td></tr></tbody></table>
