The *EDB Postgres Backup and Recovery (BART) Installation and Upgrade Guide* is a comprehensive guide that provides detailed information about how to install and upgrade BART. This guide is organized as follows:

-   [Requirements Overview](#requirements_overview) section provides information about BART installation requirements and limitations.
-   [Using an RPM Package to Install BART](#installing_bart_using_an_rpm_package) section provides instructions for installing and upgrading BART.
-   [Configuring BART](#configuration) section provides instructions for configuring BART and the database servers.
-   [Troubleshooting](#troubleshooting) section provides information about installation troubleshooting.
-   [Performing a BART Upgrade](#upgrade) section provides information about upgrading BART.
-   [Uninstallation](#uninstallation) section provides information about uninstalling BART.
