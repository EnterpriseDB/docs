---
title: "Upgrading Slony Replication"
---

<div id="upgrade" class="registered_link"></div>

The following section outlines the process of upgrading Slony Replication.

Assume identity of the root user and invoke the following command:

```text
yum upgrade edb-as<xx>-slony-replication*
```

where &lt;xx> is the EDB Postgres Advanced Server version.

For example, to upgrade Slony Replication for EDB Postgres Advanced Server version 9.6, you need to execute the command:

```text
yum upgrade epas96-replication
```
