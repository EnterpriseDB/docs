Hybrid Manager (HM) supports external identity providers (IdPs) — such as LDAP, SAML, and OIDC — for authenticating users in the HM console.

- See [Configuring your IdP in the HM console](config_idp_hm) to set up and manage your IdP integrations.
- See [Configuring SAML with Azure Active Directory](saml_azure_ad) for a step-by-step example using Azure Active Directory as your SAML identity provider.
- See [Rotating the identity provider Fernet key](rotate_dex_fernet_key) to rotate the encryption key that protects your IdP connector secrets.
