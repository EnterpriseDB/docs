You can rename and delete projects.

### Renaming a project

You can rename a project in the single-pane-of-glass view of projects and in the **Projects** tab. From either list of projects:

1.  From the ellipses menu to the right of the project name, select **Edit**. 
1.  Enter the new project name and select **Update Project**.

### Deleting a project

You can delete a project in the single-pane-of-glass view of projects and in the **Projects** tab.

!!! Note
You can't delete the default project. Hybrid Manager keeps one default project per organization and protects it from deletion.
!!!

A project must have no active resources before you can delete it. Active resources include clusters, data catalogs, marketplace apps, deployed flows, knowledge bases, and pipelines, depending on the capabilities installed in your environment. If the project still contains any of these resources, Hybrid Manager blocks the deletion and lists the resources you must remove first.

You can delete clusters from within the delete-project dialog. Remove any other resource type from its own management page before you start.

From either list of projects:

1.  From the ellipsis menu to the right of the project name, select **Delete**.

    You're prompted to confirm the deletion.

1.  If the project contains clusters, you must delete them before you can delete the project. The display shows the remaining clusters with a check box next to each one. Select the check box for each cluster you want to delete, or select **Select All**. 

1.  After you select the clusters to delete, enter `delete clusters` in the text box to confirm the deletion. Entering this text enables the **Delete All Clusters** button.

1.  Select **Delete All Clusters** to permanently delete the selected clusters.

1.  When the project contains no active resources, a dialog box appears listing the project name with a check box. Select the check box and enter `delete project` in the text field. Entering this text enables the **Delete Project** button. 

1.  Select **Delete Project** to delete the project.
