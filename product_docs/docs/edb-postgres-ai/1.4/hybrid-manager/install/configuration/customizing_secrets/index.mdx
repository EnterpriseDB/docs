By default, Hybrid Manager (HM) includes pre-configured secrets to facilitate a quick, out-of-the-box installation. However, these default secrets are not intended for production use. For any live environment, you must create custom secrets to ensure your installation remains secure and unique.

HM secrets are categorized by installation scenarios. Depending on which installation scenarios you have enabled, you must configure the corresponding secrets below.

| Scenario    | Feature             | Key secrets                            | Docs                                                     |
|-------------|---------------------|----------------------------------------|----------------------------------------------------------|
| `ai`        | GenAI Builder       | Fernet key, NVIDIA NGC credentials     | [GenAI Builder secrets](genai_secrets)                   |
| `analytics` | Lakehouse / Catalog | Confounding key                        | [Lakehouse / Catalog secrets](lakehouse_catalog_secrets) |
| `migration` | Migration Portal    | DB accounts, Copilot HTTP/Metrics auth | [Migration Portal secrets](migration_portal_secrets)     |
