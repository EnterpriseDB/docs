The EDB Postgres AI agent (`beacon-agent`) and the DMS agent (`cdcagent`) can run on the Postgres database server itself or on a separate auxiliary machine or bastion host. In most production environments, running the agents on a dedicated auxiliary machine is preferred — it avoids installing additional software on the database server, and a single agent instance can connect to databases on multiple servers.

## Required software

Ensure the following are installed on your auxiliary machine before running the agents:

- **Java 17 or higher** — required by the DMS agent (`cdcagent`). Verify with `java -version`.

- **EDB Postgres AI agent** (`beacon-agent`) — required for database registration, schema assessment, and enabling schema migration within the HM landscape. [Install the EDB Postgres AI agent](/tools/agent/installing/).

- **DMS agent** (`cdcagent`) — handles data migration and change data capture (CDC). [Install the DMS agent](/edb-postgres-ai/current/hybrid-manager/using_hybrid_manager/migration/installing_agent/).

!!!note
Unlike Oracle-sourced migrations, no additional client libraries are needed on the auxiliary machine for Postgres sources. The agents connect directly to the Postgres source over the standard Postgres port.
!!!

## Verifying connectivity

Ensure network connectivity from your auxiliary machine to both the Postgres source and Hybrid Manager. The following ports must be open:

| Destination | Port | Used by |
|-------------|------|---------|
| Postgres host | 5432 | Both agents (database connection) |
| HM `beacon_server` | 9443 | EDB Postgres AI agent |
| HM DMS service (`RW_SERVICE_HOST`) | 443 (HTTPS) | DMS agent |

Verify connectivity to each host and port with `nc`:

```shell
# Postgres source
nc -zv <postgres-host-name-or-ip> 5432

# HM beacon server
nc -zv <beacon-server-hostname> 9443

# HM DMS service
nc -zv <rw-service-hostname> 443
```

A successful connection returns `succeeded!`. If you see `Connection refused` or `Operation timed out`, the port isn't reachable and must be opened before proceeding.
