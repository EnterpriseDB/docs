## Technical requirements

- pgAdmin4: 512Mi Memory, 500m CPU (Limits: 1Gi / 1000m).
- Metabase: 1Gi Memory, 800m CPU (Limits: 2Gi / 1000m), Java 21.
- Superset/Airflow: 1Gi Memory, 800m CPU (Limits: 2Gi / 1000m).

## Configuration requirements

- **External database**: Both Metabase and Superset require an existing PostgreSQL instance. Have the host, port, database name, and credentials ready before installation.
- **Security keys**: Airflow and Superset require a Fernet Key and SecretKey to secure your installation. Generate these using `openssl` before deploying.
- **Namespace requirements**: Apps are designed to run within specific Kubernetes namespaces (e.g., `default` or `upm-beacon`).
