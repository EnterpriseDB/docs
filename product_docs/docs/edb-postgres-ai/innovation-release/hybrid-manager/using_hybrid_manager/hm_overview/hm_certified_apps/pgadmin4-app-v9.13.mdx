pgAdmin 4 is an open-source web-based interface for administering PostgreSQL databases. When deployed as a certified app in Hybrid Manager, it provides remote access to your PostgreSQL clusters for managing database objects and executing queries.

!!! warning
pgAdmin 4 is not supported on Red Hat OpenShift (RHOS).
!!!

Key features:

- Log parsing with parallel processing support
- HTML reports with query analysis, connection and session statistics, lock and checkpoint analysis, and autovacuum monitoring
- Temporary file tracking
- Supported log formats: `stderr`, `csvlog`, `syslog`

## Requirements

### Technical requirements:
 - Kubernetes or OpenShift cluster.
 - Persistence: Default 1Gi volume to store reports and uploaded logs.

### Configuration requirements:
 - Retention: reportRetentionDays (default 30) controls how long reports are kept.
 - Upload Limits: maxUploadSize (default 50MB) limits the size of log files you can analyze.
 - Supported Data Sources: PostgreSQL log files (text format).

## Installation steps

1. Navigate to **Asset Library** then select **Apps** in the HM console.
1. Select **pgAdmin 4** and enter required values in the fields.

!!! note
Once you have deployed Superset in HM, you can access it in the HM console under **Apps** or **Asset library**.
!!!
