The EDB Hybrid Manager (HM) **Chat Agent** is a conversational assistant that helps you operate HM in plain language. Ask it about your estate, provision and inspect Postgres clusters, look up active alerts, troubleshoot performance, and consult the HM documentation — all without leaving the HM console.

## Supported operations

Chat Agent covers estate browsing, cluster lifecycle management, monitoring and alerts, configuration recommendations, cluster health reports, Oracle-to-Postgres SQL translation, Data Migration Service status, and documentation Q&A — all scoped to your HM permissions.

For the complete catalog with example prompts and scope boundaries, see [What the Chat Agent can do](./capabilities).

## Model-dependent capabilities

Chat Agent's effective capability scales with the strength of the model your administrator configures. A small, low-cost model handles estate browsing and metric lookups, while more complex cluster-creation flows and structured follow-up questions need a stronger model. See [Model strength tiers](configure/#model-strength-tiers) for guidance on model selection.

## Documentation scope

Chat Agent answers HM questions from a versioned documentation snapshot shipped with each release. For details on citation behavior and the knowledge cutoff, see [HM documentation Q&A](./capabilities#hm-documentation-qa).
