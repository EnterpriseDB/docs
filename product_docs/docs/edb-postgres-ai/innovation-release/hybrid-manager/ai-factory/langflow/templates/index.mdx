Langflow in Hybrid Manager ships with pre-built flow templates for common workflows. Templates give you a working starting point — open one from the template gallery, configure a few inputs, and run it in the Playground.

To access templates, open Langflow from **Launchpad** in the HM console, select **New Flow**, and then select **All Templates**.

| Template                                         | What it does                                                                                                                        |
|--------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [Simple Migration Agent](simple-migration-agent) | Automates database migration into Hybrid Manager: creates a destination cluster, migrates the schema, and kicks off data migration. |
