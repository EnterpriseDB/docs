The **EDB** folder in Langflow contains components that connect flows to resources managed by Hybrid Manager (HM): knowledge bases, databases, vector indexes, embedding and model server clusters, embedded models, and MCP tooling.

| Component                              | Connects flow to                              | Required roles                   |
|----------------------------------------|-----------------------------------------------|----------------------------------|
| [EDB Knowledge Base](knowledge-base)   | An AIDB knowledge base in an HM cluster       | AI Model Manager, Project Viewer |
| [EDB Database](database)               | A Postgres database in an HM cluster          | AI Model Manager, Project Viewer |
| [EDB Vector Index](vector-index)       | Vector indexes on Postgres tables (pgvector)  | AI Model Manager, Project Viewer |
| [EDB Embeddings](embeddings)           | An HM-hosted embedding model                  | AI Model Manager                 |
| [EDB Model Server](model-server)       | An HM-hosted inference model (LLM or other)   | AI Model Manager                 |
| [EDB Embedded Models](embedded-models) | AIDB embedded models for CPU-based inference  | AI Model Manager, Project Viewer |
| [EDB Airman MCP](airman-mcp)           | The Airman MCP server's tools                 | AI Model Manager, Project Viewer |
| [EDB Platform](platform)               | HM platform MCP tools for the current user    | AI Model Manager                 |

See the parent [Langflow](../) page for shared prerequisites (access key, global variables, HM permissions).
