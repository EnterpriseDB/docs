The **EDB** folder in Langflow contains components that connect flows to resources managed by Hybrid Manager (HM): knowledge bases, databases, embedding and model server clusters, and MCP servers.

| Component                            | Connects flow to                                   | Required role    |
|--------------------------------------|----------------------------------------------------|------------------|
| [EDB Knowledge Base](knowledge-base) | An AIDB knowledge base in an HM cluster            | Project Viewer   |
| [EDB Database](database)             | A Postgres database in an HM cluster               | Project Viewer   |
| [EDB MCP Server](mcp-server)         | An MCP server backed by the EDB Database component | AI Model Manager |
| [EDB Embeddings](embeddings)         | An HM-hosted embedding model                       | AI Model Manager |
| [EDB Model Server](model-server)     | An HM-hosted inference model (LLM or other)        | AI Model Manager |

See the parent [Langflow](../) page for shared prerequisites (access key, global variables, HM permissions).
