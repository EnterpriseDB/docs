Perform analytics management for your Advanced High Availability (AHA) and Distributed High Availability (DHA) clusters directly from the Hybrid Manager (HM) console, including table management, tiered storage configuration, and catalog exploration.

AHA and DHA clusters include [Postgres Analytics Accelerator](/pgaa/latest) (PGAA), a high-performance extension that enables Postgres to query large-scale data stored in open table formats like Delta Lake, Apache Iceberg, and Parquet. 

Manage your cluster's analytics end-to-end from the HM console by deploying a single-node AHA or a minimal DHA cluster in your project. A dedicated cluster gives you a unified view of all analytical tables across your AHA and DHA clusters in a HM project, with table management, tiered storage configuration, and catalog exploration available from a single interface.

For concepts, architecture, and reference documentation, see the [PGAA documentation](/pgaa/latest/).
