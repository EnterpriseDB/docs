The Sovereign Data and AI Factory system is available in two configuration tiers: Core Compute and Advanced AI. Each system is preassembled and lifecycle-managed with full software and hardware support.

## Configuration options

| Configuration | Description | Total nodes |
|---------------|-------------|-------------|
| Core Compute | Designed for high-performance Postgres and hybrid DBaaS workloads | 7 |
| Advanced AI | Extends Core Compute with GPU nodes for AI and inference workloads | 11 |
