Learn about disaster recovery (DR) for your Hybrid Manager (HM) installation and clusters. This section includes the following topics:

-   [Overview](overview) &mdash; Provides an overview of the disaster recovery procedure and requirements.
-   [Configuring backups](backup_considerations) &mdash; Explains how to ensure the backup requirements for DR are met, while considering the recovery time objective (RTO) and recovery point objective (RPO) times.  
-   [Disaster recovery](hadr_recovery) &mdash; Provides a step-by-step guide to perform a disaster recovery run of your HM instance and HM-managed Postgres clusters using Velero. This includes restoring the HM control plane and data plane from backups, as well as verifying the restore process.
