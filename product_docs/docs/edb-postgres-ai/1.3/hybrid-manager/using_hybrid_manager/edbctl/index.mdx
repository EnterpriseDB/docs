edbctl is the dedicated command-line interface (CLI) for EDB Postgres AI Hybrid Manager (HM). It serves as the primary administrative tool for managing HM resources and executing sophisticated database cluster operations through a terminal environment.

By providing a unified entry point for both configuration and resource management, edbctl allows administrators to automate and streamline the lifecycle of database assets across diverse environments.

## Compatibility matrix

| edbctl version | HM version                | Notes                                                                             |
|----------------|---------------------------|-----------------------------------------------------------------------------------|
| 1.2.0          | v1.3.x(LTS) & 2026.2 (IR) | Supports the LTS and latest IR (2026.2) releases of HM.                           |
| 1.1.1          | v1.3.x (LTS)              | Patch release with improvements and bug fixes.                                    |
| 1.1.0          | v1.3.0 (LTS)              | First version compatible with HM v1.3.0+. It also has support for multi-location. |
| 1.0.0.         | Up to v1.2.0 (LTS)        | HM v1.3.0+ require edbctl 1.1.0+ .                                                |

For detailed information, see [edbctl documentation](/docs/edbctl/latest).
