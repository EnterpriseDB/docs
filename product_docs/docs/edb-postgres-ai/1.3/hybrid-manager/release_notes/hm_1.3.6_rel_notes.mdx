**Release Date:** March 12, 2026

Hybrid Manager 1.3.6 is a maintenance release that includes bug fixes and stability updates.

| Type         | Description                                                                                                     |
|--------------|-----------------------------------------------------------------------------------------------------------------|
| Enhancement  | Added support for more recent PostgreSQL minor versions (17.9 and 16.13) and removed support for PostgreSQL 15. |
| Deprecation  | Deprecated the Griptape (GenAI Builder) integration.                                                            |
| Bug&nbsp;fix | Fixed an issue whereby requests to the **Estate** endpoint failed with an HTTP 431 error.                       |
