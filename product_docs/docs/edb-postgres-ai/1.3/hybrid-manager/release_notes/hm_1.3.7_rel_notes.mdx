**Release Date:** April 13, 2026

Hybrid Manager 1.3.7 is a maintenance release that includes bug fixes and deprecations.

| Type         | Description                                                                                                                                               | 
|--------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| Bug&nbsp;fix | Fixed an inaccurate WAL storage percent metric on the monitoring page by correcting the underlying PromQL query.                                          |
| Deprecation  | Removed Spark Connect and Apache Spark support from Hybrid Manager. Spark functionality has been discontinued and is no longer available in this release. |
