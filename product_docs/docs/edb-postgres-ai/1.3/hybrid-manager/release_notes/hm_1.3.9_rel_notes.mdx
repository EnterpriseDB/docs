**Release date:** June 2, 2026

Hybrid Manager (HM) 1.3.9 is a maintenance release that includes bug fixes.

| Type         | Description                                                                                                                                                                                                                   |
|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Bug&nbsp;fix | Loki log retention policies weren't being enforced, as previously, logs declared with retention periods in the configuration were being retained indefinitely in object storage because the compactor retention feature was disabled by default. The compactor now properly enforces configured retention periods for control plane and vcluster deployments.
| Security;fix | Updated the CloudNativePG operator to version 1.28.3 to address security vulnerabilities in the operator. This update includes fixes for critical and high-severity vulnerabilities that could affect cluster security.   The fixes include a security vulnerability in  metrics exporter that could allow privilege escalation to PostgreSQL superuser and remote code execution on the operating system.. See `CVE-2026-44477` / [`GHSA-423p-g724-fr39`](https://github.com/cloudnative-pg/cloudnative-pg/security/advisories/GHSA-423p-g724-fr39) for details.
