**Release Date:** November 27, 2025

Hybrid Manager 1.3.2 is a maintenance release that includes bug fixes and stability updates.

## Core platform and deployment

- **Bug fix:** Added a fix to the `spire-agent` component which caused a connection error with the kubelet in RHOS deployments.
- **Bug fix:** Added a fix that caused `beacondb` backups to fail. 
- **Bug fix:** Added a fix to `kube-state-metrics` which prevented the service from exporting custom metrics.
- **Bug fix:** Added a fix for the `barman-plugin-v0-8-0-migrator` plugin which caused issued in RHOS deployments. 
- **Change:** Added a new `org.enterprisedb.operandset.version` annotation to improve release tracking and reporting. Also configured the `beacon-agent` service to send the annotation to the `beacon-server`.
- **Change:** Introduced new **image filtering logic** to the ImageLibrary to improve performance by limiting the number of images pulled.
- **Change:** The Helm chart configuration has been updated to correctly propagate the new ImageLibrary filters during installation.
- **Change:** The bootstrap stack now allows users to set the default backup compression status and the backup retention period (default: 30 days) directly at the time of installation. This provides control over internal storage consumption and data lifecycle policies.

## Security and authentication

- **Bug fix:** Fixed an issue that caused errors when configuring the platform to use a **custom certificate issuer**.
