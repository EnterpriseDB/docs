---
title: How-to  Set up data in the catalog
navTitle: Load data into catalog
description: Use the PyIceberg loader to write sample tables (e.g., tpch_sf_1.lineitem) into a Lakekeeper catalog.
---

```bash
python loader.py \
  --ca-cert ca.crt \
  --bucket beacon-analytics-demo-data-eu-west-1-prod \
  --prefix tpch_sf_1/lineitem \
  --endpoint $PARAMS_URI \
  --token $API_WRITE \
  --warehouse $PARAMS_WAREHOUSE \
  --namespace tpch_sf_1 \
  --table-name lineitem
```
