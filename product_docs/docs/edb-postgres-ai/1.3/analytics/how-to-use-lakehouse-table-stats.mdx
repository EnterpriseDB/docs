## Check size of an analytics reader table

```sql
SELECT * FROM pgaa.lakehouse_table_stats('lineitem'::regclass);
```

```sql
SELECT * FROM pgaa.lakehouse_table_stats('transactional_table_analytics_reader'::regclass);
```

See also: [Reference: pgaa.lakehouse_table_stats](reference/pgaa_functions.mdx).
