---
title: How-to  Restore back from analytics to transactional table
navTitle: Restore from analytics
description: Switch a PGD table back to heap, disable analytics offload, and validate access.
---

```sql
ALTER TABLE transactional_table SET ACCESS METHOD heap, SET (pgd.replicate_to_analytics = FALSE);

EXPLAIN SELECT COUNT(*) AS count, ROUND(SUM(revenue)) AS total_revenue FROM transactional_table;
```
