---
title: How-to  Reload a table to analytics with overwrites
navTitle: Overwrite analytics copy
description: Update data and re-enable analytics offload to overwrite the Iceberg copy; validate updated totals.
---

```sql
UPDATE transactional_table SET revenue = revenue * 2;
ALTER TABLE transactional_table SET (pgd.replicate_to_analytics = TRUE);

SELECT * FROM bdr.analytics_table; -- wait until initial sync finishes

SELECT COUNT(*) AS count, ROUND(SUM(revenue)) AS total_revenue FROM transactional_table_analytics_reader;
```
