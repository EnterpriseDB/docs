---
title: How-to  Query the cold data as a single table with PGAA
navTitle: Cold data view (PGAA)
description: Query the offloaded-only view created by PGAA for Tiered Tables.
---

```sql
SELECT * FROM public.application_logs_offloaded;

SELECT log_level, COUNT(*)
FROM public.application_logs_offloaded
GROUP BY log_level;
```

See also: `query-tiered-tables.mdx` for unified parent table queries.
