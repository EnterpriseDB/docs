In the Hybrid Manager (HM) console, create a machine user that the agent can use to ingest data into your estate.

## Creating a machine user

1.  Select your profile icon in the upper-right corner. 
1.  Select **User Management > Add New User**.
1.  On the **Add New User** page, enter a name for the new user. 
1.  Optionally, enter an email address for the user.
1.  Select **Create Access Key**. The form expands where you can enter an **Access Key Name** and an **Expiry** time.
1.  Select **Add User**.
1.  Store the access key safely. You won't be able to access the key again. Select **Key stored safely**. 

## Assigning roles

1.  Select the **Projects** tab to view all your projects. 
1.  Select the project from which you want to monitor the database. 
1.  Select the **Users** option in the left navigation, and locate the machine user you just created.
1.  Select the edit button (pen icon) of the user entry.
1.  Select **Assign Roles**.
1.  On the **Assign Project Roles** window, select **estate ingester**.
1.  Select **Submit**.

Your new machine user is ready to ingest data from the agent.
