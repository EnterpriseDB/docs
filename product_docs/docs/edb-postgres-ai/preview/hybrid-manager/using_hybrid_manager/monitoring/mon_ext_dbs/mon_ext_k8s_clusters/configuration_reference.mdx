Core configuration options:

| Parameter                                             | Description                              | Default            | Required              |
|-------------------------------------------------------|------------------------------------------|--------------------|-----------------------|
| agentConfig.accessKey                                 | HM machine user access key              | `""`               | Yes                   |
| agentConfig.projectID                                 | HM project ID                           | `""`               | Yes                   |
| agentConfig.beaconServer.host                         | HM server hostname                      | `""`               | Yes                   |
| agentConfig.locationID                                | Agent location identifier                | `"standalone-k8s"` | Yes                   |
| namespace                                             | Kubernetes namespace                     | `"upm-beacon"`     | No                    |
| **image.repository**                                  | Beacon agent image                       | `""`               | Yes                   |
| **image.tag**                                         | Beacon agent version                     | `""`               | Yes                   |
| **otelImage.repository**                              | OTEL collector image                     | `""`               | Yes (if OTEL enabled) |
| **otelImage.tag**                                     | OTEL collector version                   | `""`               | Yes (if OTEL enabled) |
| **additionalCharts.cert-manager.install**             | Enable cert-manager subchart             | `false`            | No                    |
| **additionalCharts.opentelemetry-collector.install**  | Enable opentelemetry-collector subchart  | `false`            | No                    |
| **additionalCharts.kube-state-metrics.install**       | Enable kube-state-metrics subchart       | `false`            | No                    |
| **additionalCharts.prometheus-node-exporter.install** | Enable prometheus-node-exporter subchart | `false`            | No                    |
| **opentelemetry-collector.image.repository**          | OpenTelemetry collector image repository | `""`               | Yes (if OTEL enabled) |
| **opentelemetry-collector.image.tag**                 | OpenTelemetry collector image tag        | `""`               | Yes (if OTEL enabled) |
