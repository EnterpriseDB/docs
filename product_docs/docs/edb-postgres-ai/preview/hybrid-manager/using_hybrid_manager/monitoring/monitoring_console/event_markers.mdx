# Event markers

With Event markers, you will have a clear record of metadata such as applications, database schemas, or datasets running at a given time on your clusters in the HM console.

!!!note
Alerts will be displayed in the Event markers, but are not logged in the activity log.
!!!

All events from the activity log associated with the clusters will be shown in the Event markers on a cluster or node level.

!!!note
Event markers reflect only the alert's start time and date for the cluster. If the issue causing the alert is solved, the Event markers will disappear.
!!!

### Using Event markers
Monitor metrics at the cluster and node levels in the HM console.

1. In the HM console, select your project.
1. In the left side pane, select **Clusters**.
1. Select **Postgres Clusters**.
1. Select the **Monitoring** tab.
1. On the right-hand side of your screen, select the dropdown menu.
1. Select the event you want to display.
