# Event markers

With Event markers you will have a clear record of metadata like application, database schema, or datasets running at a given time of your clusters in the HM console.

!!!note
Alerts will be displayed in theEvent markers but are not logged in the activity log.
!!!

All events from the activity log associated to the clusters will be shown in the Event markers on a cluster or node level.

!!!note
Event markers reflect only the starting time and date of the alert associated with the cluster. If the issue causing the alert is solved, the Event markers will disappear.
!!!

### Using Event markers
Monitor metrics on a cluster and node level in the HM console.

1. In the HM console, select your project.
1. In the left side pane, select **Clusters**.
1. Select **Postgres Clusters**.
1. Select the **Monitoring** tab.
1. On the right-hand side of your screen, select the dropdown menu.
1. Select the event you want to display.
