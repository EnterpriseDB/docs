Learn about disaster recovery (DR) for your Hybrid Manager (HM) installation and clusters. This section includes the following topics:

-   [Best practices](hadr.mdx) &mdash; Helps you plan your DR strategies and understand the requirements and considerations for a successful disaster recovery scenario.
-   [Backup considerations](backup_considerations.mdx) &mdash; Explains the backup requirements and considerations for HM, including the scope of DR, recovery time objective (RTO) and recovery point objective (RPO) times and backup readiness.  
-   [Disaster recovery](hadr_recovery.mdx) &mdash; Provides a step-by-step guide to perform a disaster recovery of your HM instance and Postgres clusters using Velero. This includes restoring the HM control plane and Postgres clusters from backups, as well as verifying the restore process.
