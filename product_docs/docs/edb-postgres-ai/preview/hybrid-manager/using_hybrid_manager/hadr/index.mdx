Learn about disaster recovery (DR) for your Hybrid Manager (HM) installation and clusters. This section includes the following topics:

-   [Overview](architecture) &mdash; Provides an overview of the disaster recovery procedure and requirements.
-   [Configuring backups](backup_considerations.mdx) &mdash; Explains how to ensure the backup requirements for DR are met, while considering their recovery time objective (RTO) and recovery point objective (RPO) times.  
-   [Disaster recovery](hadr_recovery.mdx) &mdash; Provides a step-by-step guide to perform a disaster recovery of your HM instance and Postgres clusters using Velero. This includes restoring the HM control plane and data plane from backups, as well as verifying the restore process.
