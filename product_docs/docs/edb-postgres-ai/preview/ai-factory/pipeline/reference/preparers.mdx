This reference documentation for Pipelines preparers includes information on the functions and views available in the aidb extension related to [preparers](/edb-postgres-ai/current/ai-factory/pipeline/preparers/). See [Usage](/edb-postgres-ai/current/ai-factory/pipeline/preparers/usage) and [Examples](/edb-postgres-ai/current/ai-factory/pipeline/preparers/examples) for more details.

## Views

### `aidb.preparers`

Also referenceable as `aidb.preps`, the `aidb.preparers` view contains information about the configured data preparers. It includes the following columns:

| Column                  | Type                                                           | Description                                                                                                                                    |
| ----------------------- | -------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| id                      | INTEGER                                                        |                                                                                                                                                |
| name                    | TEXT                                                           | Name of the preparer.                                                                                                                          |
| operation               | [aidb.DataPreparationOperation](#aidbdatapreparationoperation) | The kind of processing step to perform.                                                                                                        |
| destination_schema      | TEXT                                                           | Schema of the destination table to store the output data.                                                                                      |
| destination_table       | TEXT                                                           | Name of the destination table to store the output data.                                                                                        |
| destination_key_column  | TEXT                                                           | Column of the destination table that references the key in source data.                                                                        |
| destination_data_column | TEXT                                                           | Column of the destination table to store the output data.                                                                                      |
| options                 | JSONB                                                          | Configuration options for the data preparation operation. Uses the same API as the [data preparation primitives](/edb-postgres-ai/current/ai-factory/pipeline/preparers/primitives). |
| source_type             | TEXT                                                           | Type of source data the preparer is working with. Can be either 'Table' or 'Volume'.                                                           |
| source_schema           | TEXT                                                           | Schema of the table with the source data the preparer will process. Applies only to preparers of `Table` source type.                          |
| source_table            | TEXT                                                           | Name of the table with the source data the preparer will process. Applies only to preparers of `Table` source type.                            |
| source_data_column      | TEXT                                                           | Column in the source table with the source data the preparer will process. Applies only to preparers of `Table` source type.                   |
| source_key_column       | TEXT                                                           | Name of the key column in the source table for reference with the output processed data. Applies only to preparers of `Table` source type.     |
| source_volume_name      | TEXT                                                           | Name of the volume to use as a data source. Applies only to preparers of `Volume` source type.                                                 |

## Types

### `aidb.DataPreparationOperation`

The `aidb.DataPreparationOperation` type is an enum that represents the different types of preprocessing steps that can be performed:

* ChunkText
* SummarizeText
* ParseHtml
* ParsePdf
* PerformOcr

## Functions

### `aidb.create_table_preparer`

Creates a preparer with a source data table.

#### Parameters

| Parameter               | Type                                                           | Default     | Description                                                                                                                                    |
| ----------------------- | -------------------------------------------------------------- | ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| name                    | TEXT                                                           | Required    | Name of the preparer                                                                                                                           |
| operation               | [aidb.DataPreparationOperation](#aidbdatapreparationoperation) | Required    | Type of data preparation operation                                                                                                             |
| source_table            | TEXT                                                           | Required    | Name of the source data table                                                                                                                  |
| source_data_column      | TEXT                                                           | Required    | Column in the source table containing the raw data                                                                                             |
| destination_table       | TEXT                                                           | Required    | Name of the destination table                                                                                                                  |
| destination_data_column | TEXT                                                           | Required    | Column in the destination table for processed data                                                                                             |
| source_key_column       | TEXT                                                           | 'id'        | Unique column in the source table to use as key to reference the rows.                                                                         |
| destination_key_column  | TEXT                                                           | 'id'        | Key column in the destination table that references the `source_key_column`                                                                    |
| options                 | JSONB                                                          | '{}'::JSONB | Configuration options for the data preparation operation. Uses the same API as the [data preparation primitives](/edb-postgres-ai/current/ai-factory/pipeline/preparers/primitives). |

### `aidb.create_preparer_for_table` (deprecated)

**Replaced by** [`aidb.create_table_preparer`](#aidbcreate_table_preparer) with same arguments.

### `aidb.create_volume_preparer`

Creates a preparer for a given PGFS volume.

#### Parameters

| Parameter               | Type                                                           | Default     | Description                                                                                                                                    |
| ----------------------- | -------------------------------------------------------------- | ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| name                    | TEXT                                                           | Required    | Name of the preparer                                                                                                                           |
| operation               | [aidb.DataPreparationOperation](#aidbdatapreparationoperation) | Required    | Type of data preparation operation                                                                                                             |
| source_volume_name      | TEXT                                                           | Required    | Name of the source volume containing the raw data                                                                                              |
| destination_table       | TEXT                                                           | Required    | Name of the destination table                                                                                                                  |
| destination_data_column | TEXT                                                           | Required    | Column in the destination table for processed data                                                                                             |
| destination_key_column  | TEXT                                                           | 'id'        | Key column in the destination table that uniquely identifies the processed data                                                                |
| options                 | JSONB                                                          | '{}'::JSONB | Configuration options for the data preparation operation. Uses the same API as the [data preparation primitives](/edb-postgres-ai/current/ai-factory/pipeline/preparers/primitives). |
| invoker_role            | TEXT                                                           | NULL        | Role owning the tables, pipelines, and background job execution.                                                                               |

### `aidb.create_preparer_for_volume` (deprecated)

**Replaced by** [`aidb.create_volume_preparer`](#aidbcreate_volume_preparer) with same arguments.

### `aidb.bulk_data_preparation`

Executes the configured data preparation operation on all data from the specified preparer’s source.

#### Parameters

| Parameter                       | Type                           | Description                  |
| ------------------------------- | ------------------------------ | ---------------------------- |
| preparer_name                   | TEXT                           | Name of the preparer         |

### `aidb.set_auto_preparer`

Sets the automatic processing mode for this preparer. This function is used to enable and disable automatic data preparation: `Live` mode enables the Postgres trigger-based automation and `Disabled` disables all automation.

#### Parameters

| Parameter     | Type                                                                                | Default | Description             |
| ------------- | ----------------------------------------------------------------------------------- | ------- | ----------------------- |
| preparer_name | TEXT                                                                                |         | Name of preparer        |
| mode          | [aidb.PipelineAutoProcessingMode](./knowledge_bases#aidbpipelineautoprocessingmode) |         | Desired processing mode |

#### Example

```sql
SELECT aidb.set_auto_preparer('test_preparer', 'Live');
SELECT aidb.set_auto_preparer('test_preparer', 'Disabled');
```

### `aidb.set_preparer_auto_processing` (deprecated)

**Replaced by** [`aidb.set_auto_preparer`](#aidbset_auto_preparer) with same arguments.

### `aidb.delete_preparer`

Deletes the preparer's configuration.

#### Parameters

| Parameter                       | Type                           | Description                      |
| ------------------------------- | ------------------------------ | -------------------------------- |
| preparer_name                   | TEXT                           | Name of preparer to delete       |


!!! Note
This  function doesn't delete the destination table or any data in it.
!!!

## Step configuration helper functions

These helper functions simplify creating JSONB configuration objects for preparer operations. They provide proper parameter types, defaults, and validation.

### `aidb.chunk_text_config`

Creates configuration for the ChunkText operation.

#### Parameters

| Parameter       | Type    | Default | Description                                         |
| --------------- | ------- | ------- | --------------------------------------------------- |
| `desired_length`| INTEGER | Required| Target chunk length in characters                   |
| `max_length`    | INTEGER | NULL    | Maximum chunk length                                |
| `overlap_length`| INTEGER | NULL    | Overlap between chunks                              |
| `strategy`      | TEXT    | NULL    | Chunking strategy                                   |

#### Example

```sql
SELECT aidb.create_table_preparer(
    name => 'my_chunker',
    operation => 'ChunkText',
    source_table => 'source_data',
    source_data_column => 'content',
    destination_table => 'chunked_data',
    destination_data_column => 'chunks',
    options => aidb.chunk_text_config(desired_length => 500, max_length => 1000)
);
```

### `aidb.ocr_config`

Creates configuration for the PerformOcr operation.

#### Parameters

| Parameter | Type | Default | Description                |
| --------- | ---- | ------- | -------------------------- |
| `model`   | TEXT | Required| Name of the OCR model      |

#### Example

```sql
SELECT aidb.create_table_preparer(
    name => 'my_ocr',
    operation => 'PerformOcr',
    source_table => 'images',
    source_data_column => 'image_data',
    destination_table => 'ocr_results',
    destination_data_column => 'text',
    options => aidb.ocr_config(model => 'my_paddle_ocr_model')
);
```

### `aidb.html_parse_config`

Creates configuration for the ParseHtml operation.

#### Parameters

| Parameter | Type | Default | Description                                              |
| --------- | ---- | ------- | -------------------------------------------------------- |
| `method`  | TEXT | NULL    | Parsing method: 'StructuredPlaintext' or 'StructuredMarkdown' |

#### Example

```sql
SELECT aidb.create_table_preparer(
    name => 'my_html_parser',
    operation => 'ParseHtml',
    source_table => 'html_content',
    source_data_column => 'html',
    destination_table => 'parsed_content',
    destination_data_column => 'text',
    options => aidb.html_parse_config(method => 'StructuredMarkdown')
);
```

### `aidb.pdf_parse_config`

Creates configuration for the ParsePdf operation.

#### Parameters

| Parameter              | Type    | Default | Description                           |
| ---------------------- | ------- | ------- | ------------------------------------- |
| `method`               | TEXT    | NULL    | Parsing method: 'Structured'          |
| `allow_partial_parsing`| BOOLEAN | NULL    | Allow partial parsing on errors       |

#### Example

```sql
SELECT aidb.create_table_preparer(
    name => 'my_pdf_parser',
    operation => 'ParsePdf',
    source_table => 'pdf_files',
    source_data_column => 'pdf_data',
    destination_table => 'parsed_pdfs',
    destination_data_column => 'text',
    options => aidb.pdf_parse_config(method => 'Structured', allow_partial_parsing => true)
);
```

### `aidb.summarize_text_config`

Creates configuration for the SummarizeText operation.

#### Parameters

| Parameter        | Type    | Default | Description                              |
| ---------------- | ------- | ------- | ---------------------------------------- |
| `model`          | TEXT    | Required| Name of the summarization model          |
| `chunk_config`   | JSONB   | NULL    | Chunk config (use `aidb.chunk_text_config()`) |
| `prompt`         | TEXT    | NULL    | Custom summarization prompt              |
| `strategy`       | TEXT    | NULL    | Summarization strategy                   |
| `reduction_factor`| INTEGER| NULL    | Target reduction factor                  |

#### Example

```sql
SELECT aidb.create_table_preparer(
    name => 'my_summarizer',
    operation => 'SummarizeText',
    source_table => 'documents',
    source_data_column => 'content',
    destination_table => 'summaries',
    destination_data_column => 'summary',
    options => aidb.summarize_text_config(model => 'my_t5_model')
);
```

### `aidb.knowledge_base_config`

Creates configuration for knowledge base pipeline steps.

#### Parameters

| Parameter          | Type                       | Default | Description                                |
| ------------------ | -------------------------- | ------- | ------------------------------------------ |
| `model`            | TEXT                       | Required| Name of the embedding model                |
| `data_format`      | aidb.PipelineDataFormat    | Required| Data format for the pipeline               |
| `topk`             | INTEGER                    | NULL    | Number of results to return                |
| `distance_operator`| aidb.DistanceOperator      | NULL    | Distance operator for similarity           |
| `index_type`       | aidb.vector_index_type     | NULL    | Type of vector index                       |
| `vector_index`     | JSONB                      | NULL    | Vector index config (use vector_index helpers) |

#### Example

```sql
aidb.knowledge_base_config(
    model => 'my_embeddings',
    data_format => 'Text',
    topk => 10,
    vector_index => aidb.vector_index_hnsw_config(m => 16, ef_construction => 64)
)
