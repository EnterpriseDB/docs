---
title: "Version 2.5"
---

LiveCompare 2.5 includes the following new features, enhancements, bug fixes, and other changes:

|  Type   |                                                 Description                                                 |
| ------- | ----------------------------------------------------------------------------------------------------------- |
| New feature | Support for EDB Postgres Distributed 5.|
| Improvement | LiveCompare now holds a version agnostic list of reserved words for Oracle and Postgres, used to properly handle quoted identifiers.|
| Bug fix | Fixed an issue where the HandleIgnoredColumns method was failing in mixed comparison mode. |
| Bug fix | Fixed an issue where the PrepareMask method was failing when using a custom collate.|
| Bug fix | Fixed an issue where using the `node_name` caused LiveCompare to try the pglogical metadata for PGD 4 and 5.|
