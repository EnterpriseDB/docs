---
title: pgd show-events
---

Shows background worker errors, node membership changes etc.

### Synopsis


Shows background worker errors, node membership changes etc. Output is sorted
by 'Time' column in descending order. 'Message' column is truncated after few
lines. To view complete message use json output format (-o json).

For more details on each node state please check show-nodes command help
(pgd show-nodes -h)

```
pgd show-events [flags]
```
