At this point in time of the migration, you have taken a snapshot of the source database and imported it to the EDB PostgresÂ® AI Console.

To ensure that the target cluster is up-to-date with the source cluster and allow the reader to be able to stream the latest updates on the source database to the cluster mark the migration as completed.

1.  In the EDB Postgres AI Console, in your project, select **Migrate** > **Migrations**. 

1.  Select the **Mark as completed** button.
