title: The pgd command
---

'pgd' is a command-line interface (CLI) for 'EDB Postgres Distributed'

### Synopsis


The PGD Command Line Interface (CLI) is a tool to manage your BDR cluster. PGD
stands for 'EDB Postgres Distributed'. It allows you to run commands against
BDR clusters. You can use it to inspect and manage cluster resources, and view
membership events, worker errors etc.

### Options

```
  -f, --config-file string   config file; ignored if
                             --dsn flag is present (default "/etc/edb/pgd-config.yml")
      --dsn string           database connection string
                             e.g."host=bdr-a1 port=5432 dbname=bdrdb user=postgres "
  -h, --help                 help for pgd
  -L, --log-level string     logging level: debug, info, warn, error (default "error")
  -o, --output string        output format: json, yaml
```
