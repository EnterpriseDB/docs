Released: 17 June 2026

The edbctl 1.7.0 release includes new features to enhance the functionality and performance of the Hybrid Manager CLI tool.

| Type    | Description                                                                                                                                                                                      |
|---------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Feature | Added `edbctl disaster-recovery` command group with `create-restore-objectstores` and `prepare-cluster-configs` subcommands for managing disaster recovery operations on HM-internal clusters.   |
| Feature | Added `edbctl image cleanup-registry` command to remove unused images from the HM registry and reclaim storage after an upgrade.                                                                 |
| Feature | Added `edbctl hybrid-manager upgrade-operator` command to install or upgrade the Hybrid Manager Operator via Helm from EDB Cloudsmith or a private registry.                                     |
