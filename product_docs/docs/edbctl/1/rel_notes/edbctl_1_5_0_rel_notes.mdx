Released: 17 Jun 2026

The edbctl 1.5.0 release includes new features to enhance the functionality and performance of the Hybrid Manager CLI tool.

| Type    | Description                                                                                                                                                                                                                                                                                                                    |
|---------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Feature | Added support for custom load balancer annotations when creating or editing clusters. Set `networking: custom` in the cluster configuration YAML to provide key-value annotation pairs for read-write and read-only connections. This option is only available when the location has `custom_annotations_enabled` set to `true`. |
