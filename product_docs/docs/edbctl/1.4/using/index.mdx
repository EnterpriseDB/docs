Use the `edbctl --help or -h` command to return the complete list of commands:

```
edbctl --help
__OUTPUT__
Usage:
  edbctl [options]
  edbctl [command]

Management Commands:
  credential        Manage authentication credentials.
  project           Manage projects associated with the database clusters.
  cluster           Manage database cluster resources.
  backup            Manage cluster backups.
  hybrid-manager    Hybrid Manager management, monitoring, and diagnostics.

Configuration Commands:
  config            Manages configuration settings for edbctl tool
  help              Help about any command
  completion        Generate the autocompletion script for the specified shell

Setup Commands:
  image             Asset management commands
  operand           Operand image management commands
  image-pull-secret Pull Secret management commands

Options:
  -h, --help      help for edbctl
  -v, --version   version for edbctl

Use "edbctl [command] --help" for more information about a command.
```

## edbctl commands list

-   [edbctl backup](edbctl_backup.mdx)	— Manage cluster backups.
-   [edbctl catalog](edbctl_catalog.mdx)	— Manage database catalog resources.
-   [edbctl cluster](edbctl_cluster.mdx) — Manage database cluster resources.
-   [edbctl cluster-template](edbctl_cluster_template.mdx)	— Manage database cluster template resources.
-   [edbctl config](edbctl_config.mdx)	— Manage configuration settings for `edbctl` tool.
-   [edbctl credential](edbctl_credential.mdx)	— Manage authentication credentials.
-   [edbctl hybrid-manager](edbctl_hybrid_manager.mdx) — Hybrid Manager management, monitoring, and diagnostics.
-   [edbctl image](edbctl_image.mdx) — Manage asset images.
-   [edbctl image-pull-secret](edbctl_image_pull_secret.mdx) — Pull secret management commands.
-   [edbctl location](edbctl_location.mdx) — Manage locations for database cluster provisioning.
-   [edbctl operand](edbctl_operand.mdx) — Manage operand images.
-   [edbctl operator](edbctl_operator.mdx) — Manage the Postgres AI Operator.
-   [edbctl project](edbctl_project.mdx) — Manage projects associated with the database clusters.
