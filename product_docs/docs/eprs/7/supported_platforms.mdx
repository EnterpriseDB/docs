Replication Server is certified to work with the following Java platforms:

| Operating systems                       | JDK versions                                                                                                          |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| CentOS 7 and Rocky Linux/AlmaLinux 8    | Red Hat OpenJDK 8                                                                                               |
| SLES 15                                 | OpenJDK 11                                                               | 
| SLES 12                                 | Red Hat OpenJDK 8                                                                                               |
| PPCLE RHEL 8                            | Red Hat OpenJDK 8 |
| RHEL 7 and 8                            |  -   Red Hat OpenJDK 8     <br /> <br /> -   Oracle JDK 8     <br />         |
| Windows Server 2019                     | Red Hat OpenJDK 8                                                                                          |
| Debian 10 and 11                        | Red Hat OpenJDK 11                                                                          |               
| Ubuntu 18, 20, 22                        | OpenJDK 11                                                        |

See [Product Compatibility](https://www.enterprisedb.com/platform-compatibility#eprs) for more information.
