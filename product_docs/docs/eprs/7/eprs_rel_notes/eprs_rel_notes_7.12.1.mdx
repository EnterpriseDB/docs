Released: 07 Aug 2025

New features, enhancements, bug fixes, and other changes in Replication Server 7.12.1 include the following:

| Type         | Description                                                                                                                                                                              | Ticket |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------|
| Bug&nbsp;fix | Fixed an issue where the Oracle shadow table cleanup procedure was failing on shadow tables with a large volume of historical data.                                                      | #49863 |
| Bug&nbsp;fix | Fixed an issue that was causing a sync failure in a 3-node MMR cluster whenever a previously failed node was rejoining the replication cluster.                                          | #49021 |
| Bug&nbsp;fix | Fixed an issue where an Oracle publication creation was failing for the tables with UID columns.                                                                                         | #50125 |
| Bug&nbsp;fix | Fixed an issue where sync was failing in BUP mode with a `NumberFormatException` error when large numeric values were incorrectly parsed as integer values instead of BigDecimal values. | #49863 |
