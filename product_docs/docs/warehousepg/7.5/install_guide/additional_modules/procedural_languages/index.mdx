The following procedural languages ship with WarehousePG and don't require a separate package install. Enable them with `CREATE EXTENSION` in the target database:

-   PL/pgSQL — `CREATE EXTENSION plpgsql;`
-   PL/Perl — `CREATE EXTENSION plperl;`
-   PL/Python — `CREATE EXTENSION plpython3u;`

No additional procedural languages are available yet.
