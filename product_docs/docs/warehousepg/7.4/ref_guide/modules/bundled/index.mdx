These modules ship with WarehousePG. Activate them in each database with `CREATE EXTENSION <module_name>`, unless the individual reference page specifies a different activation method.

-   adminpack - Provides administrative support functions used by management tools such as pgAdmin.
-   amcheck - Provides functions for verifying the logical consistency of indexes and heap relations.
-   [auto_explain](auto-explain.mdx) - Logs execution plans of slow statements automatically.
-   bloom - Implements a Bloom filter-based index access method for equality comparisons on multiple columns.
-   [btree_gin](btree_gin.mdx) - Provides GIN operator classes that implement B-tree equivalent behavior for certain data types.
-   btree_gist - Provides GiST operator classes that implement B-tree equivalent behavior for a range of data types.
-   [citext](citext.mdx) - Provides a case-insensitive, multibyte-aware text data type.
-   cube - Provides a data type for representing multidimensional cubes.
-   [dblink](dblink.mdx) - Provides connections to other WarehousePG databases.
-   dict_int - Provides a text search dictionary template that controls indexing of integers.
-   dict_xsyn - Provides a text search dictionary template for extended synonym processing.
-   earthdistance - Provides two approaches to calculating great-circle distances on Earth's surface.
-   file_fdw - Provides a foreign-data wrapper for accessing data files in the server's file system.
-   [gp_exttable_fdw](gp_exttable_fdw.mdx) - Built-in foreign-data wrapper that internally converts an external table to a foreign table.
-   [gp_legacy_string_agg](gp_legacy_string_agg.mdx) - Implements a legacy, single-argument `string_agg()` aggregate function.
-   [gp_sparse_vector](gp_sparse_vector.mdx) - Implements a data type that uses compressed storage of zeros to make vector computations on floating point numbers faster.
-   [hstore](hstore.mdx) - Provides a data type for storing sets of key/value pairs within a single value.
-   intarray - Provides additional functions, operators, and index support for arrays of integers with no null elements.
-   [isn](isn.mdx) - Provides data types for international product numbering standards (EAN13, UPC, ISBN, ISMN, ISSN).
-   lo - Provides support for managing large objects.
-   [ltree](ltree.mdx) - Provides data types for representing labels of data stored in a hierarchical tree-like structure.
-   [orafce](orafce_ref.mdx) - Provides Oracle SQL compatibility functions.
-   [pageinspect](pageinspect.mdx) - Provides functions for low-level inspection of the contents of database pages. Available to superusers only.
-   [pg_buffercache](pg_buffercache.mdx) - Provides access to views for obtaining cluster-wide shared buffer metrics.
-   pg_freespacemap - Provides a function for examining the free space map.
-   pg_hint_plan - Allows controlling query execution plans using hints specified in SQL comments.
-   pg_prewarm - Provides functions for loading relation data into the buffer cache.
-   [pg_stat_statements](pg_stat_statements.mdx) - Tracks execution statistics of all SQL statements. Requires `shared_preload_libraries`.
-   [pg_trgm](pg_trgm.mdx) - Provides functions and operators for determining the similarity of alphanumeric text based on trigram matching.
-   pg_visibility - Provides functions for examining the visibility map and page-level visibility information of a relation.
-   [pgcrypto](pgcrypto.mdx) - Provides cryptographic functions.
-   pgrowlocks - Provides a function that shows row locking information for a given table.
-   pgstattuple - Provides functions for obtaining tuple-level and index-level statistics.
-   [PL/Perl](../../../admin_guide/analytics/procedural_languages/pl_perl.mdx) - Enables writing functions and triggers in Perl.
-   [PL/pgSQL](../../../admin_guide/analytics/procedural_languages/pl_sql.mdx) - Enables writing functions, triggers, and procedural code in a SQL-like language.
-   [PL/Python](../../../admin_guide/analytics/procedural_languages/pl_python.mdx) - Enables writing functions in Python.
-   [postgres_fdw](postgres_fdw.mdx) - Provides a foreign data wrapper for accessing data stored in an external Postgres database.
-   seg - Provides a data type for representing line segments or floating-point intervals.
-   [sslinfo](sslinfo.mdx) - Provides information about the SSL certificate of the current client connection.
-   [tablefunc](tablefunc.mdx) - Provides various functions that return tables (multiple rows).
-   [tsm_system_rows](tsm_system_rows.mdx) - Implements the `SYSTEM_ROWS` table sampling method.
-   [tsm_system_time](tsm_system_time.mdx) - Implements the `SYSTEM_TIME` table sampling method.
-   unaccent - Provides a text search dictionary that removes accents from lexemes.
-   [uuid-ossp](uuid-ossp.mdx) - Provides functions to generate universally unique identifiers (UUIDs).
-   xml2 - Provides XPath querying and XSLT functionality.
