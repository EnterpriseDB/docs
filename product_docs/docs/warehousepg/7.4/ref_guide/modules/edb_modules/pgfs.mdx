The Postgres File System (PGFS) extension provides the storage layer for object storage connectivity in WarehousePG. It handles credential management and abstracts access to S3, Google Cloud Storage (GCS), and Azure, and is a required dependency of the [Postgres Analytics Accelerator (PGAA)](pgaa) extension.

## Installing PGFS

PGFS is installed automatically as a dependency when you install PGAA. For full installation instructions, see [Installing PGAA on WarehousePG](/pgaa/latest/whpg/installing/).

## Using PGFS

PGFS is used through PGAA. For full documentation, see [Scaling with WarehousePG](/pgaa/latest/whpg/).
