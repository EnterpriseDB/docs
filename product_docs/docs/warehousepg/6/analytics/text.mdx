WarehousePG offers two different methods for text search and analytics, **WarehousePG full text search** and **WarehousePG Text**.

<a id="section_yby_nv1_rqb"></a>

## WarehousePG Full Text Search

WarehousePG text search is PostgreSQL text search ported to the WarehousePG MPP platform. WarehousePG text search is immediately available to you, with no need to install and maintain additional software. For full details on this topic, see [WarehousePG text search](../admin_guide/textsearch/intro.mdx).

<a id="section_ywf_4v1_rqb"></a>

## WarehousePG Text

For advanced text analysis applications, VMWare also offers [WarehousePG Text](https://docs.vmware.com/en/VMware-Greenplum-Text/index.html), which integrates WarehousePG with the Apache Solr text search platform. WarehousePG Text installs an Apache Solr cluster alongside your WarehousePG cluster and provides WarehousePG functions you can use to create Solr indexes, query them, and receive results in the database session.

Both of these systems provide powerful, enterprise-quality document indexing and searching services. WarehousePG Text, with Solr, has many capabilities that are not available with WarehousePG text search. In particular, WarehousePG Text is better for advanced text analysis applications. For a comparative between these methods, see [Comparing WarehousePG Text Search with WarehousePG Text](../admin_guide/textsearch/intro.mdx#gptext).
