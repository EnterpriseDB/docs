Extensions require installing a package on each host in your cluster before enabling with `CREATE EXTENSION`. See [Extensions](../../../ref_guide/modules/extensions) for the full list.

## Downloading and installing an EDB-provided extension

EDB provides extension packages through the EDB repository.

1.  Set up the EDB repository:

    <TabContainer>
    <Tab label="RHEL 8 or 9">

    ```bash
    dnf install -y https://yum.enterprisedb.com/edbrepos/edb-repo-latest.noarch.rpm
    ```

    </Tab>
    <Tab label="RHEL 7">

    ```bash
    yum install -y https://yum.enterprisedb.com/edbrepos/edb-repo-latest.noarch.rpm
    ```

    </Tab>
    </TabContainer>

2.  Replace `<USERNAME>:<PASSWORD>` with your EDB repository credentials in the repository configuration:

    ```bash
    sed -i "s/<USERNAME>:<PASSWORD>/your_username:your_password/" /etc/yum.repos.d/edb.repo
    ```

3.  Install the package:

    <TabContainer>
    <Tab label="RHEL 8 or 9">

    ```bash
    dnf install -y <package_name>
    ```

    </Tab>
    <Tab label="RHEL 7">

    ```bash
    yum install -y <package_name>
    ```

    </Tab>
    </TabContainer>

After installing, connect to the target database and enable the extension:

```sql
CREATE EXTENSION <extension_name>;
```
