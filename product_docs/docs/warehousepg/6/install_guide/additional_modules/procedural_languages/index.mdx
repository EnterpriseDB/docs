---
title: Installing procedural languages
description: How to install procedural language extensions that require a separate package install.
displayBanner: ''

---

PL/pgSQL, PL/Perl, and PL/Python ship with WarehousePG and don't require a separate install. Enable them with `CREATE EXTENSION` in the target database.

The following procedural languages require a separate package install on each host:

-   **[Installing PL/Java](../../../admin_guide/analytics/procedural_languages/pl_java.mdx)**  
    Enables writing functions in the Java programming language.

-   **[Installing PL/R](../../../admin_guide/analytics/procedural_languages/pl_r.mdx)**  
    Enables writing functions in the R programming language.
