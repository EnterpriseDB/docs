Additional programs available in the WarehousePG installation.

The following PostgreSQL `contrib` server utility programs are installed:

-   [pg_upgrade](https://www.postgresql.org/docs/9.4/pgupgrade.html) - Server program to upgrade a Postgres Database server instance.

    > **Note** `pg_upgrade` is not intended for direct use with WarehousePG 6.

-   `pg_upgrade_support` - supporting library for `pg_upgrade`.

-   [pg_xlogdump](https://www.postgresql.org/docs/9.4/pgxlogdump.html) - Server utility program to display a human-readable rendering of the write-ahead log of a WarehousePG cluster.
