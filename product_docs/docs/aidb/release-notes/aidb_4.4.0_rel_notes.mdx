Released: 13 August 2025

AIDB 4.4.0 includes the following enhancements and bug fixes:

| Type              | Description                                                                                                                                                                              |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Enhancement&nbsp; | **Added support for non-superuser:** `aidb` can now run with non-superuser execution support for enhanced security and access control.                                                    |
| Enhancement       | **Improved performance:** Improved performance when running knowledge base pipelines on large datasets.                                                                                   |
