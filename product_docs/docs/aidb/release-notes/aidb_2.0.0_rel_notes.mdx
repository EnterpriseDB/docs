Released: 13 January 2025

AIDB 2.0.0 updates the GA release of AIDB in response to user feedback, focusing on improved API naming consistency.

| Type              | Description                                                                                                                                                                                                                                                                   |
|-------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Enhancement&nbsp; | **Renamed management functions and arguments:** Management functions now use `create` rather than `register` and `delete` rather than `drop`. Function argument names no longer use the preceding `p_` prefix, for consistency across the API.                                  |
