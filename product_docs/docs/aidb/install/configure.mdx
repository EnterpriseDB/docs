title: Configuring AIDB
navTitle: Configuring
---

To use AIDB, you need to add it to `shared_preload_libraries` and create the extension in the database. 

1. In the `postgresql.conf` file, add `aidb` to the `shared_preload_libraries` parameter:

   ```ini
   shared_preload_libraries = 'aidb'
   ```

   !!! Note
   If `shared_preload_libraries` has other extensions, then you can add `aidb` to the list. The order doesn't matter. 
   !!!

1. Restart Postgres. 

1. Create the AIDB extension in your database: 

   ```sql
   CREATE EXTENSION aidb;
   ```
