## AIDB

### Supported platforms

| Platform             | x86/64 | ARM64 |
|----------------------|--------|-------|
| Ubuntu 22.04 LTS     | ✓      | —     |
| Ubuntu 24.04 LTS     | ✓      | —     |
| Debian 12 (Bookworm) | ✓      | ✓     |
| RHEL 8               | ✓      | —     |
| RHEL 9               | ✓      | ✓     |

SLES, earlier Debian versions, and non-Linux platforms are not currently supported.

### Supported Postgres versions

AIDB supports Postgres versions 14–18, including EDB Postgres Advanced Server, EDB Postgres Extended, and community PostgreSQL.

## PGFS

AIDB pipelines that read from cloud storage (S3, GCS, Azure) require the PGFS extension. For platform and Postgres version compatibility, see [PGFS compatibility](/pg_extensions/pgfs/compatibility).
