AIDB is EDB's PostgreSQL extension for building AI-powered applications. It provides pipelines for knowledge base management, data preparation, and retrieval-augmented generation (RAG), along with integrations for internal and external AI models.

The AIDB documentation describes the latest version of AIDB. These release notes provide information on what was new in each release.
