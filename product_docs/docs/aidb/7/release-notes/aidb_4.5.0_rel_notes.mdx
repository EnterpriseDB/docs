navTitle: "Version 4.5.0"

Released: 10 Sep 2025

AIDB 4.5.0 includes the following enhancements and bug fixes:

| Type              | Description                                                                                                                                                                              |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Enhancement       | **Vector index module:** Introduced a dedicated vector index module, providing a unified interface for managing and configuring vector indexes in AIDB.                                   |
| Enhancement       | **Generic background workers:** Added support for generic background worker infrastructure, enabling more flexible and extensible background processing for pipelines.                    |
| Enhancement       | **Auto-processing updates:** Improved pipeline auto-processing behavior for more reliable background execution.                                                                           |
| Enhancement       | **Non-superuser support:** Refactored permission handling to allow non-superusers to use AIDB with appropriate roles and privileges.                                                      |
| Enhancement       | **Special characters in volume file names:** Volume file names can now include special characters, expanding compatibility with a wider range of file naming conventions.                 |
| Enhancement       | **TLS and cryptography updates:** Updated TLS and core cryptography dependencies to improve security and compatibility across supported platforms.                                        |
