AIDB functions are available as standalone SQL functions that you can call directly in queries for one-off transformations, testing and exploration, or to build custom workflows outside of the pipeline framework.

| Function                                                     | Description                                                                           |
|--------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [`aidb.chunk_text()`](chunk-text)                            | Divides long text into smaller, semantically coherent segments.                       |
| [`aidb.parse_html()`](parse)                                 | Extracts readable text from HTML, stripping tags while preserving structure.          |
| [`aidb.parse_pdf()`](parse)                                  | Extracts text from binary PDF data, with page-level `part_id` output.                 |
| [`aidb.perform_ocr()`](perform-ocr)                          | Extracts text from images using an OCR-capable AI model.                              |
| [`aidb.summarize_text()`](summarize-text)                    | Generates concise summaries of long text passages using an AI model.                  |
| [`aidb.summarize_text_aggregate()`](summarize-text)          | Summarizes text across multiple rows using a SQL aggregate pattern.                   |
| [`aidb.encode_text()`](encode-decode)                        | Encodes a text string into a vector using a registered embedding model.               |
| [`aidb.encode_text_batch()`](encode-decode)                  | Encodes multiple text strings into vectors in a single batched request.               |
| [`aidb.encode_image()`](encode-decode)                       | Encodes an image into a vector using a registered multimodal embedding model.         |
| [`aidb.decode_text()`](encode-decode)                        | Generates text from a language model given a prompt.                                  |
| [`aidb.decode_text_batch()`](encode-decode)                  | Generates text responses for multiple prompts in a single batched request.            |
| [`aidb.inference_config()`](encode-decode)                   | Builds an inference configuration object to control language model generation behavior. |
| [`aidb.rerank_text()`](rerank-text)                          | Scores and reorders candidate text passages by relevance to a query.                  |

!!!Note
To use these operations as steps inside a pipeline, see [Pipeline steps](../data-pipelines/pipeline-steps/pipeline-steps.mdx).
