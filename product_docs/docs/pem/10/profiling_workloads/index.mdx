The SQL Profiler extension works with PEM to allow you to profile a server's workload in fine detail.
Use SQL Profiler to collect query plans, buffer usage, execution times etc.

- See [Installing the SQL Profiler extension](installing_sql_profiler)
- See [Upgrading SQL Profiler](../upgrading/upgrading_sql_profiler)
- See [Using SQL Profiler](using_sql_profiler)
