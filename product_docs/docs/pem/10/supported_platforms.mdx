For information about the platforms and versions supported by PEM, see [Platform Compatibility](https://www.enterprisedb.com/platform-compatibility#pem).

!!! Note
    Postgres Enterprise Manager 8.3 and later is supported on SLES.

!!! Warning
    Part of PEM 10.3.x is not supported with RHEL9.6. Consequently, users on PEM 10.3 (ARM) and PEM 10.3.1 (Intel/ARM) must upgrade to PEM 10.4 to maintain full support.
