---
title: Deleting an alert
---

To mark an alert for deletion, select the **Delete** button next to the alert name and click **Yes** to confirm. This returns you to the main Alert templates list. 

You can repeat this process to mark multiple alerts for removal. 

To permanently apply these deletions, select **Save** in the top-right corner. This "bulk save" approach allows you to queue up multiple changes and commit them all with a single click.
