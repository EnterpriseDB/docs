PEM can send a notification or execute a script if an alert is triggered or cleared. You can send notifications using the following options:

- [SMTP](./smtp.mdx)
- [Webhooks](./webhooks.mdx)
- [SNMP](./snmp.mdx)
- [Nagios](./nagios.mdx)
