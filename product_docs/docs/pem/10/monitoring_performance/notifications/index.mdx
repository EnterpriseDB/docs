PEM can send a notification or execute a script if an alert is triggered or cleared. You can send notifications using the following options:

- [SMTP](/pem/10/monitoring_performance/notifications/smtp.mdx)
- [Webhooks](/pem/10/monitoring_performance/notifications/webhooks.mdx)
- [SNMP](/pem/10/monitoring_performance/notifications/snmp.mdx)
- [Nagios](/pem/10/monitoring_performance/notifications/nagios.mdx)
