`POST /probe/custom/import/`

Imports custom probes using exported probes.

**Body parameter**

```json
{
  "version": 1,
  "skip_overwrite": true,
  "probes": [
    {
      "probe_name": "string",
      "collection_method": "w",
      "enabled": true,
      "target_type": 0,
      "any_server_version": true,
      "any_extension_version": true,
      "extension_name": "string",
      "interval": 0,
      "lifetime": 0,
      "discard_history": true,
      "platform": "unix",
      "probe_code": "string",
      "probe_columns": [
        {
          "pc_name": "string",
          "pc_data_type": "bigint",
          "pc_unit": "string",
          "pc_col_type": "m",
          "pc_graphable": true,
          "pc_calc_pit": true,
          "pc_pit_default": true
        }
      ],
      "alternate_code": [
        {
          "server_version_id": 0,
          "server_probe_code": "string",
          "extension_version": 0
        }
      ],
      "internal_name": "string"
    }
  ]
}
```

<span id="post__probe_custom_import_-parameters"></span>

### Parameters



| Name         | In     | Type                                                            | Required | Description            |
| ------------ | ------ | --------------------------------------------------------------- | -------- | ---------------------- |
| X-Auth-Token | header | string                                                          | true     | Token of authorization |
| body         | body   | [CustomProbeImportData](schemas.mdx#tocs_customprobeimportdata) | true     | none                   |

**Example responses**

<Details summary="200 Response">

```json
[
  {
    "name": "string",
    "status": "string",
    "msg": "string"
  }
]
```

</Details>
