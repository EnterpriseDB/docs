### Parameters



| Name         | In     | Type    | Required | Description                                           |
| ------------ | ------ | ------- | -------- | ----------------------------------------------------- |
| probe_id     | path   | integer | true     | Probe ID for which probe information will be returned |
| X-Auth-Token | header | string  | true     | Token of authorization                                |
