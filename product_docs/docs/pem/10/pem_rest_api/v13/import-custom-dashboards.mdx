`POST /dashboard/custom/import/`

Imports custom alert templates using exported dashboards.

**Body parameter**

```json
{
  "version": 1,
  "skip_overwrite": true,
  "skip_overwrite_probe": true,
  "skip_overwrite_chart": true,
  "dashboards": [
    {
      "name": "string",
      "descp": "string",
      "reference_id": "string",
      "font": "string",
      "font_size": 0,
      "level": 0,
      "is_ops": true,
      "show_title": true,
      "design_layout": [
        {}
      ],
      "c_charts": [
        {}
      ],
      "shared": [
        0
      ],
      "shared_all": true
    }
  ]
}
```

<span id="post__dashboard_custom_import_-parameters"></span>

### Parameters



| Name         | In     | Type                                                                            | Required | Description            |
| ------------ | ------ | ------------------------------------------------------------------------------- | -------- | ---------------------- |
| X-Auth-Token | header | string                                                                          | true     | Token of authorization |
| body         | body   | [CustomDashboardImportPostData](schemas.mdx#tocs_customdashboardimportpostdata) | true     | none                   |

**Example responses**

<Details summary="200 Response">

```json
[
  {
    "name": "string",
    "status": "string",
    "msg": "string"
  }
]
```

</Details>
