Released: 14 Jul 2023

New features, enhancements, bug fixes, and other changes in PEM 9.2.2 include:

| Type        | Description                                                                                      |
| ----------- | -------------------------------------------------------------------------------------------------|
| Security fix | This is a security fix for [CVE-2023-2650](https://nvd.nist.gov/vuln/detail/CVE-2023-2650) and is recommended for all the Windows users. This security fix includes updates for the Apache HTTPD bundled with the Windows installer of PEM. This patch affects the PEM server only, no need to update PEM agents.|
