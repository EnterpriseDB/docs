navTitle: Version 9.8.1

Released: 16th January 2026

This is a security patch with the following fixes. This patch is recommended for all users of PEM 9.

| Type         | Description                                                                                                                     |
|--------------|---------------------------------------------------------------------------------------------------------------------------------|
| Bug&nbsp;fix | Resolved a vulnerability (CVE-2025-2946) that exposed a possible cross-site scripting risk.                                     |
| Bug&nbsp;fix | Resolved a command-injection vulnerability allowing arbitrary command execution on Windows (CVE-2025-12763).                    |
| Bug&nbsp;fix | Resolved a vulnerability allowing unsafe deserialization and remote  code execution by an authenticated user (CVE-2024-2044).   |
| Bug&nbsp;fix | Resolved a cross-site scripting vulnerability in the /settings/store endpoint (CVE-2024-4216).                                  |
| Bug&nbsp;fix | Added UI validation to mitigate a remote code execution issue in the validation of the binary path (CVE-2024-3116).             |
| Bug&nbsp;fix | Move to urllib3 2.6.3 to fix CVE-2026-21441.                                                                                    |
| Bug&nbsp;fix | Resolved a cross-site scripting vulnerability in Manage Charts and Manage Dashboards (CVE-2026-0949)                            |
| Bug&nbsp;fix | Fixed an issue whereby TRACK/TRACE HTTP methods were not blocked in the Apache HTTPD configuration provided with PEM.           |
