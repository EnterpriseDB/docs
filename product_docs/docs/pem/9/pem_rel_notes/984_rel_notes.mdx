title: "Postgres Enterprise Manager 9.8.4 release notes"
navTitle: Version 9.8.4
---

Released: 21 May 2026

This is a security and bug-fix release. This patch is recommended for all PEM 9 users.

| Type         | Description                                                                                                                                                  |
|--------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Change       | PEM 9 now runs on Python 3.10 on RHEL 9.                                                                                                                     |
| Bug&nbsp;fix | Fixed a pickle deserialization vulnerability in the session manager (CVE-2026-7818).                                                                         |
| Bug&nbsp;fix | Fixed a symlink path traversal vulnerability in the file manager (CVE-2026-7819).                                                                            |
| Bug&nbsp;fix | Fixed vulnerabilities in urllib3 (CVE-2026-44431, CVE-2026-44432). urllib3 was updated to 2.7.0.                                                             |
| Bug&nbsp;fix | Fixed vulnerabilities in axios (CVE-2026-42033, CVE-2026-42035). axios was updated to 1.16.1.                                                                |
| Bug&nbsp;fix | Fixed vulnerabilities in Mako (CVE-2026-44307, CVE-2026-41205). Mako was updated to 1.3.12.                                                                  |
| Bug&nbsp;fix | Fixed a vulnerability in PostCSS (CVE-2026-41305). PostCSS was updated to 8.5.14.                                                                            |
| Bug&nbsp;fix | Fixed a vulnerability in ip-address (CVE-2026-42338). ip-address was updated to 10.2.0.                                                                      |
