navTitle: Version 9.8.3
---

Released: 15th April 2026

This is a security and bug-fix patch with the following fixes. This patch is recommended for all PEM 9 users.

| Type         | Description                                                                                                                                                                                     |
|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Change       | PEM now depends on the OS-provided libcurl package on RHEL 8 x86 (this was already the case for all other platforms). The deprecated `libcurl-pem` package can be safely removed after upgrade. |
| Bug&nbsp;fix | Fixed an issue affecting PEM 9.8.2 packages whereby the PEM web application would fail to run properly on RHEL 9.0-9.6. PEM 9.8.3 runs on all RHEL 9.x versions.                                                                                                                                  |
| Bug&nbsp;fix | Werkzeug was updated to 3.1.7 (Fixes: CVE-2026-27199, CVE-2025-66221).                                                                                                         |
| Bug&nbsp;fix | cryptography was updated to 46.0.7 (Fixes: CVE-2026-39892).                                                                                                                                          |
| Bug&nbsp;fix | ajv was updated to 6.14.0 (Fixes: CVE-2025-69873).                                                                                                  |
| Bug&nbsp;fix | brace-expansion was updated to 1.1.13 (Fixes: CVE-2026-33750).                                                                                                                         |
| Bug&nbsp;fix | minmatch was updated to 3.1.5 (Fixes: CVE-2026-26996, CVE-2026-27903).                                                                                                                                           |
| Bug&nbsp;fix | underscore.js was updated to 1.13.8 (Fixes: CVE-2026-27601). |                                                                                                                                           |
