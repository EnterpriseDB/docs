title: "Version 7.10"
---

Features, enhancements, bug fixes, and other changes in PEM 7.10 include:

| Highlights | Type |
| --------- | -----|
| PEM now has the capability to configure and manage a BART server using PEM console. You can perform BART operations on a managed database from PEM console such as backing up a managed database server, restoring a backup, point-in-time recovery, marking backups as Keep, Unkeep, or Obsolete. For more information, see the BART Management Guide available at: https://www.enterprisedb.com/edb-docs/p/edb-postgres-enterprise-manager | Feature |
| PostgreSQL version 12 and Advanced Server version 12 support: You can now manage, monitor, and tune a PostgreSQL version 12 or Advanced Server version 12 database. | Feature |
| PostgreSQL version 12 and Advanced Server version 12 support: You can use a PostgreSQL version 12 or Advanced Server version 12 database as the backing database for PEM server. | Feature |
| PostgreSQL version 12 and Advanced Server version 12 support: You can now use the Compound Triggers feature available with Advanced server version 12. | Feature |
| PostgreSQL version 12 and Advanced Server version 12 support: PEM also supports the Generated Columns and Planner Support function database features. | Feature |
| PEM now allows you to display a banner with custom text on the Login page. | Enhancement |
| PEM 7.10 differentiates between editable and read-only columns by using icons and tool tips in the column headers.  | Enhancement |
| You can now see the internal queries generated by View Data during Save Data operation.  | Enhancement |
| Copy with headers functionality has been introduced for copying data from Query Tool/View Data. | Enhancement |
