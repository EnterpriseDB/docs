title: "Version 7.16"

Enhancements, bug fixes, and other changes in PEM 7.16 include:

| Type        | Description | ID |
|-------------|------------ | -- |
| Enhancement | **Enhanced BART Integration**: You can now automate the cleanup of obsolete backups, allow incremental backups from a parent backup in a `tar.gz` format,  and manage the BART scanner via the PEM Console. | PEM-3615 |
| Enhancement | **Enhanced EFM Integration**: You can now perform cluster switchover and monitor cluster property values such as `missingnodes`, `minimumstandbys` and `membershipcoordinators` for a Failover Manager cluster via the PEM Console. | PEM-688 |
| Enhancement | **Auto-discovery of server clusters on Debian Platforms**: You can now quickly locate the database servers that reside on the monitored system.   | PEM-2200 |
| Enhancement | **Extended the REST API on alerts states for agents, servers, and databases**. PEM has now exposed endpoints to: current threshold alert violations; state change history on available alerts for agent, server, and database; and current state of all agents and monitored servers. | PEM-3528 |
| Enhancement | You can use the same agent-id on agent registration using `--force-registration` and regenerate the certificates.  | PEM-322 |
