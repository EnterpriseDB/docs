---
title: "Version 7.12"
---

Features, enhancements, bug fixes, and other changes in PEM 7.12 include:

| Highlights | Type |
| --------- | -----|
|  Improved BART Integration: Option to create SSH certificates to allow passwordless authentication for backup and restore operations and you can enable passwordless SSH authentication from the database server’s Properties dialog or from the Backup Restore dialog. | Enhancement |
|  Improved BART Integration: You can now set or override the archive_command of the database server’s configuration by providing inputs in the database server’s Properties dialog.  | Enhancement |
| Accessibility Improvements - Attributes have been added to provide an invisible label to the UI elements where a text label cannot be used. | Enhancement |
| Dark Theme (Beta): You can now choose to use the dark theme while accessing the PEM console. The dark theme does not work in PEM console for the background of the pie charts under monitoring dashboards and all the reports generated for Capacity Manager, Log Analysis Expert, Postgres Expert, Tuning Wizard, and so on. | Enhancement |
| Rename the context menu from 'Drop Server' to 'Remove Server'. | Bug |
| Fix issue where Drop and Disconnect connection menu points are too close to each other. | Bug |
| Don't quote bigints when copying them from the Query Tool results grid. | Bug |
| Fix issue where the Reset button does not get enabled till all the mandatory fields are provided in the dialog. | Bug |
| Ensure context menus never get hidden below the menu bar. | Bug |
| Ensure the correct "running at" pgAgent is shown when a pgAgent job is executing. | Bug |
| Fix an issue in the Query Tool where shortcut keys could be ignored following a query error. | Bug |
| Fix the toggle case shortcut key combination. | Bug |
| Ensure comments are shown in reverse engineered SQL for table partitions. | Bug |
| Handle NULL values appropriately when sorting backgrid tables. | Bug |
| Give appropriate error messages when the user tries to use a blank master password. | Bug |
| Remove arbitrary (and incorrect) requirement that composite types must have at least two members. | Bug |
| Ensure compression level is passed to pg_dump when backing up in directory format. | Bug |
| Ensure the number of jobs can be specified when backing up in directory format. | Bug |
| Remove the sorting of table headers with no labels. | Bug |
| Suppress Enter key presses in Alertify dialogues when the come from Select2 controls to allow item selection with Enter. | Bug |
| Ensure that units are respected when sorting by file size in the File dialog. | Bug |
| Updated documentation for default privileges for the database objects to clarify more on the grantor. | Bug |
| Ensure all messages are retained in the Query Tool from long running queries. | Bug |
| Updated documentation for the delete row button that only strikeout the row instead of deleting it. | Bug |
| Fix an issue where the wrong type is displayed when changing the datatype from timestamp with time-zone to timestamp without time zone. | Bug |
| Updated documentation for the query tool toolbar buttons. | Bug |
| Ensure that the superuser should be able to create database, as the superuser overrides all the access restrictions. | Bug |
| Ensure that the superuser should be able to create role, as the superuser overrides all the access restrictions. | Bug |
| Fix reverse engineering SQL issue for partitions when specifying digits as comments. | Bug |
| Fix an issue where escape key not working to close the open/save file dialog. | Bug |
| Fix an issue where keyboard shortcut for context menu is not working when using Firefox on CentOS7. | Bug |
| Shown some text on process watcher till the initial logs are loaded. | Bug |
| Fix main window tab navigation accessibility issue. | Bug |
| Fix XSS issue in explain and explain analyze for table and type which contain HTML. | Bug |
| Fix an issue where PEM unable to retrieve table node if the trigger is already disabled and the user clicks on Enable All. | Bug |
| Fix reverse engineering SQL issue for triggers when passed a single argument to trigger function. | Bug |
| Fix an issue where length and precision are not removed from table/column dialog. | Bug |
| Fix an issue where the Interval data type is not displayed in the properties dialog of table/column. | Bug |
| Fix 'Could not find the object on the server.' error while refreshing the check constraint. | Bug |
| Fix reverse engineering SQL issue where when clause is not visible for PG/EPAS 12. | Bug |
| Added statistics and storage information in reverse engineering SQL of table/column. | Bug |
| Fix an issue where the inherited table name with quotes did not escape correctly. | Bug |
| Fix an issue where context menu is open along with submenu and the focus is not on context menu or submenu. | Bug |
