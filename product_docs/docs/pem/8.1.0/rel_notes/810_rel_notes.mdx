---
title: "What's New in Version 8.1.0"
---

Enhancements and and updates in 8.1.0 include:

-   Ability to monitor a BDR (Bi-Replication Replication) server using the PEM console.
-   Support for Kerberos (GSSAPI) authentication for the PEM Server on Linux Platforms Web Console.
-   Support of EDB Postgres Extended (formerly known as 2ndQPostgres) for Administration and Monitoring.
-   Added Entity-Relationship Diagram (ERD) design tool to create ER diagrams of the databases.
-   Support for adding Publishers and Subscribers of Logical Replication using the PEM Console.
-   Added Quick Search option to locate features and relevant help articles quickly.
-   Added ability to configure the parameters through the following PEM tools without restarting the database server:
    -   Log Manager
    -   Audit Manager
    -   Tuning Wizard
