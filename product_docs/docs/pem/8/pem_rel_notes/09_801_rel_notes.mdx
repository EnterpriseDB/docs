Enhancements, bug fixes, and other changes in PEM 8.0.1 include:

| ID | Highlight | Type |
| -- |--------- | -----|
| Allowing the dash (-) character in the superuser name that is provided during configuration. | Enhancement |
| Gracefully closing the operating system resources during batch probe and command execution to avoid errors. | Enhancement |
| Improved support for Unicode string handling in pemAgent. | Enhancement |
| Option to hide or unhide the Dashboard, Statistics, Dependents, and Dependencies tabs. | Enhancement |
| Improve the explain plan details by showing popup instead of tooltip on clicking of the specified node. | Enhancement |
| Added appropriate help message and a placeholder for letting users know about the account password expiry for Login/Group Role. | Enhancement |
| Added informative message when there is no difference found for schema diff. | Enhancement |
| Added 'Count Rows' option to the partition sub tables. | Enhancement |
| Dashboard refresh not displaying BART backups. [Support Ticket # 1046037] | Bug Fix |
| Bundle pgaevent.dll in agent Windows installer to suppress event message log error. [Support Ticket # 1106021] | Bug Fix |

| Pressing the back button shows another instance of the main page inside of the Query Tool tab. | Bug Fix |
