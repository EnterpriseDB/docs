---
title: "PEM v8.1"
---

Release date: 2021-06-09

## Features

PEM-795 - Configure Audit manager without restarting the monitored server. \[Support ticket # 649078]  
PEM-1465 - Added ability to configure Tuning wizard without restarting the monitored server.  
PEM-3559 - Added support for Kerberos (GSSAPI) authentication for the PEM Server (Web Console) \[Support ticket #1044678]  
PEM-3829 - Added ability to configure Log manager without restarting the monitored server.  
PEM-3893 - Added capability to monitor BDR.  
[Issue #1591](https://redmine.postgresql.org/issues/1591) - Added Grant Wizard option under Package node.  
[Issue #4064](https://redmine.postgresql.org/issues/4064) - Added window maximize/restore functionality for properties dialog.  
[Issue #5404](https://redmine.postgresql.org/issues/5404) - Show the login roles that are members of a group role be shown when examining a group role.  
[Issue #5912](https://redmine.postgresql.org/issues/5912) - Added support for Logical Replication.  
[Issue #5954](https://redmine.postgresql.org/issues/5954) - Added support to set auto width of columns by content size in the data output window.  
[Issue #6148](https://redmine.postgresql.org/issues/6148) - Added Quick Search functionality for menu items and help articles.  
[Issue #6153](https://redmine.postgresql.org/issues/6153) - Added publication and subscription support in Schema Diff.  
[Issue #6395](https://redmine.postgresql.org/issues/6395) - Added support to rotate the log file on the basis of size and age.  
[Issue #6397](https://redmine.postgresql.org/issues/6397) - Added "IF NOT EXISTS" clause while creating tables and partition tables which is convenient while using the ERD tool.

## Housekeeping

PEM-3974 - Corrected the RestAPI document for probe history.  
[Issue #3976](https://redmine.postgresql.org/issues/3976) - Use schema qualification while accessing the catalog objects.  
[Issue #5319](https://redmine.postgresql.org/issues/5319) - Improve code coverage and API test cases for Server module.  
[Issue #6176](https://redmine.postgresql.org/issues/6176) - Make the 'Save Data Changes' icon to be more intuitive.  
[Issue #6207](https://redmine.postgresql.org/issues/6207) - Updated the JS dependencies to the latest.
