navTitle: Version 4.3

Released: 18 Dec 2021

Enhancements, bug fixes, and other changes in EFM 4.3 include:

| Type | Description |
| ---- |------------ |
| Security Fix | Failover Manager has been upgraded to use log4j version 2.17.0 to address security vulnerabilities CVE-2021-44228, CVE-2021-45046, and CVE-2021-45105. As CVE-2021-44228 is considered a critical vulnerability with a CVSS score of 10.0 this update should be applied as soon as possible. |
