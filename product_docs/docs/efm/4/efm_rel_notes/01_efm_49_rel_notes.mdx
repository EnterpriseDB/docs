---
title: "Version 4.9"
---

Enhancements, bug fixes, and other changes in EFM 4.9 include:

| Type | Description |
| ---- |------------ |
| Enhancement | Failover Manager has been upgraded to use PostgreSQL JDBC Driver version 42.7.2. |
| Bug Fix | Handle a rare case of a thread being interrupted while stopping the primary database during a switchover. [Support ticket: #102362] |
| Bug Fix | Fixed an issue with improper property settings that could prevent the startup process from exiting. |
