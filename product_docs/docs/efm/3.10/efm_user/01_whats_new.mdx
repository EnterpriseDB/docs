The following changes have been made to EDB Postgres to create version 3.10:

  - Support for the physical replication slot feature of Postgres
  - Support for NAT addresses
  - Staggering standbys for restarts
  - Introduction of a new command to check the status of a local node
  - Replace `trigger_file` with `pg_ctl` utility for standby promotion
