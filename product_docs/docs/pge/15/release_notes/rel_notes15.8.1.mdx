Released: 22 Aug 2024

New features, enhancements, bug fixes, and other changes in EDB Postgres Extended Server 15.8.1 include:

| Type    | Description                                                                                                                                                                                                                                                                                                                                                                   | Ticket  |
|---------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------|
| Bug&nbspfix | A previous release introduced a new `in_create` field to the LogicalDecodingContext structure and changed its memory layout. This means that any code that expects the old layout will no longer be compatible. The release fixed the issue to ensure that the layout of previous database server versions is still compatible with the creation of a logical replication slot. |  |
