Released: 26 Feb 2026
EDB Postgres Extended Server 18.3.0 includes the following enhancements and bug fixes:

| Type              | Description                                                                                                                                                                                                                                                                                                                                                                                                                                     | Addresses |
|-------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------|
| Security&nbsp;fix | This release includes the security fixes for [CVE-2026-2003](https://www.postgresql.org/support/security/CVE-2026-2003/), [CVE-2026-2004](https://www.postgresql.org/support/security/CVE-2026-2004), [CVE-2026-2005](https://www.postgresql.org/support/security/CVE-2026-2005/), [CVE-2026-2006](https://www.postgresql.org/support/security/CVE-2026-2006/) and [CVE-2026-2007](https://www.postgresql.org/support/security/CVE-2026-2007/). |           |
| Bug&nbsp;fix      | Fixed an issue with `password_profiles` and `pg_upgrade` that was causing failure while upgrading from PostgreSQL 18 to Postgres Extended Server 18.                                                                                                                                                                                                                                                                                            |           |
