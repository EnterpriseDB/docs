---
title: "Removing a password profile"
description: "How to use the PG_DROP_PROFILE api to remove a password profile"
---

The `PG_DROP_PROFILE` api removes a password profile for password management.

## Synopsis

```sql
PASSWORD_PROFILE.PG_DROP_PROFILE('profilename',
                                 missing_ok)
```
## Description

`PG_DROP_PROFILE` removes the specified password profile for password management.

To use `PG_DROP_PROFILE`, you must not have any users assigned to the profile.

## Parameters

- `profilename` &mdash; Name of the password profile to remove.

- `missing_ok` &mdash; If set to true, the api does not raise an error if the specified profile does not exist. If set to false, an error is raised when the specified profile does not exist.
