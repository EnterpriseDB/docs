---
title: "UTL_TCP"
---

The `UTL_TCP` package provides a way to access TCP/IP network services. EDB Postgres Advanced Server supports the following functions and procedures:

| Function/Procedure                                                                                                                                                              | Return Type  | Description                                                                                                                                                         |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `OPEN_CONNECTION(remote_host, remote_port, local_host, local_port, in_buffer_size, out_buffer_size, charset, newline, tx_timeout, wallet_path, wallet_password, ssl_cert_path)` | `CONNECTION` | Use this function to establish a connection to a remote host and port with optional parameters for buffer size, character set, and SSL/Wallet security credentials. |
| `CLOSE_CONNECTION(c)`                                                                                                                                                           |              | Use this procedure to terminate a specific active connection and update the connection object.                                                                                          |
| `CLOSE_ALL_CONNECTIONS()`                                                                                                                                                       |              | Use this procedure to close every active connection currently managed by the session.                                                                                                    |
| `WRITE_TEXT(c, data, len)`                                                                                                                                                      | `INTEGER`    | Use this function to send a specified amount of text data through the connection.                                                                                                       |
| `WRITE_LINE(c, data)`                                                                                                                                                           | `INTEGER`    | Use this function to write a line of text to the connection, typically appending a newline character.                                                                                   |
| `WRITE_RAW(c, data)`                                                                                                                                                            | `INTEGER`    | Use this function to transmit raw binary data (bytea) through the connection.                                                                                                           |
| `READ_LINE(c, data)`                                                                                                                                                            |              | Reads a single line of data from the connection and stores it in the provided VARCHAR2 variable.                                                                    |
| `SECURE_CONNECTION(c)`                                                                                                                                                          |              | Upgrades an existing connection to a secure (encrypted) state using defined security protocols.                                                                     |
