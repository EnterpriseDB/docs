---
title: "PG_SHADOW"
---

The `PG_SHADOW` view provides information about properties of all roles that are marked as `rolcanlogin` in `PG_AUTID`. It provides information specific to EDB Postgres Advanced Server in addition to the information provided in [PostgreSQL pg_shadow view](https://www.postgresql.org/docs/current/view-pg-shadow.html).

| Name         | Type                     | Description                                |
|--------------|--------------------------|--------------------------------------------|
| uselastlogin | timestamp with time zone | Specifies the time of the last user login. |
