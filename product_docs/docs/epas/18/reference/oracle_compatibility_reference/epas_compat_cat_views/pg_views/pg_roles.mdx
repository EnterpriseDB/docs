---
title: "PG_ROLES"
---

The `PG_ROLES` view provides information about the database roles. It provides information specific to EDB Postgres Advanced Server in addition to the information provided in [PostgreSQL pg_roles view](https://www.postgresql.org/docs/current/view-pg-roles.html).

| Name         | Type                     | Description                                |
|--------------|--------------------------|--------------------------------------------|
| rollastlogin | timestamp with time zone | Specifies the time of the last role login. |
