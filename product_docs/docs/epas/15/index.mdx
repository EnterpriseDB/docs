EDB is the only worldwide company to deliver innovative and low-cost, open-source-derived database solutions with commercial quality, ease of use, compatibility, scalability, and performance for small or large-scale enterprises. EDB Postgres Advanced Server adds extended functionality to open-source PostgreSQL, including:
- database administration
- enhanced SQL capabilities
- database and application security
- performance monitoring and analysis
- application development utilities. 

When you launch EDB Postgres Advanced Server in Oracle compatibility mode you get additional features which facilitate Oracle to Postgres migrations.

EDB also makes available a full suite of tools and utilities that helps you monitor and manage your EDB Postgres Advanced Server deployment.
