EDB Postgres Advanced Server supports installations on Linux and Windows platforms. 

To determine the platform support for EDB Postgres Advanced Server you can either refer to the platform support for EDB Postgres Advanced Server on the [Platform Compatibility page](https://www.enterprisedb.com/platform-compatibility#epas) on the EDB web site or refer to [Installing EDB Postgres Advanced Server](../installing).
