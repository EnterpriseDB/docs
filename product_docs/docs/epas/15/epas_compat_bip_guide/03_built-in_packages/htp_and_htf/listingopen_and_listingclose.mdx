title: LISTINGOPEN and LISTINGCLOSE
---

This generates the `<LISTING>` and `</LISTING>` tags, which mark a section of fixed-width text in the body of an HTML page.

## Syntax

The following is the syntax for HTP:
```
htp.listingOpen;

htp.listingClose;
```
