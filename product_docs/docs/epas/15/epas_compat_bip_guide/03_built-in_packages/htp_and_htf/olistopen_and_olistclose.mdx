---
title: OLISTOPEN and OLISTCLOSE
---

This generates the `<OL>` and `</OL>` tags, which define an ordered list. An ordered list presents a list of numbered items

## Syntax

The following is the syntax for HTP:
```
htp.olistOpen (
   cclear         in       varchar2   DEFAULT NULL
   cwrap          in       varchar2   DEFAULT NULL
   cattributes    in       varchar2   DEFAULT NULL);

htp.olistClose;
```

## Parameters

| Parameter | Function |
| --------- | -------- |
| cclear | The value for the CLEAR attribute. |
| cwrap | The value for the WRAP attribute. |
| cattributes | Other attributes to be included as-is in the tag. |
