title: BIG
---

This generates the `<BIG>` and `</BIG>` tags, which increases font size

## Syntax 

The following is the syntax for HTP:
```
htp.big(
ctext       IN VARCHAR2 CHARACTER SET ANY_CS,
cattributes IN VARCHAR2 DEFAULT NULL);
```

The following is the syntax for HTF:
```
htf.big(ctext, cattributes) return varchar2;
```

## Parameters 

| Parameter | Function |
| --------- | -------- |
| ctext | The text that goes between the tags. |
| cattributes | Other attributes to be included as-is in the tag. |
