title: PARAM
---

This generates the `<PARAM>` tag, which specifies parameter values for Java applets.

## Syntax

The following is the syntax for HTP:
```
htp.param(
   cname          in       varchar2
   cvalue         in       varchar2);
```

The following is the syntax for HTF:
```

```

## Parameters

| Parameter | Function |
| --------- | -------- |
| cname | The value for the NAME attribute. |
| cvalue | The value for the VALUE attribute. |
