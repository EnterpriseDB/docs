title: META
---

This generates the `<META>` tag, which embeds meta-information about the document and also specifies values for HTTP headers.

## Syntax

The following is the syntax for HTP:
```
htp.meta (
   chttp_equiv    in       varchar2
   cname          in       varchar2
   ccontent       in       varchar2);
```

The following is the syntax for HTF:
```

```

## Parameters

| Parameter | Function |
| --------- | -------- |
| chttp_equiv | The value for the HTTP-EQUIV attribute. |
| cname | The value for the NAME attribute. |
| ccontent | The value for the CONTENT attribute. |
