This function and procedure generates the `<WBR>` tag, which inserts a soft line break in text marked as NOBR.

## Syntax

The following is the syntax for HTP:

```
HTP.WBR;
```

The following is the syntax for HTF:

```
HTF.WBR RETURN WBR;
```
