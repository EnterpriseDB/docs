<div id="limitations" class="registered_link"></div>

The following limitation applies to the EDB Postgres Advanced Server:

- The LLVM JIT package is supported on RHEL 7.x and SLES. LLVM JIT is not supported on RHEL 6.x and PPC-LE 64 (running RHEL or CentOS 7.x).
