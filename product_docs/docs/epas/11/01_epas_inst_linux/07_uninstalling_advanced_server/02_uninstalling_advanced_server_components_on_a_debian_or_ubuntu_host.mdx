```
title: "Uninstalling Advanced Server Components on a Debian or Ubuntu Host"
```

<div id="uninstalling_advanced_server_components_on_a_debian_or_ubuntu_host" class="registered_link"></div>

1. To uninstall Advanced Server, invoke the following command:

   ​			`apt-get remove edb-as11-server*`

   Please note: the configuration files and data directory remains intact.

   

2. To uninstall Advanced Server, configuration files, and data directory, invoke the following command:

   ​			`apt-get purge edb-as11-server*`
