The `RUN` procedure forces the job to run even if its state is broken.

```sql
RUN(<job> BINARY_INTEGER)
```

## Parameter

`job`

 Identifier of the job to run.

## Examples

Force a job to run.

```sql
BEGIN
    DBMS_JOB.RUN(104);
END;
```
