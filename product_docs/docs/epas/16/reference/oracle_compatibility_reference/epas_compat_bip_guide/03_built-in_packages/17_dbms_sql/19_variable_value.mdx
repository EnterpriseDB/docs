The `VARIABLE_VALUE` procedure returns the value of a named value for a given cursor. It returns the values of bind variables inside PL/SQL blocks or DML statements with a returning clause.

## Parameters

`c`

ID of the cursor from which to get values.

`name`

Name of the variable from which data is retrieved.

`value`

- single row: Value of the specified variable
- array: Local variable that has been declared a <table_type>. For bulk operations, value is an OUT NOCOPY parameter.

## Examples

The following anonymous block uses bind variables to insert a row into the `emp` table.

```sql
/* Create Table */
CREATE TABLE zaiko (product_id numeric,zaiko_count numeric);

/*  Create a procedure with IN/OUT parameters  */

CREATE OR REPLACE PROCEDURE demo3 ( 
  pi_prd     in    int , 
  pi_cnt     in    int ,
  po_rtn    out   int) 
AS
BEGIN
   INSERT INTO zaiko VALUES (pi_prd,pi_cnt);
   po_rtn := 100; /* Set any value to OUT parameter */
END;
/

SET SERVEROUTPUT ON

DECLARE
    curid    number;
    v_sql    varchar2(200);
    v_rtn    int;
    var1     int;
    var2     int;
    var3     int;
BEGIN
    var1 := 100;
    var2 := 200;

    v_sql := 'BEGIN demo3(:var1,:var2,:var3);END;' ;

    curid := DBMS_SQL.OPEN_CURSOR;
    DBMS_SQL.PARSE(curid, v_sql, DBMS_SQL.V7);

    DBMS_SQL.BIND_VARIABLE(curid,':var1',var1);
    DBMS_SQL.BIND_VARIABLE(curid,':var2',var2);
    DBMS_SQL.BIND_VARIABLE(curid,':var3',var3);

    v_rtn := DBMS_SQL.EXECUTE(curid);
    DBMS_SQL.VARIABLE_VALUE(curid,':var3',var3);
    DBMS_SQL.CLOSE_CURSOR(curid);

    DBMS_OUTPUT.ENABLE;
    DBMS_OUTPUT.PUT_LINE(var3);
END;
/
```
