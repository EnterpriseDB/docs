---
title: "TO_CURSOR_NUMBER"
---

The `TO_CURSOR_NUMBER` function transforms a ref cursor into a SQL cursor number.

This function is useful to retrieve the metadeta (columns, column types, and so forth) from a ref cursor, when you have a SYS_REFCURSOR variable with an unknown return structure.

```sql
TO_CURSOR_NUMBER (<rc> IN OUT SYS_REFCURSOR)
    RETURN INTEGER;
```

## Parameters

`rc`

 The ref cursor to be transformed into a cursor number.
