Released: 13 Nov 2025

EDB Postgres Advanced Server 16.11.0 includes the following enhancements and bug fixes:

| Type           | Description                                                                                                                                                | Addresses&nbsp; |
|----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------|
| Upstream merge | Merged with community PostgreSQL 16.11. See the [PostgreSQL 16.11 Release Notes](https://www.postgresql.org/docs/release/16.11/) for more information.     |                 |
| Bug&nbsp;fix   | Fixed an issue with pg_dumpall where dependent roles could be dumped before their resource groups, which caused restoration errors.                        | #51160          |
| Bug&nbsp;fix   | Fixed an issue in ECPG that could lead to invalid pointer dereferencing when performing Oracle-style dynamic SQL fetches.                                  | #50148          |
| Bug&nbsp;fix   | Fixed an issue preventing COPY TO from correctly omitting generated columns on tables with an enabled data redaction policy.                               | #44853          |
| Bug&nbsp;fix   | Fixed an issue that caused CALL statements to fail when invoking functions returning a TABLE type.                                                         | #49217          |
| Bug&nbsp;fix   | Fixed an issue causing the drop operation for the dbms_job extension to fail as a result of internal dependency conflicts.                                 |                 |
| Bug&nbsp;fix   | Fixed an issue where enabling Transparent Data Encryption (TDE) caused a server startup crash on Windows when the pg_stat_statements extension was in use. | #52100          |
