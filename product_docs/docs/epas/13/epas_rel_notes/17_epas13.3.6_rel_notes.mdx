title: Version 13.3.6

New features, enhancements, bug fixes, and other changes in EDB Postgres Advanced Server 13.3.6 include:

| Type        | Description      | 
| ----------- | ---------------- | 
| Bug fix     | DB-1183 Prevent some uses of COMMIT and ROLLBACK in stored procedures that were unsafe and could cause crashes (#1234722) | 
| Bug fix     | DB-1154 Prevent index_advisor from interfering with materialized view creation (#1189859) | 
| Bug fix     | DB-1139 Support password redaction in edb_filter_log with the extended protocol, specially used with connectors (#1234131) | 
| Bug fix     | DB-1129 Correct QUEUE object handling in EVENT TRIGGER | 
| Bug fix     | DB-1129 Correct REDACTION COLUMN object handling in EVENT TRIGGER | 
| Bug fix     | DB-1116 Correct mishandling of CREATE and DROP views with autonomous transactions | 
| Bug fix     | DB-1084 Fix pg_upgrade to not fail when a custom configuration file directory is used (#1200560) | 
| Bug fix     | DB-1063 Refrain from dropping trigger on parent table through partitioning dependency (#1187215) | 
| Bug fix     | DB-1061 dblink_ora: Free temporary memory to avoid PGA memory accumulation and exceeds errors (#1129386) | 
| Bug fix     | DB-1042 edb_dblink_oci: Fix possible server crash with partition-wish join push-down code path | 
| Bug fix     | DB-871 Fix SPL plancache reference leak warning with parallel query | 
| Bug fix     | DB-826 Fix incorrect error message in edbldr (#1104048) | 
| Bug fix     | DB-731 Allow partition creation with ROWIDS option |
