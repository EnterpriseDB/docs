Released: 13 Nov 2025

EDB Postgres Advanced Server 13.23.29 includes the following enhancements and bug fixes:

| Type           | Description                                                                                                                                            | Addresses&nbsp; |
|----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------|
| Upstream merge | Merged with community PostgreSQL 13.23. See the [PostgreSQL 13.23 Release Notes](https://www.postgresql.org/docs/release/13.23/) for more information. |                 |
| Bug&nbsp;fix   | Fixed an issue with pg_dumpall where dependent roles could be dumped before their resource groups, which caused restoration errors.                    | #51160          |
| Bug&nbsp;fix   | Fixed an issue in ECPG that could lead to invalid pointer dereferencing when performing Oracle-style dynamic SQL fetches.                              | #50148          |
| Bug&nbsp;fix   | Fixed an issue preventing COPY TO from correctly omitting generated columns on tables with an enabled data redaction policy.                           | #44853          |
