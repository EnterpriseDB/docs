EDB Postgres Advanced Server 13.3.7 includes the following bug fixes:

| Description      | ID |
| ---------------- | -- |
| Fix `pg_upgrade` to allow the system catalog composite type used in user tables. | DB-1237 |
| Fix possible misbehavior when aborting an autonomous transaction and also fix interaction of autonomous transactions with `edb_stmt_level_tx=on`. | DB&#8209;1034 |
