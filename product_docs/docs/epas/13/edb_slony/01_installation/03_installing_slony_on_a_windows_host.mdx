---
title: "Installing Slony Replication on a Windows Host"
---

<div id="using_the_graphical_installer" class="registered_link"></div>


Graphical installers for Slony Replication are available via StackBuilder Plus (for Advanced Server hosts) or Stack Builder (on PostgreSQL hosts).

The following steps walk you through installing Slony Replication via StackBuilder Plus. You can access StackBuilder Plus through your Windows start menu.

1.  Open StackBuilder Plus and select your Advanced Server installation from the drop-down list on the `Welcome` window. Click `Next` to continue to the application selection page.

<figure><img src="../images/slony_welcome1.png" class="align-center" alt="The StackBuilder Plus Welcome window" /><figcaption aria-hidden="true"><em>The StackBuilder Plus Welcome window</em></figcaption></figure>

1.  Expand the `Replication Solutions` node, and check the box next to the Slony Replication version. Click `Next` to continue.

<figure><img src="../images/Slony_selection_window1.png" class="align-center" alt="The Slony Selection window" /><figcaption aria-hidden="true"><em>The Slony Selection window</em></figcaption></figure>

1.  The selected packages and the default download directory are displayed. Change the download directory location if required. Click `Next`.

<figure><img src="../images/slony_selected_packages3.png" class="align-center" alt="The Slony Selected Packages Window" /><figcaption aria-hidden="true"><em>The Slony Selected Packages Window</em></figcaption></figure>

1.  Once you have downloaded the installation files, a confirmation message is displayed. Click `Next` to start the Slony Replication installation.

<figure><img src="../images/sbp_start_installation.png" class="align-center" alt="Installing Slony png" /><figcaption aria-hidden="true"><em>Installing Slony png</em></figcaption></figure>

1.  Select an installation language and click `OK`.

<figure><img src="../images/slony_language_selection4.png" class="align-center" alt="The Slony language selection window" /><figcaption aria-hidden="true"><em>The Slony language selection window</em></figcaption></figure>

1.  The installer welcomes you to the setup wizard. Click `Next`.

<figure><img src="../images/slony_setup5.png" class="align-center" alt="The Slony Welcome window" /><figcaption aria-hidden="true"><em>The Slony Welcome window</em></figcaption></figure>

1.  Use the `Installation Directory` field to specify the directory in which you wish to install Slony Replication. Then, click `Next` to continue.

<figure><img src="../images/slony_installation_directory.png" class="align-center" alt="The Installation Directory window" /><figcaption aria-hidden="true"><em>The Installation Directory window</em></figcaption></figure>

1.  The `Ready to Install` window notifies you when the installer has all of the information needed to install Slony Replication on your system. Click `Next` to install Slony Replication. Progress bars inform you as the installation progresses.

<figure><img src="../images/slony_ready_to_install.png" class="align-center" alt="The Ready to Install window" /><figcaption aria-hidden="true"><em>The Ready to Install window</em></figcaption></figure>

1.  The installer notifies you when the setup wizard has completed the installation. Click `Finish` to exit the installer.

<figure><img src="../images/slony_setup_complete9.png" class="align-center" alt="The installation is complete" /><figcaption aria-hidden="true"><em>The installation is complete</em></figcaption></figure>

1.  Click `Finish` to exit the StackBuilder Plus.

<figure><img src="../images/installation_is_complete1.png" class="align-center" alt="Exit StackBuilder Plus" /><figcaption aria-hidden="true"><em>Exit StackBuilder Plus</em></figcaption></figure>
