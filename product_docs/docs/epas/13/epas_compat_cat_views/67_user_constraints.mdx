---
title: "USER_CONSTRAINTS"
---

<div id="user_constraints" class="registered_link"></div>

The `USER_CONSTRAINTS` view provides information about all constraints placed on tables that are owned by the current user.

<table style="width:14%;"><colgroup><col style="width: 2%" /><col style="width: 2%" /><col style="width: 8%" /></colgroup><tbody><tr class="odd"><td>Name</td><td>Type</td><td>Description</td></tr><tr class="even"><td>owner</td><td>TEXT</td><td>The name of the owner of the constraint.</td></tr><tr class="odd"><td>schema_name</td><td>TEXT</td><td>Name of the schema in which the constraint belongs.</td></tr><tr class="even"><td>constraint_name</td><td>TEXT</td><td>The name of the constraint.</td></tr><tr class="odd"><td><p>constraint_type</p></td><td><p>TEXT</p></td><td><p>The constraint type. Possible values are:</p><blockquote><p>C – check constraint</p><p>F – foreign key constraint</p><p>P – primary key constraint</p><p>U – unique key constraint</p><p>R – referential integrity constraint</p><p>V – constraint on a view</p><p>O – with read-only, on a view</p></blockquote></td></tr><tr class="even"><td>table_name</td><td>TEXT</td><td>Name of the table to which the constraint belongs.</td></tr><tr class="odd"><td>search_condition</td><td>TEXT</td><td>Search condition that applies to a check constraint.</td></tr><tr class="even"><td>r_owner</td><td>TEXT</td><td>Owner of a table referenced by a referential constraint.</td></tr><tr class="odd"><td>r_constraint_name</td><td>TEXT</td><td>Name of the constraint definition for a referenced table.</td></tr><tr class="even"><td><p>delete_rule</p></td><td><p>TEXT</p></td><td><p>The delete rule for a referential constraint. Possible values are:</p><blockquote><p>C – cascade</p><p>R – restrict</p><p>N – no action</p></blockquote></td></tr><tr class="odd"><td>deferrable</td><td>BOOLEAN</td><td>Specified if the constraint is deferrable (T or F).</td></tr><tr class="even"><td>deferred</td><td>BOOLEAN</td><td>Specifies if the constraint has been deferred (T or F).</td></tr><tr class="odd"><td>index_owner</td><td>TEXT</td><td>User name of the index owner.</td></tr><tr class="even"><td>index_name</td><td>TEXT</td><td>The name of the index.</td></tr><tr class="odd"><td>constraint_def</td><td>TEXT</td><td>The definition of the constraint.</td></tr></tbody></table>
