---
title: "ALL_SYNONYMS"
---

<div id="all_synonyms" class="registered_link"></div>


The `ALL_SYNONYMS` view provides information on all synonyms that may be referenced by the current user.

|                     |      |                                                      |
|---------------------|------|------------------------------------------------------|
| Name                | Type | Description                                          |
| owner               | TEXT | User name of the synonym’s owner.                    |
| schema\_name        | TEXT | The name of the schema in which the synonym resides. |
| synonym\_name       | TEXT | Name of the synonym.                                 |
| table\_owner        | TEXT | User name of the object’s owner.                     |
| table\_schema\_name | TEXT | The name of the schema in which the table resides.   |
| table\_name         | TEXT | The name of the object that the synonym refers to.   |
| db\_link            | TEXT | The name of any associated database link.            |
