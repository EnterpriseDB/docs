<div id="dba_types" class="registered_link"></div>

The `DBA_TYPES` view provides information about all object types in the database.

<table><tbody><tr class="odd"><td>Name</td><td>Type</td><td>Description</td></tr><tr class="even"><td>owner</td><td>TEXT</td><td>The owner of the object type.</td></tr><tr class="odd"><td>schema_name</td><td>TEXT</td><td>The name of the schema in which the type is defined.</td></tr><tr class="even"><td>type_name</td><td>TEXT</td><td>The name of the type.</td></tr><tr class="odd"><td>type_oid</td><td>OID</td><td>The object identifier (OID) of the type.</td></tr><tr class="even"><td><p>typecode</p></td><td><p>TEXT</p></td><td><p>The typecode of the type. Possible values are:</p><blockquote><p>OBJECT</p><p>COLLECTION</p><p>OTHER</p></blockquote></td></tr><tr class="odd"><td>attributes</td><td>INTEGER</td><td>The number of attributes in the type.</td></tr></tbody></table>
