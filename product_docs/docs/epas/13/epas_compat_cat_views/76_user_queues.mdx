title: "USER_QUEUES"
---

<div id="user_queues" class="registered_link"></div>

The `USER_QUEUES` view provides information about any queue on which the current user has usage privileges.

<table><tbody><tr class="odd"><td>Name</td><td>Type</td><td>Description</td></tr><tr class="even"><td>name</td><td>TEXT</td><td>The name of the queue.</td></tr><tr class="odd"><td>queue_table</td><td>TEXT</td><td>The name of the queue table in which the queue resides.</td></tr><tr class="even"><td>qid</td><td>OID</td><td>The system-assigned object ID of the queue.</td></tr><tr class="odd"><td><p>queue_type</p></td><td><p>CHARACTER</p><p>VARYING</p></td><td><p>The queue type; may be EXCEPTION_QUEUE, NON_PERSISTENT_QUEUE, or NORMAL_QUEUE.</p></td></tr><tr class="even"><td>max_retries</td><td>NUMERIC</td><td>The maximum number of dequeue attempts.</td></tr><tr class="odd"><td>retrydelay</td><td>NUMERIC</td><td>The maximum time allowed between retries.</td></tr><tr class="even"><td><p>enqueue_enabled</p></td><td><p>CHARACTER</p><p>VARYING</p></td><td><p>YES if the queue allows enqueuing; NO if the queue does not.</p></td></tr><tr class="odd"><td><p>dequeue_enabled</p></td><td><p>CHARACTER</p><p>VARYING</p></td><td><p>YES if the queue allows dequeuing; NO if the queue does not.</p></td></tr><tr class="even"><td><p>retention</p></td><td><p>CHARACTER</p><p>VARYING</p></td><td><p>The number of seconds that a processed message is retained in the queue.</p></td></tr><tr class="odd"><td><p>user_comment</p></td><td><p>CHARACTER</p><p>VARYING</p></td><td><p>A user-specified comment.</p></td></tr><tr class="even"><td><p>network_name</p></td><td><p>CHARACTER</p><p>VARYING</p></td><td><p>The name of the network on which the queue resides.</p></td></tr><tr class="odd"><td><p>sharded</p></td><td><p>CHARACTER</p><p>VARYING</p></td><td><p>YES if the queue resides on a sharded network; NO if the queue does not.</p></td></tr></tbody></table>
