The following features have been added to EDB\*Plus version 36:

-   Advanced Server supports use of Internet Protocol version 6 (IPv6) to log into EDB\*Plus. For more information, see <span class="title-ref">Starting EDB\*Plus &lt;starting\_edb\_plus&gt;</span>.
-   Advanced Server supports use of a Secure Sockets Layer (SSL) connection from EDB\*Plus to the Advanced Server database. For more information, see <span class="title-ref">Using a Secure Sockets Layer (SSL) Connection &lt;using\_edb\_plus\_with\_ssl&gt;</span>.
