title: "Conflicting Hints"
---

<div id="conflicting_hints" class="registered_link"></div>

If a command includes two or more conflicting hints, the server will ignore the contradictory hints. The following table lists hints that are contradictory to each other.

<table><tbody><tr class="odd"><td>Hint</td><td>Conflicting Hint</td></tr><tr class="even"><td><code>ALL_ROWS</code></td><td><blockquote><p><code>FIRST_ROWS</code> - all formats</p></blockquote></td></tr><tr class="odd"><td><p><code>FULL(table)</code></p></td><td><p><code>INDEX(table [ index ])</code></p><p><code>PARALLEL(table [ degree ])</code></p></td></tr><tr class="even"><td><p><code>INDEX(table)</code></p></td><td><p><code>FULL(table)</code></p><p><code>NO_INDEX(table)</code></p><p><code>PARALLEL(table [ degree ])</code></p></td></tr><tr class="odd"><td><p><code>INDEX(table index)</code></p></td><td><p><code>FULL(table)</code></p><p><code>NO_INDEX(table index)</code></p><p><code>PARALLEL(table [ degree ])</code></p></td></tr><tr class="even"><td><p><code>PARALLEL(table [ degree ])</code></p></td><td><p><code>FULL(table)</code></p><p><code>INDEX(table)</code></p><p><code>NO_PARALLEL(table)</code></p></td></tr><tr class="odd"><td><code>USE_HASH(table)</code></td><td><code>NO_USE_HASH(table)</code></td></tr><tr class="even"><td><code>USE_MERGE(table)</code></td><td><code>NO_USE_MERGE(table)</code></td></tr><tr class="odd"><td><code>USE_NL(table)</code></td><td><code>NO_USE_NL(table)</code></td></tr></tbody></table>
