<div id="if_statement" class="registered_link"></div>

IF statements let you execute commands based on certain conditions. SPL has four forms of `IF:`

-   `IF ... THEN`
-   `IF ... THEN ... ELSE`
-   `IF ... THEN ... ELSE IF`
-   `IF ... THEN ... ELSIF ... THEN ... ELSE`
