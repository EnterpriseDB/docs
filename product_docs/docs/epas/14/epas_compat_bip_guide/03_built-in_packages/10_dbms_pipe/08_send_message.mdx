title: "SEND_MESSAGE"
---

The `SEND_MESSAGE` function sends a message from the sessionâ€™s local message buffer to the specified pipe.

```text
<status> SEND_MESSAGE(<pipename> VARCHAR2 [, <timeout> INTEGER ]
   [, <maxpipesize> INTEGER ])
```

## Parameters

`pipename`

 Name of the pipe.

`timeout`

 Wait time (seconds). Default is 86400000 (1000 days).

`maxpipesize`

 Maximum capacity of the pipe in bytes. Default is 8192 bytes.

`status`

 Status code returned by the operation.

 The possible status codes are:

| Status Code | Description          |
| ----------- | -------------------- |
| `0`         | Success              |
| `1`         | Time out             |
| `3`         | Function interrupted |
