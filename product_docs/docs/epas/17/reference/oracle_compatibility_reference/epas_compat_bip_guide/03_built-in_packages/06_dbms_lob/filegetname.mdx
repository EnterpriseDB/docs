---
title: "FILEGETNAME"
---

The `FILEGETNAME` procedure obtains the directory alias and filename for a BFILE locator. This procedure does not determine whether the physical file or directory actually exists.

```sql
FILEGETNAME (<file_loc> IN BFILE, <dir_alias> OUT VARCHAR2, 
   <filename> OUT VARCHAR2)
```

## Parameters

`file_loc`

 Locator for the BFILE.

 `dir_alias`

 Alias for the directory. The maximum value for the `dir_alias` buffer is 30, and the maximum for the entire pathname is 2000.

`filename`

 Name of the the BFILE.

## Example

``` sql
CREATE OR REPLACE PROCEDURE fileread IS
    fil BFILE;
BEGIN
    SELECT p2 INTO fil FROM test WHERE p1 = 20; 
    IF (dbms_lob.fileexists(fil))
    THEN
        dbms_lob.filegetname(fil, dir_alias, name);
        dbms_lob.fileopen(fil, dbms_lob.file_readonly);
        -- file operations
        dbms_output.fileclose(fil); 
    END IF;
END;
```
