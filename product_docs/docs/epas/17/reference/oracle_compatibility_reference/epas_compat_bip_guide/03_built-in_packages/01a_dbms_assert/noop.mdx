---
title: "NOOP"
---

The `NOOP` procedure returns a string without any checking.

```sql
NOOP(
  <str> VARCHAR2 CHARACTER SET ANY_CS)
RETURN VARCHAR2 CHARACTER SET str%CHARSET;

NOOP(
  <str> CLOB CHARACTER SET ANY_CS)
RETURN CLOB CHARACTER SET str%CHARSET;
```

## Parameters

`str`

 The input string.
