The `IS_OPEN` function tests whether the given cursor is open.

```sql
<status> BOOLEAN IS_OPEN(<c> NUMBER)
```

## Parameters

`c`

 Cursor ID of the cursor to test.

`status`

 Set to `TRUE` if the cursor is open, set to `FALSE` if the cursor isn't open.
