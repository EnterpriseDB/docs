---
title: "FILEEXISTS"
---

The `FILEEXISTS` procedure determines whether a BFILE exists.

```sql
FILEEXISTS(<file_loc> IN BFILE ) RETURN INTEGER
```

## Parameters

`file_loc`

Locator for the BFILE.

## Example

``` sql
CREATE OR REPLACE PROCEDURE filecheck IS
   File_loc      BFILE;
BEGIN
   /* Select the LOB: */
   SELECT p2 INTO fil FROM test WHERE p1 = 20; 
   /* See If the BFILE exists: */
   IF (DBMS_LOB.FILEEXISTS(File_loc) != 0)
   THEN
      DBMS_OUTPUT.PUT_LINE('Processing given that the BFILE exists');
   ELSE
      DBMS_OUTPUT.PUT_LINE('Processing given that the BFILE does not exist');
   END IF;
EXCEPTION
   WHEN OTHERS THEN
      DBMS_OUTPUT.PUT_LINE('Operation failed');
END;
```
