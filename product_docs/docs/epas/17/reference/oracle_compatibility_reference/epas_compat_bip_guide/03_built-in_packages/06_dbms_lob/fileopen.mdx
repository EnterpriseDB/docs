---
title: "FILEOPEN"
---

The `FILEOPEN` procedure opens a BFILE for read-only purposes. 

```sql
FILEOPEN (<file_loc> IN OUT NOCOPY BFILE, 
   <open_mode> IN BINARY_INTEGER := file_readonly)
```

## Parameters

`file_loc`

 Locator for the BFILE.

 `open_mode`

 File can only be accessed in read-only mode.

## Example

``` sql
CREATE OR REPLACE PROCEDURE fileread IS
    fil BFILE;
BEGIN
    SELECT p2 INTO fil FROM test WHERE p1 = 20; 
    IF (dbms_lob.fileexists(fil))
    THEN
        dbms_lob.filegetname(fil, dir_alias, name);
        dbms_lob.fileopen(fil, dbms_lob.file_readonly);
        -- file operations
        dbms_output.fileclose(fil); 
    END IF;
END;
```
