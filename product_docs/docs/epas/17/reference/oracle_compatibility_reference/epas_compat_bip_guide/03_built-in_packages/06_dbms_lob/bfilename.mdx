---
title: "BFILENAME"
---

The `BFILENAME` procedure returns a BFILE locator that is associated with a physical LOB binary file on the server file system.

Create the directory object and associate a BFILE value with the physical file before using them as arguments to the BFILENAME procedure.

```sql
BFILENAME (<directory>, <filename>); 
```

## Parameters

`directory`

The database object that is an alias for a full path name on the server file system where the files are actually located. This value is case sensitive. You must specify if using the case and punctuation requirements for your operation system. 

`filename`

 Name of the the file in the file system.


## Example

!!! sql
DECLARE
  src_loc     bfile := bfilename('ADVERT_DIR','ad_source_1000') ;
  dst_loc     clob  ;
  amt         number := dbms_lob.lobmaxsize;
  src_offset  number := 1 ;
  dst_offset  number := 1 ;
  lang_ctx    number := dbms_lob.default_lang_ctx;
  warning     number;
BEGIN
   SELECT p2 INTO dst_loc FROM bfiletest 
      WHERE p1=20;   /* Open the BFILEs: */
  dbms_lob.fileopen(src_loc, dbms_lob.file_readonly);
