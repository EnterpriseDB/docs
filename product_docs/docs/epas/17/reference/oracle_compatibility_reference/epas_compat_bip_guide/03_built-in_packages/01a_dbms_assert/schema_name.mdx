---
title: "SCHEMA_NAME"
---

The `SCHEMA_NAME` function determines whether the input string is an existing schema name. An error occurs if the input value is not a valid schema name. 

```sql
SCHEMA_NAME(str VARCHAR2) RETURN VARCHAR2;
```

## Parameters

`str`

 The input string. White space is trimmed from the string. Mixed case is allowed. 

## Examples

```sql
edb=# SELECT SYS.DBMS_ASSERT.SCHEMA_NAME(current_database() || '."sys"') FROM DUAL;
 schema_name 
-------------
 edb."sys"
(1 row)

edb=# SELECT SYS.DBMS_ASSERT.SCHEMA_NAME('SYS') FROM DUAL;
 schema_name 
-------------
 SYS
(1 row)
```
