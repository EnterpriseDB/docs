---
title: "Using the EDB Migration Portal"
---

<div id="mp_using_portal" class="registered_link"></div>


The Migration Portal allows you to easily migrate your database schema from Oracle to Advanced Server. You can upload schemas for assessment and get immediate feedback and suggestions. The portal allows you to download assessed DDLs for all objects and create your EDB Postgres database on-premises or in the cloud.

![*The assessment and migration process*](../images/mp_using_portal.png)

To access the migration portal:

1.  Open a browser and navigate to <https://www.enterprisedb.com/>
2.  On the EnterpriseDB home page, click `Enterprise Postgres` \> `EDB Postgres Migration Portal`.

![*Accessing the Migration Portal.*](../images/mp_using_portal_accessing.png)
