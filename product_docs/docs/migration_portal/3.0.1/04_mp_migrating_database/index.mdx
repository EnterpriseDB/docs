title: "Migrating a Database"

<div id="mp_migrating_database" class="registered_link"></div>

To migrate a database, you must complete the following steps:

1.  Perform a `Schema Extraction <mp_schema_extraction>`.
2.  Perform a `Schema Assessment <mp_schema_assessment>`.
3.  Perform a `Schema Migration <mp_schema_migration>`.
4.  Perform a `Data Migration <mp_data_migration>`.

The following sections provide detailed information about each step in the migration process.
