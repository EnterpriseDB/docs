The EDB ODBC Connector provides connectivity between EDB Postgres Advanced Server (Advanced Server) and ODBC-compliant applications.

New features, enhancements, bug fixes, and other changes in the EDB ODBC Connector `13.01.0000.02` include:

| Type        | Description                                              |
| ----------- | -------------------------------------------------------- |
| Enhancement | Support for EDB Postgres Advanced Server version 14.1.0. |
