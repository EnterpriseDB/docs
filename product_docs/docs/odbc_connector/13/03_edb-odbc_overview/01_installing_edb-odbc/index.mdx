title: "Installing the EDB ODBC Connector"

---

To install the EDB ODBC Connection, you must have credentials that allow access to the EnterpriseDB repository. To request credentials that allow you to access an EnterpriseDB repository, see the [EDB Repository Access instructions](https://www.enterprisedb.com/repository-access-request).
