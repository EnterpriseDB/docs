Backup the PostgreSQL cluster to the opened Klio server

```
klio backup run [flags]
```

### Options

```
      --enable-tier2-backup   When enabled, require the backup to be sent to tier2
  -h, --help                  help for run
  -n, --name string           The backup name
      --wait-for-wals         When enabled, wait until all the required WAL files have been archived in tier1 before declaring the backup completed. (default true)
```
