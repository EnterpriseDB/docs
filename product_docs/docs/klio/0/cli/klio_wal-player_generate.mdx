Generate a directory of WAL files

```
klio wal-player generate [output-directory] [flags]
```

### Options

```
  -h, --help           help for generate
      --length int     How many WAL files should be generated. Required.
      --wal-size int   The WAL file size in MBs. Defaults to 16 MB (default 16)
```
