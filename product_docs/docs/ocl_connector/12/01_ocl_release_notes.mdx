The EDB OCL Connector provides an API similar to the Oracle Call Interface.

New features, enhancements, bug fixes, and other changes in the EDB OCL Connector `12.1.2.1` include:

| Type        | Description                                                              |
| ----------- | -------------------------------------------------------------------------|
| Enhancement | EDB OCL Connector now provides multithreading support.                   |
| Enhancement | EDB OCL Connector now supports EDB Postgres Advanced Server 12.          |
| Enhancement | EDB OCL Connector is now also supported on Windows Server 2019 platform. |
