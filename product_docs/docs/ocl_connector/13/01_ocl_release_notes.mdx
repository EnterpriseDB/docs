The EDB OCL Connector provides an API similar to the Oracle Call Interface.

New features, enhancements, bug fixes, and other changes in the EDB OCL Connector `13.1.4.2` include:

| Type        | Description                                                                                                                                                     |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Enhancement | EDB OCL now provides support for OTL 4.0 Certification. For more information, refer to [OTL Support](04_open_client_library/08_otl_support/#otl_support).       |
| Enhancement | EDB OCL installation always uses latest libpq. For more information, refer to [libpq Cross-version Compatibility](03_libpq_compatibility/#libpq_compatibility). |
