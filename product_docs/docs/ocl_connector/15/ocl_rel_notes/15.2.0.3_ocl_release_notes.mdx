Released: 20 Jun 2023

The EDB OCL Connector provides an API similar to the Oracle Call Interface.

New features, enhancements, bug fixes, and other changes in the EDB OCL Connector 15.2.0.3 include:

|  Type   |                                                                       Description                                                                        |
| ------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Bug fix | Fixed regression whereby `tnsnamesparse()` tries to parse tnsnames.ora, but reads it from stdin (fd 0) instead of the file. [Support tickets: 93920, 93994] |
