Released: 22 May 2025

The EDB OCL Connector provides an API similar to the Oracle Call Interface.

New features, enhancements, bug fixes, and other changes in the EDB OCL Connector 17.2.0.2 include:

| Type         | Description                                                                                                                                                                        | Addresses |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------|
| Enhancement  | OCL now supports client-side prepared statements for SELECT queries, executing them as simple queries to fetch all data in a single round trip, enabling parallel query execution. |           |
| Bug&nbsp;fix | Fixed a syntax error that occurred after upgrading the EDB client to version 17.2.                                                                                                 | #47024    |
