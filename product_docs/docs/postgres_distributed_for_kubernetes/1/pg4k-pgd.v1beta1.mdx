#### Backup

Backup configures the backup of cnp-pgd nodes

*Appears in:*

-   [PGDGroupSpec](#pgdgroupspec)

| Field                                                                                                 | Description                                                                                                                                                  | Required | Default | Validation |
| ----------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ | -------- | ------- | ---------- |
| `configuration` *[BackupConfiguration](/postgres_for_kubernetes/latest/pg4k.v1/#backupconfiguration)* | The CNP configuration to be used for backup. ServerName value is reserved by the operator.                                                                   |          |         |            |
| `cron` *[ScheduledBackupSpec](#scheduledbackupspec)*                                                  | The scheduled backup for the data.<br />Deprecated: This field is deprecated and will be removed in future versions.                                         |          |         |            |
| `schedulers` *[ScheduledBackupSpec](#scheduledbackupspec) array*                                      | Define schedulers for the backup. Each scheduler has a different backup method.<br />Only one of either `backup.cron` or `backup.schedulers` can be defined. |          |         |            |
