This section covers how to manually deploy and configure EDB Postgres Distributed (PGD). 

If you want to install a learning/test environment, we recommend using the [Quickstart guide](/pgd/latest/overview/quickstart).

Manually deploying a PGD cluster consists of the following steps:

1. Meet the [prerequisites](01-prerequisites).
1. [Configure the EDB repositories](02-configure-repositories).
1. [Install the database and PGD software](03-installing-database-and-pgd).
1. [Configure the cluster](04-configuring-cluster).
1. [Check the cluster](05-check-cluster).
1. Learn how to [connect to your cluster](connections).
