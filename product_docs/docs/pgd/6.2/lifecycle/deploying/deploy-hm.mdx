EDB Hybrid Manager (HM) is the centralized control plane for the EDB Postgres AI platform. It allows you to deploy and manage the full lifecycle of **Distributed High-Availability (DHA)** clusters powered by PGD.

Refer to [Creating a database cluster](/edb-postgres-ai/latest/hybrid-manager/using_hybrid_manager/cluster_management/create-clusters/) for instructions on how to deploy **Distributed High Availability** using Hybrid Manager.
