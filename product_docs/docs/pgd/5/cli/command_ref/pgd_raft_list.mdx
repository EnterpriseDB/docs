Lists BDR Raft (consensus protocol) details

## Synopsis

Lists BDR Raft (consensus protocol) details such as Raft instance id, Raft
state (leader, follower), and Raft term. If Raft is enabled at subgroup
level, then that subgroup's Raft instance is also shown.

In some cases, such as network partition, output might vary based on the node
to which the CLI is connected.


```sh
pgd raft list [flags]
```

## Examples

### Viewing Raft details

In this example, we have a multi-group cluster with a subgroup Raft and a
witness, logical standby, and subscriber-only nodes.

Note: Unlike data or witness node, logical standby and subscriber-only nodes 
don't have Raft voting rights (see the **Voting Nodes** column).
