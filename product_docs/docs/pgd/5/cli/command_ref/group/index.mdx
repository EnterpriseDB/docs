The `pgd group` commands are used to manage the groups in the EDB Postgres Distributed cluster.

## Subcommands

- [show](show): Show group-level information.
- [set-option](set-option): Set group-level options.
- [get-option](get-option): Get group-level options.
- [set-leader](set-leader): Set the leader of a group (perform a switchover).
