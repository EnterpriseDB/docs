## Synopsis

The `pgd group show` command is used to display group-level information in the EDB Postgres Distributed cluster.

## Syntax

```
pgd group <GROUPNAME> show [OPTIONS]
```

Where `<GROUPNAME>` is the name of the group for which you want to display information.

## Options

No command specific options. See [Global Options](/pgd/latest/cli/command_ref/#global-options).

## Examples

### Show group information

```shell
pgd group dc1 show
__OUTPUT__
# Summary
Group Property    Value
----------------- -----
Group Name        dc1
Parent Group Name pgd
Group Type        data

# Nodes
Node Name  Node Kind       Join State Node Status
---------- --------------- ---------- -----------
node-one   data            ACTIVE     Up
node-two   data            ACTIVE     Up
node-three data            ACTIVE     Up
sonode-one subscriber-only ACTIVE     Up
sonode2    subscriber-only ACTIVE     Up

# Options
Option Name             Option Value
----------------------- --------------------
apply_delay             00:00:00 (inherited)
check_constraints       true
default_commit_scope     (inherited)
enable_proxy_routing    true
enable_raft             true
enable_wal_decoder      false
location                London
num_writers             -1 (inherited)
route_reader_max_lag    -1
route_writer_max_lag    -1
route_writer_wait_flush false
streaming_mode          default
```
