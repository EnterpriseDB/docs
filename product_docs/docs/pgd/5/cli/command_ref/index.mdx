pgd is the command name for the PGD command line interface

## Synopsis

The EDB Postgres Distributed Command Line Interface (PGD CLI) is a tool to
manage your EDB Postgres Distributed cluster. It allows you to run commands
against EDB Postgres Distributed clusters. You can use it to inspect and manage
cluster resources.


## Commands

### `cluster`
* `show`
  * [`clockskew`](pgd_cluster_show_clockskew) - Show clock skew
  * [`health`](pgd_cluster_show_health) - Check the cluster health
* `verify`
  * [`settings`](pgd_cluster_verify_settings) - Verify the cluster settings
  * [`architecture`](pgd_cluster_verify_architecture) - Verify the cluster architecture

### `events`
* [`list`](pgd_events_list) - List events

### `group`
* [`list`](pgd_group_list) - List groups
* `set`
  * [`options`](pgd_group_set_options) - Set group options
  * [`leader`](pgd_group_set_leader) - Set group leader

### `node`
* [`list`](pgd_node_list) - List nodes
* `set`
  * [`options`](pgd_node_set_options) - Set node options
* `show`
  * [`version`](pgd_node_show_version) - Show node version
  
### `proxy`
* [`create`](pgd_proxy_create) - Create a proxy
* [`delete`](pgd_proxy_delete) - Delete a proxy
* [`list`](pgd_proxy_list) - List proxies
* `set`
  * [`options`](pgd_proxy_set_options) - Set proxy options

### `raft`
* [`list`](pgd_raft_list) - List Raft nodes

### `replication`
* `show`
  * [`slots`](pgd_replication_show_slots) - Show replication slots
  * [`subscription`](pgd_replication_show_subscriptions) - Show replication subscriptions

## Global Options

| Flag | Long Flag | Description |
|------|-----------|-------------|
| -f | --config-file | config file name.<br/>This is ignored if --dsn flag is present.<br/>Default "/etc/edb/pgd-cli/pgd-cli-config.yml") |
|  | --dsn | database connection string<br/>such as "host=bdr-a1 port=5432 dbname=bdrdb user=postgres" |
| -h | --help | help for pgd |
| -L | --log-level | logging level: debug, info, warn, error (default "error") |
| -o | --output | output format: json, yaml |
