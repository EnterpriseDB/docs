The `pgd node` commands are used to manage the nodes in the EDB Postgres Distributed cluster.

## Subcommands

- [show](show): Show node-level information.
- [set-option](set-option): Set node-level options.
- [get-option](get-option): Get node-level options.
- [upgrade-postgres](upgrade-postgres): Perform a major version upgrade of a PGD Postgres node.
