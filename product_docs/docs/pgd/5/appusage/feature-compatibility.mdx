#### Notes

⛔︎&nbsp;:&nbsp;The Async column in the table represents PGD without a synchronous commit scope in use; Lag Control is not a synchronous commit scope, but a controlling commit scope and is therefore available with asynchronous operations. 

<sup>❗️</sup>&nbsp;:&nbsp;Attempting to use Group Commit and Transaction Streaming presents a warning suggesting that transaction streaming be disabled and the transaction appears to take place. In the background, group commit has been automatically disabled to allow the transaction to occur.
