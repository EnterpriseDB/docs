### Coverity

Coverity Scan was used to verify the PGD stack providing coverage
against vulnerabilities using the following rules and coding standards:

-   MISRA C
-   ISO 26262
-   ISO/IEC TS 17961
-   OWASP Top 10
-   CERT C
-   CWE Top 25
-   AUTOSAR
