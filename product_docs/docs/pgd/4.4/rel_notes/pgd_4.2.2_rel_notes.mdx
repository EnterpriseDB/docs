EDB Postgres Distributed version 4.2.2 is a patch release of EDB Postgres Distributed 4, which includes bug fixes for issues identified in previous versions.

| Component | Version |  Type   |                                                   Description                                                           |
| --------- | ------- | ------- | ------------------------------------------------------------------------------------------------------------------------|
| BDR       | 4.2.2   | Bug fix | Fix timeout issue related to global lock handling (BDR-2836) <p> Correctly lock Raft maintained tables when needed </p> |
