EDB Postgres Distributed version 4.3.1 is a patch release of EDB Postgres Distributed 4, which includes bug fixes for issues identified in previous versions.

!!! Note
    This version is required for EDB Postgres Advanced Server versions 12.14.18, 13.10.14, 14.7.0 and later.

| Component | Version |  Type           |                                                   Description                                                           |
| --------- | ------- | --------------- | ------------------------------------------------------------------------------------------------------------------------|
| BDR       | 4.3.1   | Feature         | Add pid to the log message emitted upon writer process death (BDR-2534) |
