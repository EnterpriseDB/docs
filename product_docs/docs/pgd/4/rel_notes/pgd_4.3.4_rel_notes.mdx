Released: 5 Mar 2024

EDB Postgres Distributed version 4.3.4 is a patch release of EDB Postgres Distributed 4, which includes bug fixes for issues identified in previous versions.

!!! Note
This version is required for EDB Postgres Advanced Server versions 12.15, 13.11, 14.10, 15.5 and later.
!!!

| Component | Version | Type         | Description                                                                                 |
|-----------|---------|--------------|---------------------------------------------------------------------------------------------|
| BDR       | 4.3.4   | Bug&nbsp;fix | Change default value of bdr.writers_per_subscription to 1.                                       |
| BDR       | 4.3.4   | Bug&nbsp;fix | Run the ANALYZE on the internal raft tables, RT97735.                                       |
| BDR       | 4.3.4   | Bug&nbsp;fix | Fix segfault in I2PC concurrent abort case, RT93962.                                        |
| BDR       | 4.3.4   | Bug&nbsp;fix | Avoid bypassing other extensions in BdrProcessUtility when processing COPY..TO, RT99345.      |
| BDR       | 4.3.4   | Bug&nbsp;fix | Ensure that consensus connection are handled correctly, RT97649.                            |
| BDR       | 4.3.4   | Bug&nbsp;fix | Fix selected memory leaks, RT99231/RT95314.                                                |
| BDR       | 4.3.4   | Bug&nbsp;fix | Add default_table_access_method to replicated GUCs.                                         |
| BDR       | 4.3.4   | Bug&nbsp;fix | Add missing GUCs to DDL replication GUC list.                                               |
| BDR       | 4.3.4   | Bug&nbsp;fix | Improve local node connection failure logging.                                              |
| BDR       | 4.3.4   | Bug&nbsp;fix | Rework raft snapshot group restore algorithm.                                               |
| BDR       | 4.3.4   | Bug&nbsp;fix | Apply node group change to subscription in raft snapshot restore.                           |
| BDR       | 4.3.4   | Bug&nbsp;fix | Fix commit queue cleanup and flush queueing up for prepared transactions in parallel apply. |
| BDR       | 4.3.4   | Bug&nbsp;fix | Increase default bdr.raft_keep_min_entries to 1000 for 100.                                 |
