Released: 17 June 2025

EDB Postgres Distributed 5.8.1 includes a number of enhancements and bug fixes.

## Highlights

- Fix node sync bugs

## Bug Fixes

<table class="table w-100"><thead><tr><th>Component</th><th>Version</th><th>Description</th><th width="10%">Addresses</th></tr></thead><tbody>
<tr><td>PGD CLI</td><td>5.8.1</td><td><details><summary>Remove the <code>--new-socketdir</code> CLI option and corresponding <code>PGSOCKETDIRNEW</code> env variable for <code>pgd node upgrade</code> command.</summary><hr/><p>The <code>--new-socketdir</code> CLI option and corresponding <code>PGSOCKETDIRNEW</code> env variable was introduced to address a scenario where the old and new socket dirs are different.
This fix addresses the issue by fetching the value of new socket dirs from the initialized new server.</p>
</details></td><td>49024</td></tr>
<tr><td>BDR</td><td>5.8.1</td><td><details><summary>Fix node sync bugs</summary><hr/><p>This release fixes several issues related to node synchronization in BDR.
The node sync process now properly cleans up slots and subscriptions when any of the source, target, or origin parts are involved.
Additionally, the node part has been fixed to prevent hanging during node sync operations.
Subscriber-only nodes will also now be correctly synced.</p>
</details></td><td></td></tr>
