Part of the nature of a distributed system like PGD is that new nodes can be added (and removed) at any time. This means that you do need to know how to install PGD on a fresh node, how to configure it, and how to add it to an existing cluster. This section covers the essential SOPs for installing PGD on a new node, configuring it, and adding it to an existing cluster.
