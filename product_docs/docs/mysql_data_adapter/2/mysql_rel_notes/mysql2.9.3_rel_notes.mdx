title: "MySQL Foreign Data Wrapper 2.9.3 release notes"
navTitle: Version 2.9.3
---

Released: 01 Oct 2025

Enhancements, bug fixes, and other changes in MySQL Foreign Data Wrapper 2.9.3 include:

| Type         | Description                                                                                                                                                                                |
|--------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Enhancement  | Added support for PostgreSQL 18.                                                                                                                                                           |
| Deprecation  | Removed support for versions 12 of EDB Postgres Advanced Server, EDB Postgres Extended Server, and PostgreSQL.                                                                             |
| Bug&nbsp;fix | Improved the handling of `NULL` values when executing parameterized queries or execution-timed parameters to prevent unpredictable behavior due to dangling pointers.                      |
| Bug&nbsp;fix | Changed behavior where planner internal parameters were pushed to the remote server, which could cause the server to crash. Internal parameters are no longer pushed to the remote server. |
