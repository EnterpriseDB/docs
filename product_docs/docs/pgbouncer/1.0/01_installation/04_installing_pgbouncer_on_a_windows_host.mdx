---
title: "Installing PgBouncer on a Windows Host"
---

<div id="using_the_graphical_installer" class="registered_link"></div>


Graphical installers for PgBouncer are available via StackBuilder Plus (for Advanced Server hosts) or Stack Builder (for PostgreSQL hosts).

The following steps walk you through installing PgBouncer 1.15 by accessing StackBuilder Plus through Windows start menu

1.  Access StackBuilder Plus through the Windows start menu; on the `Welcome` window, select your Advanced Server installation from the drop-down list. Click `Next` to continue to the application selection page.

<figure><img src="../images/sbp_welcome.png" class="align-center" alt="The StackBuilder Plus Welcome window" /><figcaption aria-hidden="true"><em>The StackBuilder Plus Welcome window</em></figcaption></figure>

1.  Expand the `Add-ons, tools and utilities` node, and check the box next to the `PgBouncer v1.15.0.1-1`. Click `Next` to continue.

<figure><img src="../images/add-ons.png" class="align-center" alt="Expand Add-ons, Tools and Utilities" /><figcaption aria-hidden="true"><em>Expand Add-ons, Tools and Utilities</em></figcaption></figure>

1.  The selected packages and the default download directory where the package will be installed are displayed; change the download directory location if required. Click `Next`.

<figure><img src="../images/sbp_selected_packages.png" class="align-center" alt="Installation Directory" /><figcaption aria-hidden="true"><em>Installation Directory</em></figcaption></figure>

1.  Once you have downloaded the installation files, a confirmation message is displayed. Click `Next` to start the PgBouncer installation.

<figure><img src="../images/sbp_installation_complete.png" class="align-center" alt="Installing PgBouncer" /><figcaption aria-hidden="true"><em>Installing PgBouncer</em></figcaption></figure>

1.  Select an installation language and click `OK`.

<figure><img src="../images/sbp_language_selection.png" class="align-center" alt="The Language Selection window" /><figcaption aria-hidden="true"><em>The Language Selection window</em></figcaption></figure>

1.  The installer welcomes you to the setup wizard. Click `Next`.

<figure><img src="../images/pgbouncer_welcome.png" class="align-center" alt="The PgBouncer Welcome window" /><figcaption aria-hidden="true"><em>The PgBouncer Welcome window</em></figcaption></figure>

1.  Use the `Installation Directory` field to specify the directory in which you wish to install PgBouncer (the default installation directory is `C:\Program Files\edb`) Then, click `Next` to continue.
2.  Use fields on the `EDB Postgres Advanced Server Installation Details` window to provide connection information for the Advanced Server host:

> -   Use the `Host` field to identify the system on which Advanced Server resides.
> -   Provide the name of the role that PgBouncer will use for connections to the server in the `User Name` field.
> -   Provide the password associated with the role in the `Password` field.
> -   Use the `Port` field to identify the listener port that Advanced Server monitors for client connections.
>
> Then, click `Next` to continue.

<figure><img src="../images/pg_advanced_server_details.png" class="align-center" alt="The Advanced Server Installation Details window" /><figcaption aria-hidden="true"><em>The Advanced Server Installation Details window</em></figcaption></figure>

1.  Use fields on the `PgBouncer Configurations` window to provide your preferences for the PgBouncer installation:

> -   Use the `Listening Port` field to specify the port that PgBouncer monitors for connections.
> -   Use the `Operating System User` field to specify the name of the Linux operating system user that PgBouncer will change to after startup. This option is not supported on Windows hosts.

<figure><img src="../images/pgbouncer_configuration_window.png" class="align-center" alt="The PgBouncer Configuration window" /><figcaption aria-hidden="true"><em>The PgBouncer Configuration window</em></figcaption></figure>

1.  The `Ready to Install` window notifies you when the installer has all of the information needed to install PgBouncer on your system. Click `Next` to install PgBouncer. Progress bars inform you as the installation progresses.

<figure><img src="../images/ready_to_install.png" class="align-center" alt="The Ready to Install window" /><figcaption aria-hidden="true"><em>The Ready to Install window</em></figcaption></figure>

1.  The installer notifies you when the setup wizard has completed the installation. Click `Finish` to exit the installer.

<figure><img src="../images/pgbouncer_installation_complete.png" class="align-center" alt="The installation is complete" /><figcaption aria-hidden="true"><em>The installation is complete</em></figcaption></figure>
