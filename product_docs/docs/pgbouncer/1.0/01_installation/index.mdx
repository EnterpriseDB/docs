The following table lists the latest PgBouncer versions, their supported corresponding Advanced
Server versions, and the supported platforms for each PgBouncer version.

The PgBouncer version required by your Advanced Server installation is version-specific, but the
documented and supported functionality of each version is the same. The information in this guide
applies to each version listed in the table below.

| PgBouncer Version    | Advanced Server Version          | Supported Platforms                                                                                                                                                                                                                   |
|----------------------|----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| PgBouncer 1.14.0.1   | Advanced Server 13               | RHEL/CentOS 7 - x86_64 and RHEL/CentOS 8 - x86_64 <br/>RHEL/CentOS 7 - ppc64le <br/>Debian 9x Stretch and Debian 10x Buster <br/>Ubuntu 18.04 Bionic Beaver and Ubuntu 20.04 LTS Focal Fossa <br/>Windows 86 x64 Interactive Installer                    |
| PgBouncer 1.14.0.1   | Advanced Server 12               | RHEL/CentOS 7 - x86_64 and RHEL/CentOS 8 - x86_64 <br/>RHEL 7 - ppc64le <br/>Debian 9x Stretch and Debian 10x Buster <br/>Ubuntu 18.04 Bionic Beaver <br/>Windows 86 x64 Interactive Installer <br/>SUSE Linux Enterprise Server (SLES) 12 SP4 |
| PgBouncer 1.13.0.1   | Advanced Server 12               | RHEL/CentOS 7 - x86_64 and RHEL/CentOS 8 - x86_64 <br/>RHEL 7 - ppc64le <br/>Debian 9x Stretch <br/>Ubuntu 18.04 Bionic Beaver <br/>Windows 86 x64 Interactive Installer                                                                  |
| PgBouncer 1.12.0.1   | Advanced Server 12               | RHEL/CentOS 7 - x86_64 and RHEL/CentOS 8 - x86_64 <br/>RHEL 7 - ppc64le <br/>Debian 9x Stretch <br/>Ubuntu 18.04 Bionic Beaver <br/>Windows 86 x64 Interactive Installer                                                                  |
| PgBouncer 1.9.0.1    | Advanced Server 11               | RHEL/CentOS 7 - x86_64 <br/>RHEL 7 - ppc64le <br/>Debian 9x Stretch <br/>Ubuntu 18.04 Bionic Beaver <br/>Windows 86 x64 and Linux 86_64 Interactive Installer                                                                           |
| PgBouncer 1.7.2. 1-1 | Advanced Server 10 and 9.6       | RHEL/CentOS 7 - x86_64 <br/>RHEL 7 - ppc64le <br/>Linux 86_64 and Windows 86 x64 Interactive Installer   

PgBouncer is no longer supported on CentOS/RHEL/OEL 6.x platforms. It is strongly recommended that EDB products running on these platforms be migrated to a supported platform.
