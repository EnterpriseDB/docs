To install EDB PgBouncer, you must have credentials that allow access to the EnterpriseDB repository. To request credentials that allow you to access an EnterpriseDB repository, see the [EDB Repository Access instructions](https://www.enterprisedb.com/repository-access-request).

For platform-specific install instructions, see:

  - [CentOS 7](x86_amd64/pgbouncer_centos7_x86)
  - [SLES 15](x86_amd64/pgbouncer_sles15_x86)
  - [SLES 12](x86_amd64/pgbouncer_sles12_x86)
  - [Ubuntu 20.04](x86_amd64/pgbouncer_ubuntu20_x86)

  - [Debian 11](x86_amd64/pgbouncer_deb11_x86)
  - [Debian 10](x86_amd64/pgbouncer_deb10_x86)
