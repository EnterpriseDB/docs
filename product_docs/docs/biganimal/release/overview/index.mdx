BigAnimal is a fully managed database-as-a-service, running in your cloud account or BigAnimal's cloud account, and operated by the Postgres experts. BigAnimal makes it easy to set up, manage, and scale your databases. You can use [PostgreSQL](https://www.enterprisedb.com/docs/supported-open-source/postgresql/), [EDB Postgres Advanced Server](/epas/latest/) with Oracle compatibility, or [EDB Postgres Extended Server](/pge/latest/) with extreme high availability for your Postgres database.
