---
title: "Administering your account"
---

Administrative activities for the BigAnimal account include portal and database user access management as well as account activity reviews.

If you have coordinated with [BigAnimal Support](../overview/support) to have the Apache Superset feature enabled, see [Managing Superset access](01a_superset_access) for information on setting up roles and permission for access to the Superset data sources.
