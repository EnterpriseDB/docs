The following functionality has been added to Migration Toolkit 53.0.0:

-   Migration Toolkit now supports PostgreSQL version 12 and Advanced Server version 12 as the source and the target database.

-   Migration Toolkit supports the migration of the following Oracle features to Advanced Server version 12:  
    -   Compound triggers
    -   SYS_GUID
    -   MEDIAN
    -   LISTAGG
    -   CAST_MULTISET
    -   Interval Partition
    -   SELECT UNIQUE

For more information, see [EDB Postgres Advanced Server v12 Features](https://www.enterprisedb.com/edb-docs/static/docs/epas/12/EPAS_Release_Notes_v12_Beta.pdf).
