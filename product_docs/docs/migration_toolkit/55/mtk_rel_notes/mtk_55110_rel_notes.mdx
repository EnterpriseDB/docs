navTitle: Version 55.11.0

Released: 20 Feb 2025

New features, enhancements, bug fixes, and other changes in Migration Toolkit 55.11.0 include:

| Type         | Description                                                                                                                                                                                | Ticket |
|--------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------|
| Enhancement  | Added support for migration of user-defined data types from a source Microsoft SQL Server database to a target PostgreSQL/EDB Postgres Advanced Server database.                           | #41617 |
| Enhancement  | Added certification for installation and use on RHEL 9 ARM64 architectures.                                                                                                                |        |
| Bug&nbsp;fix | Fixed the issue where an offline data migration generated invalid COPY format contents when table-level parallel loading option was enabled.                                               | #41558 |
| Bug&nbsp;fix | Fixed an issue where an Oracle view based on a database link was skipped during migration.                                                                                                 | #42798 |
| Bug&nbsp;fix | Fixed an issue that caused a migration of a MySQL table containing a `TIMESTAMP` column with an `ON UPDATE` clause to fail.                                                                |        |
| Bug&nbsp;fix | Fixed an issue where migrating a partitioned table in a PostgreSQL source database with a partition key column named ‘partition’ to a target EDB Postgres Advanced Server database failed. | #43595 |
| Bug&nbsp;fix | Fixed an issue that caused a migration of a Microsoft SQL Server table containing a `SYSNAME` type column to fail.                                                                         | #43034 |
| Bug&nbsp;fix | Fixed an issue that caused a migration of an Oracle table containing a `VARRAY` type column to fail.                                                                                       |        |
| Bug&nbsp;fix | Improved the handling and logging of target database connection-related errors that were previously only being reported as Null Pointer Exception (NPE) type errors.                       |        |
