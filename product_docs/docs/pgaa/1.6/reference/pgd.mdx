## BDR node group operations

| Option  | Description |
| ------ | ----------- | 
| analytics_storage_location | The PGFS target location for analytics writes. | 
| analytics_storage_format | The data format for offloaded files: `delta` (default) or `iceberg`. | 
| analytics_write_catalog | The catalog used for analytics writes. |


**Example: Setting node group options:**

```sql
SELECT bdr.alter_node_group_option(
  'my_node_group',
  'analytics_storage_location',
  's3://my-analytics-bucket/pgp-data'
);
```
