## Postgres compatibility

Supported versions of Postgres for PGAA 1.6:

| PostgreSQL distribution                 | PostgreSQL version |
|:----------------------------------------|:-----------------------|
| **EDB Postgres Advanced Server (EPAS)** | 16, 17, 18     | 
| **PostgreSQL (PG)**                     | 16, 17, 18     | 
| **EDB Postgres Extended Server (PGE)**  | 16, 17, 18     | 



Standalone PGAA, PGAA in PGD, PGAA in PGD in HM supports:
- read Parquet using the Seafowl, Spark and Spark w/RAPIDS engine (only for non HM Spark) on local FS/S3 .
- Read Iceberg using the Seafowl and Spark engines on local FS, S3, GCP, Azure; but not yet RAPIDS
- Read Delta Lake using the Seafowl engine on local FS, S3, GCP, Azure

Standalone PGAA without PGD supports:
- Write to Iceberg / Delta Lake using CREATE TABLE AS statements (full table rewrite)
