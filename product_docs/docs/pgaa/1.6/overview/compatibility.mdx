### Postgres compatibility

| PostgreSQL distribution                 | PostgreSQL version |
|:----------------------------------------|:-----------------------|
| **EDB Postgres Advanced Server (EPAS)** | 16, 17, 18     | 
| **PostgreSQL (PG)**                     | 16, 17, 18     | 
| **EDB Postgres Extended Server (PGE)**  | 16, 17, 18     | 


### Platform compatibility

Debian 11/12 or Ubuntu 22/24 on amd64 architectures, and RHEL 8/9 on both amd64 and arm64.

### Spark

Apache Spark: Version 3.4 or later.

### PGD

PGD version 5.9.1 or later.

### Hybrid Manager

Hybrid Manager Innovation Release 2026.2 or later. 

### Supported storage formats and providers

The Postgres Analytics Accelerator (PGAA) supports:

- Reading Iceberg, Delta Lake, and Parquet files on local filesystem, S3, GCP, and Azure via storage locations.
- Reading Iceberg files on local filesystem, S3, GCP, and Azure via an external Iceberg REST catalog.
- Reading Parquet files on local filesystem and S3 via storage locations, using a Spark engine with RAPIDS acceleration.
- Writing Iceberg and Delta Lake files on local filesystem, S3, GCP, and Azure using `CREATE TABLE AS SELECT` (CTAS).
