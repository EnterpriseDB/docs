### Postgres compatibility

| PostgreSQL distribution                 | PostgreSQL version |
|:----------------------------------------|:-----------------------|
| **PostgreSQL (PG)**                     | 16, 17     | 
| **EDB Postgres Extended Server (PGE)**  | 16, 17     | 
| **EDB Postgres Advanced Server (EPAS)** | 16, 17     | 



### Platform compatibility

Debian 11/12 or Ubuntu 22/24 on amd64 architectures, and RHEL 8/9 on both amd64 and arm64.

### Spark

Apache Spark: Version 3.4 or later.

### PGD

PGD version 6.1 or later.

### Hybrid Manager

Hybrid Manager version 1.3 or later. 

### Supported storage formats and providers

| Data Format | Operation | Object storage | Engine support |
| ----------- | --------- | ---------------------------------- | -------------- | 
| Iceberg | Read | Local, S3, GCP, Azure (via storage locations) | Seafowl |
| Iceberg | Read | Iceberg REST catalog | Seafowl  <br /> Spark |
| Delta Lake | Read | Local, S3, GCP, Azure (via storage locations) | Seafowl |
| Iceberg | Write (via `CTAS`) | Local, S3, GCP, Azure (via storage locations) | PostgreSQL engine via CompatScan |
| Iceberg | Write (via `CTAS`) | Iceberg REST catalog | PostgreSQL engine via CompatScan |
| Delta Lake | Write (via `CTAS`) | Local, S3, GCP, Azure (via storage locations) | PostgreSQL engine via CompatScan |
