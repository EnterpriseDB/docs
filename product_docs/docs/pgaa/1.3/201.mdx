Build on foundational knowledge and begin applying key Analytics Accelerator technologies in your environment.

## What you will learn

- How to set up and query EDB Postgres Lakehouse clusters
- How to work with Apache Iceberg® and Delta Lake tables
- How to implement tiered tables with PGD
- How to connect BI tools to the Analytics Accelerator
- How to apply these technologies to BI reporting, historical data analysis, and operational analytics
