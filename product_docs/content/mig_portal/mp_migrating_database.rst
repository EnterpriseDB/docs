*****************************
`Migrating a Database`:index:
*****************************

To migrate a database, you must complete the following steps:

1. Perform a :doc:`Schema Extraction <mp_schema_extraction>`.

2. Perform a :doc:`Schema Assessment <mp_schema_assessment>`.

3. Perform a :doc:`Schema Migration <mp_schema_migration>`.

4. Perform a :doc:`Data Migration <mp_data_migration>`.

The following sections provide detailed information about each step in the
migration process.
