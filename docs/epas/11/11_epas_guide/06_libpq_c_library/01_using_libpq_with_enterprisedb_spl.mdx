title: "Using libpq with EnterpriseDB SPL"
---

<div id="using_libpq_with_enterprisedb_spl" class="registered_link"></div>

The EnterpriseDB SPL language can be used with the libpq interface library, providing support for:

- Procedures, functions, packages
- Prepared statements
- `REFCURSORs`
- Static cursors
- `structs` and `typedefs`
- Arrays
- DML and DDL operations
- `IN/OUT/IN OUT` parameters
