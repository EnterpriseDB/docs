The following database compatibility for Oracle features have been added to Advanced Server 10 to create Advanced Server 11:

- Advanced Server now supports SPL standalone procedure overloading. Note that this feature is not compatible with Oracle databases. For more information, see Section 3.2.3.1.
- Advanced Server now supports the `PRAGMA` `AUTONOMOUS_TRANSACTION` directive within any SPL block to provide the autonomous transaction capability. For more information, see Section 3.6.3.
