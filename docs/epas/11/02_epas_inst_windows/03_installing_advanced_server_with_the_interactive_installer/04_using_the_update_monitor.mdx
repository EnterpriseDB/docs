---

title: "Using the Update Monitor"
---

<div id="using_the_update_monitor" class="registered_link"></div>

The Update Monitor utility polls the EnterpriseDB website and alerts you to security updates and enhancements for Windows hosts as they become available. Update Monitor is automatically installed by the graphical installer.

When Update Monitor is actively monitoring, the Postgres elephant icon is displayed in the system tray (see Figure 3.26).

![](images/the_update_monitor_icon.png)

<center><i>Figure 3.26 -The Update Monitor icon.</i></center>

If you have installed more than one version of Advanced Server, Update Monitor watches for updates and alerts for all installed versions. When Update Monitor finds an update or alert, it displays an alert symbol to let you know that an update or alert is available for one of the Advanced Server installations (see Figure 3.27).

![](images/the_update_monitor_icon_displays_an_alert.png)

<center><i>Figure 3.27 -The Update Monitor icon displays an alert.</i></center>

Right click on the symbol to access the Update Monitor context menu (shown in Figure 3.28).

![](images/the_update_monitor_context_menu.png)

<center><i>Figure 3.28 -The Update Monitor context menu.</i></center>

Click `Install components` to open StackBuilder Plus, and check for component updates. See Section [3.3](https://www.enterprisedb.com/edb-docs/d/edb-postgres-advanced-server/installation-getting-started/installation-guide-for-windows/11/EDB_Postgres_Advanced_Server_Installation_Guide_Windows.1.13.html#) *Using StackBuilder Plus* for detailed information about the update process.

If one or more alerts are available for your Advanced Server installation, the context menu displays the alert count. Select `View alerts` to display the `EnterpriseDB Advanced Server` Alerts window (see Figure 3.29).

![](images/an_enterprisedb_technical_alert.png)

<center><i>Figure 3.29 - An EnterpriseDB Technical alert.</i></center>

The technical alert will provide information about updates and issues that might impact your installation.

- Click `Run StackBuilder Plus` to open StackBuilder Plus and install relevant software updates.
- Click `Next` to access each successive alert.

When you have reviewed all of the alerts, the icon no longer displays the alert symbol; use the X icon in the upper-right corner of the dialog to close Update Monitor.
