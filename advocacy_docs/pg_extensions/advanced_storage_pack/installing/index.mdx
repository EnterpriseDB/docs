Advanced Storage Pack is supported on the same platforms as the Postgres distribution you're using. Support starts with Postgres 14 for Autocluster and Refdata, and with Postgres 15 for Bluefin. For details, see:

- [EDB Postgres Advanced Server Product Compatibility](https://www.enterprisedb.com/platform-compatibility#epas)
- [PostgreSQL Product Compatibility](https://www.enterprisedb.com/resources/platform-compatibility#pg)
- [EDB Postgres Extended Server Product Compatibility](https://www.enterprisedb.com/resources/platform-compatibility#epas_extended)

| Operating system   | TAMs available                | Minimum version                                           | Installation method               |
|--------------------|-------------------------------|-----------------------------------------------------------|-----------------------------------|
| [Linux](linux)     | Autocluster, Refdata, Bluefin | Postgres 14 (Autocluster, Refdata), Postgres 15 (Bluefin) | EDB repository package manager    |
| [Windows](windows) | Bluefin only                  | PostgreSQL / EPAS 15                                      | StackBuilder or StackBuilder Plus |
