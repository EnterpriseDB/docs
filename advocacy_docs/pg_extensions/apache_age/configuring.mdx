navTitle: Configuring
---

1. Create the Apache AGE extension:
   
   ```sql
   CREATE EXTENSION age;
   ```
   

2. Load the extension:

   ```sql
   LOAD 'age';
   ```


After loading the extension, you can perform additional configuration. See the [Apache AGE official documentation](https://age.apache.org/age-manual/master/intro/setup.html#post-installation-setup).
