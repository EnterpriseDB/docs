PostgreSQL Anonymizer(pg_anonymizer) is an extension released as open source software under the PostgreSQL License. 

pg_anonymizer is an extension to mask or replace personally identifiable information (PII) or commercially sensitive data from a Postgres database. 

For more information about pg_anonymizer, see:

- [Installing pg_anonymizer](installing.mdx)
- [Configuring pg_anonymizer](configuring.mdx)
