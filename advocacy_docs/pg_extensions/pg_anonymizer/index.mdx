PostgreSQL Anonymizer (`postgresql_anonymizer`) is an extension released as open source software under the PostgreSQL License. 

`postgresql_anonymizer` is an extension to mask or replace personally identifiable information (PII) or commercially sensitive data from a Postgres database. 

For more information about `postgresql_anonymizer`, see:

- [Installing `postgresql_anonymizer`](installing.mdx)
- [Configuring `postgresql_anonymizer`](configuring.mdx)
