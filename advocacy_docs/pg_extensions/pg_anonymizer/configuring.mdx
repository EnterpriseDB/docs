---
title: Configuring pg_anonymizer
navTitle: Configuring
---

1. In `postgresql.conf`, add `anon` to the `shared_preload_libraries` parameter: 

   ```ini
   shared_preload_libraries = '$libdir/anon'
   ```
   
   If `shared_preload_libraries` has other extensions, then you can add `anon` to the list. The order doesn't matter. 
   

2. Restart Postgres.


3. Create the pg_anonymizer extension in the database:

```sql
CREATE EXTENSION anon;
```

After enabling the extension, you can perform additional configuration. See the [pg_anonymizer official documentation](https://postgresql-anonymizer.readthedocs.io/en/stable/).
