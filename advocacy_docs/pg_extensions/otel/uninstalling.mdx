---
title: "Uninstalling EDB OTEL"
navTitle: Uninstalling
description: Removing the EDB OTEL extension from Postgres
---

To uninstall edb_otel:

1. Drop the edb_otel extension: 

   ```sql
   # Connect to your database using psql
   psql -d postgres -U enterprisedb
   # Drop the extension
   DROP EXTENSION edb_otel;
   ```

1. On the server where you want to remove the extension, edit the `postgresql.conf` file. Delete the `shared_preload_libraries` parameter line.

   In Linux, remove: 

   ```ini
   shared_preload_libraries = `$libdir/edb_otel`
   ```

1. To apply the changes, restart the Postgres database server.
