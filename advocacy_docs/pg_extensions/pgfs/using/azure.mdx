---
title: Using PGFS with Azure
navTitle: Azure
---

PGFS supports Azure Blob Storage and Azure Data Lake Storage Gen2 (ADLS Gen2). These examples shows how to use PGFS for Azure Blob Storage and Azure Data Lake Storage Gen2 (ADLS Gen2).


1. Create a storage location in Azure Blob storage using a shared access key credential:

   ```sql
   SELECT pgfs.create_storage_location('edb_ai_example_images', 'https://edb-account.blob.core.windows.net', 
       NULL, 
       '{
           "account_name": "edb-account",
           "container_name": "ai-images"      
       }', 
       '{
           "account_key": "<your azure storage key>"
       }'
   );
   ```

1. Create a storage location in Azure Blob storage using client credentials:

   ```sql
   SELECT pgfs.create_storage_location('edb_ai_example_images', 'https://edb-account.blob.core.windows.net', 
       NULL, 
       '{
           "account_name": "edb-account",
           "container_name": "ai-images"      
       }', 
       '{
           "client_id": "<your azure clientID>", 
           "client_secret":  "<your azure client secret>",
           "tenant_id": "<your azure tenantID>"
       }'
   );
   ```
