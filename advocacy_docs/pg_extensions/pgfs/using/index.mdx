To use a storage location with AIDB, you need to create a volume from the storage location. To do that, see [Creating a volume](https://www.enterprisedb.com/docs/edb-postgres-ai/latest/ai-factory/pipeline/volumes/).

To use a PGFS storage location with AIDB, a volume must be created in AIDB:

- **PGFS storage location**: Represents the external storage provider with URL, credentials, and settings.

- **AIDB Volume**: A connection point that maps a PGFS storage location to the AI database (AIDB), specifying the data format (e.g., Image, Text).
