`tds_fdw` is a PostgreSQL foreign data wrapper (FDW) released as open source software under the PostgreSQL License.

`tds_fdw` lets you connect PostgreSQL to databases that use the [Tabular Data Stream (TDS) protocol](https://en.wikipedia.org/wiki/Tabular_Data_Stream), such as Sybase databases and Microsoft SQL Server. It uses the [FreeTDS](https://www.freetds.org) library to implement the DB-Library interface.

`tds_fdw` supports WHERE clause and column pushdowns when `match_column_names` is enabled. It doesn't currently support JOIN push-down or write operations.
