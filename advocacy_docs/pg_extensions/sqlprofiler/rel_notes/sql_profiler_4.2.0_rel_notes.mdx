Release date: 18 July 2025

This release of SQL Profiler includes:

| Type        | Description                                                                                                                                                                                                                         |
|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Enhancement | Added a new role named `sql_profiler_admin`.  This role allows non-superusers to profile queries, deactivate traces, and list active traces that belong to other users.|
