Release date: 12 Dec 2025

This release of Wait States includes:

| Type         | Description                                                                                             | Addresses |
|--------------|---------------------------------------------------------------------------------------------------------|-----------|
| Enhancement  | Added support for EDB Postgres Advanced Server, EDB Postgres Extended Server and PostgreSQL version 18. |           |
| Bug&nbsp;fix | Added support for new query types (`RTE_GROUP`, `RTE_RESULT`) to prevent query normalization errors.    |           |
| Bug&nbsp;fix | Streamlined MaxBackends management using upstream `shmem_request_hook` for better compatibility.        |           |
| Bug&nbsp;fix | Improved log file handling. Logs are now automatically reinitialized if unlinked by the backend.        |           |
