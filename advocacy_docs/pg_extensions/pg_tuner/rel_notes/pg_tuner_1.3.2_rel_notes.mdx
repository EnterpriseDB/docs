title: Release notes for Postgres Tuner version 1.3.2
navTitle: "Version 1.3.2"
---

This release of Postgres Tuner includes:

| Type    | Description                                                                                                                                                                          |
|---------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Feature | Added a new configuration parameter, `edb_pg_tuner.wal_path` that enables automatic limiting of the suggested `max_wal_size` to 50% of the total `pg_wal` disk size.                 |
