All feedback is welcome:

1. an idea for something new
2. you feel there's something missing
3. a mistake

Github Issues is the fastest way to get that feedback in front of us.

[Open an issue on Github &rarr;](https://github.com/rocketinsights/edb_docs_advocacy/issues)

---

### Have code or content to contribute?

The only thing we love more than Github issues are Github pull requests. If you have one to share, great! 

Before you click that button, please review our [contribution guidelines](/community/contribute).
