Implementing Quest Foglight for Databases with EDB Postgres Advanced Server and/or EDB Postgres Extended Server requires the following components:

- EDB Postgres Advanced Server and/or EDB Postgres Extended Server
- Quest Foglight Software

### Prerequisites

- A running EDB Postgres Advanced Server instance or EDB Postgres Extended Server instance
- Quest Foglight Software installed
- Login credentials for your EDB Advanced Server or EDB Postgres Extended instance and Foglight interface

### Configuring Foglight for EDB Postgres Advanced Server and EDB Postgres Extended Server

The following steps show how to configure Quest Foglight for Databases to connect to EDB Postgres Advanced servers and EDB Postgres Extended Servers:

1. Log on to the Quest Foglight console

   ![Quest Login](Images/QuestLoginFrame.png)

2. Once on the home page, ensure you are in ‘Expert View’ by navigating to the toolbar on the left and switching from ‘Standard’ to ‘Expert’.  Then navigate to the Databases section by clicking on 'Databases' in the menu:

   ![Quest Databases Menu](Images/QuestDatabasesScreen.png)

3. Select PostgreSQL as the database server and click on 'Monitor' ( + icon) to launch the Agent Installer Wizard:

   ![Quest Monitor Agent Type](Images/QuestMonitorAgentType.png)

4. Enter the desired agent name and click Next

**EDB Postgres Advanced Server example:**

   ![Quest Agent Name](Images/QuestAgentName.png)

5. Enter the Agent properties for the database you are connecting to and click Next:

**EDB Postgres Advanced Server example:**

   ![Quest Agent Properties](Images/QuestAgentProperties.png)

6. Verify the Agent Summary and click Finish to create the EDB Postgres Advanced Server or EDB Postgres Extended Server agent for monitoring:

**EDB Postgres Advanced Server example:**

   ![Quest Agent Summary](Images/QuestAgentSummary.png)

7. The agents are displayed once they are successfully created and activated:

   ![Quest Agent Displays](Images/QuestAgentDisplays.png)
