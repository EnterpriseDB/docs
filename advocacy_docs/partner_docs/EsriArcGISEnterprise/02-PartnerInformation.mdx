| **Product description**  | ArcGIS is a geographic information system (GIS) technology that provides tools to capture, view, edit, manage, analyze, and share data in the context of location. Esri ArcGIS Pro is the single desktop GIS application that users can connect their EDB Postgres Advanced Server and Postgres databases to. Esri ArcGIS Pro allows users to:<br />- Explore, visualize, and analyze data<br />- Create 2D maps and 3D scenes<br />- Share their work to an ArcGIS Enterprise Portal<br />Esri ArcGIS Enterprise is the foundational software system for GIS. It's the backbone for running the Esri suite of applications and custom applications. Esri ArcGIS Enterprise and Esri ArcGIS Pro are integrated for creating maps and data. |
