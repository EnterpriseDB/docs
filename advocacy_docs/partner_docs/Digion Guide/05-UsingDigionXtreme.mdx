### Sample User Scenarios
DigionXtreme manages your data in all of its forms and allows you to easily change the format of data that is stored within your database.  

To use DigionXtreme:

1. Log into Digion

2. Access the user dashboard where you will see the servers that are connected

3. Ensure that you are connected to the server

4. Your data is now ready to be managed
