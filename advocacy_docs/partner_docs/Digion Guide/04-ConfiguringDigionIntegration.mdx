Implementing DigionXtreme with EDB Postrgres Advanced Server requires the following components:
- EDB Postgres Advanced Server
- Digion Software 


### Prerequisites

- A running Digion networkl
- Access to the Digion web console.

### Configuring DigionXtreme

- Download the DigionXtreme software from (www.digion.com)
- Ensure that EDB Advanced server is installed on your machine and running
