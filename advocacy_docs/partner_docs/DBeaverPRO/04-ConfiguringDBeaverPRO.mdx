Implementing DBeaver PRO with EDB Postgres Advanced Server and EDB Postgres Extended Server requires the following components: 

- A running EDB Postgres Advanced Server or EDB Postgres Extended Server instance
- DBeaver PRO application installed on a system where you manage the running database

## EDB Postgres Advanced Server configuration

1. Launch DBeaver PRO on your system. 
2. From the **Database** menu at the top left, select **New Database Connection**. In the **All** tab, select **EDB**, and then select **Next**.    

![Configuring EDB Postgres Advanced Server with DBeaver Pro](Images/EPASConfig.png)

!!! Note
    You can also connect to your EDB database instance by selecting **Database > New Connection from JDBC URL**. DBeaver PRO selects **EnterpriseDB** in the **Drivers** field when you provide the EDB JDBC URL in the **JDBC URL** field. If you use this method, select **Proceed** and continue to step 3.

![JDBC URL Connection Method](Images/JDBCConnection.png)

3. Provide connection information for the user, password, host, port, and database for your EDB Postgres Advanced Server instance.

![Connection Settings](Images/EPASConnectionSettings.png)

4. Select **Test Connection**. The success message appears.

![Successful Connection Message](Images/EPASTestConnection.png)

5. Respond **OK** to the message. Then select **Finish** on the Connection Settings window. The connection is then created and appears in the left-side tree.  

![Display of Connection](Images/ConnectionOnLeftSide.png)

## EDB Postgres Extended Server configuration

1. Launch DBeaver PRO on your system. 
2. From the **Database** menu in the top left, select **New Database Connection**. In the **All** tab, select **PostgreSQL**. Then select **Next**.

![Extended Server Configuration](Images/ExtendedConfiguration.png)

!!! Note
    You can also connect to your PostgreSQL database instance by selecting **Database > New Connection from JDBC URL**. When you provide the PostgreSQL JDBC URL in the **JDBC URL** fieldD, Beaver PRO selects **PostgreSQL** in the **Drivers** field. If you use this method, select **Proceed** and continue to step 3.

![New Connection from JDBC URL](Images/NewConnectionExtended.png)

3. Provide connection information for the user, password, host, port, and database for your PostgreSQL instance.

![Connection Information for PostgreSQL instance](Images/PostgreSQLInformation.png)

4. Select **Test Connection**. The success message appears.

[Successful Connection](Images/ExtendedTestConnection.png)

5. Respond **OK** to the message. Then, on the **Connection Settings** window, select **Finish**. The connection is created and shown in the left-side tree. 

![Connection Setting Completed](Images/PostgresConnectionSettings.png)
