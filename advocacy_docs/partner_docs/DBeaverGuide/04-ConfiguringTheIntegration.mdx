## Prerequisites

- A running EDB Postgres Advanced (EPAS) / EDB Postgres Extended instance
- DBeaver Community Edition
- EDB JDBC Driver

The following steps show how to configure  DBeaver for EDB Postgres Advanced and EDB Postgres Extended:

### EDB Postgres Advanced

1. Download EDB JDBC Driver (by default, the Driver should download into the directory `C:\Program Files\edb\jdbc)`
2. Setup DBeaver with the EDB JDBC Driver. This can be done using 2 methods:

    **Method 1:** Change Driver for the `EnterpriseDB` database type
    
    1. Select `EDB` as database type and click `Next`.

        ![](Images/DBeaverMethod1.png)

    2. Click `Edit Driver Settings` and replace the jar files with the downloaded EDB JDBC Driver jar file `C:\Program Files\edb\jdbc\edb-jdbc18.jar`

        ![](Images/DBeaverMethod1Part2.png)

    **Method 2:** Add Custom Driver for EDB JDBC 

    1. From the `Database` menu select `Driver Manager` and click `New`
    2. In the `Create new driver` screen enter the values in the fields as shown below:
    
        ![](Images/DBeaverMethod2.png)

**EDB Postgres Extended**
 
1. Select EDB `Postgresql` as database type and click `Next`.
2. On the  Connection Settings screen, click `Test Connection` to verify connectivity to the database server.
