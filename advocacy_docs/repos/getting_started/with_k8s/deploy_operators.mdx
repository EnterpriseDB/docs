## Prerequesites

EDB's container registries are built on Cloudsmith, which in turn relies on AWS's CloudFront CDN. In order to reliably use these registries, you'll want to ensure that your network infrastructure isn't blocking access! See: [Configuring Your Firewall to Accept Cloudsmith and AWS CDN IP Ranges](https://support.cloudsmith.com/hc/en-us/articles/28434685694097-Configuring-Your-Firewall-to-Accept-Cloudsmith-and-AWS-CDN-IP-Ranges)
