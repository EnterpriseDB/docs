Additional programs available in the WarehousePG installation.

The following PostgreSQL `contrib` server utility programs are installed:

-   [pg_xlogdump](https://www.postgresql.org/docs/12/pgxlogdump.html) - Server utility program to display a human-readable rendering of the write-ahead log of a WarehousePG cluster.
