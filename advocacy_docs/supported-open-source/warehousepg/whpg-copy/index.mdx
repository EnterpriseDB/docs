WarehousePG (WHPG) Copy, or `wgph-copy` is a high-performance utility designed to copy data between WarehousePG databases efficiently.

WareHousePG Copy is versioned independently of WareHousePG. Refer to [Supported platforms]](/warehousepg/whpg-copy/overview/supported_platforms) for compatibility information.
