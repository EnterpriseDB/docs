Installing

## Installation overview

Assumption: you have installed Grafana, Loki and Prometheus
Prepare for the installation: download packages and create a hosts_all file.
Install Extension and Collector on the WHPG cluster
Install Exporter on a host of your choice
Configure Grafana

## Prerequisites

- WarehousePG version 6.x for RH7, RH8
- WarehousePG version 7.x for RH8, RH9
- Grafana 10.4.8 or later
- Loki 3.5 or later 
- Prometheus 3.5.0 or later
- User permissions

The following table lists the connection requirements among the different components. Note that the ports indicated are referring to their default vaues, you can customize them according to your environment:

| Source | Destination | Use  
| ------ | ----------- | ---
| WHPG Coordinator | Prometheus:9090 | Push host level metrics
| WHPG Coordinator | Loki:3100       | Push log files
| Exporter         | Prometheus:9090 | Push SQL metrics for historical data
| Grafana          | Exporter:9187   | Expose live query endpoint
| Grafana          | Prometheus:9090 | Display host level metrics
| Grafana          | Loki:3100       | Display log metrics
| User             | Grafana:3000    | Access to Observability
