WarehousePG (WHPG) Observability uses Grafana to visualize your WHPG cluster. 

WHPG Observability provides three pre-defined dashboards in JSON format, availble here:

- WHPG Overview
- WHPG Detailed
- WHPG Logs

In Grafana, a Dashboard is the main canvas for data, containing Rows for grouping related visualizations, and Panels (like graphs, tables, gauges) are the individual charts or widgets displaying metrics.

##  WHPG Overview

The Dashboard provides a quick view of the current system status, Segment Health, Queries, CPU, Memory, and Disk usage.
