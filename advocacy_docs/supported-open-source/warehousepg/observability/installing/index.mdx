!!! Note
This guide assumes you have installed Grafana, Prometheus, and Loki. You can use the following [ready-to run stack](https://github.com/warehouse-pg/warehouse-pg-grafana-dashboards) using Docker Compose to quickly spin up an environment that comprises Grafana, Prometheus, and Loki.

!!!

The installation process consists of the following steps:

- Verify the [Prerequisites](prerequisites).
- [Install the WHPG Extension and WHPG Collector](collector_extension) on your WarehousePG (WHPG) cluster.
- [Install the WHPG Exporter](exporter) on your designated host.
- [Configure Grafana](grafana) to visualize your dashboards.
