!!! Note
This guide assumes you have installed Grafana, Loki and Prometheus. You can use the following [ready-to run stack](https://github.com/warehouse-pg/warehouse-pg-grafana-dashboards) using Docker Compose to quickly spin up an environment that comprises Grafana, Prometheus, and Loki.

!!!

The installation process consists of the following steps:

- Verify the [Prerequisites](prerequisites).
- [Install the Extension and Collector](collector_extension) on your WHPG cluster.
- [Install the Exporter](exporter) on your designated host.
- [Configure Grafana](grafana) to visualize your dashboards.
