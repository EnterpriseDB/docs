## Supported WarehousePG versions

FlowServer for WarehousePG 1.0 is compatible with the following version of WarehousePG:

- WarehousePG (WHPG) version 6.x running on RH7, RH8, or RH9.
- WarehousePG version 7.x running on RH8 or RH9.
