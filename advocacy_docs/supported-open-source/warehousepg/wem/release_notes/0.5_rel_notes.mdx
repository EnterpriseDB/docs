**Release Date:** February 27, 2026

**Status:** Technical preview

Version 0.5 introduces transformative features for database administrators and developers, most notably the integration of an AI-powered query assistant and significant improvements to session handling and installation stability.

## Release highlights

- Core improvements
    - Detect WarehousePG (WHPG) major versions (6 or 7) automatically during setup.
    - Install monitoring views on all databases, including the Postgres system database, automatically.
    - Run a comprehensive validation script to verify the integrity of the entire installation.

- Data skew & performance
    - View distribution imbalances across segments to find empty segments and poor distribution keys.
    - Analyze skew percentages through new visual charts and hot-spot indicators.
    - Utilize five specialized Canary check types (query, connection, replication, performance, and health).

- Query & session management
    - Save executed queries and full result data automatically for 24 hours.
    - Reopen past queries in the editor to modify and re-run them.
    - Detect expired sessions globally with clear messaging and a 3-second auto-redirect to login.

- AI query assistant (Claude-powered)
    - Write queries in plain English using natural language processing.
    - Perform automatic `EXPLAIN` plan analysis and apply performance fixes with a single click.
    - Get plain-English explanations and fixes for complex SQL errors.
    - Use dedicated "Explain," "Optimize," and "Debug" shortcuts via `Ctrl+K`.
    - Configure the assistant by exporting the `ANTHROPIC_API_KEY` environment variable.
