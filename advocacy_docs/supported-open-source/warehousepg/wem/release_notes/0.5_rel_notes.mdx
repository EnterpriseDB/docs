## Release Highlights

- **AI-Powered Query Assistant Powerd by Claude:**
- Natural Language to SQL: Write queries in plain English
- Query Optimization: Automatic EXPLAIN plan analysis with optimization suggestions
- Error Debugging: AI-powered error explanation and fixes
- Schema-Aware: Context-aware suggestions based on your database schema
- One-Click Fixes: Apply suggested optimizations with a single click Quick Actions
- Ctrl+K - Open AI Assistant
- Explain Query - Get plain English explanation
- Optimize - Receive performance improvement suggestions
- Debug Error - Understand and fix SQL errors
- Configuration: export ANTHROPIC_API_KEY="your-key-here"
- Query Results Auto-Save All executed queries are automatically saved for 24 hours with full result data.

**Reopen queries in editor for modification:**
- View success/failure status and execution time
- Full result data with row counts and timing
- One-click delete for cleanup
- Session Timeout Management

**Global Detection:** 
- Automatically detects when your session expires
- Clear Messaging: Shows friendly banner: "Your session has expired" instead of JSON errors
- Auto-Redirect: Redirects to login after 3 seconds
- Data Skew Analysis

**Shows distribution imbalance across segments:**
- Highlights empty segments (poor distribution keys)
- Visual charts for skew percentage
- Canary Check Enhancements

**Check types:**
- Query, connection, replication, performance, health
- Default 15-minute check intervals
- Migration script for existing installations
- Installation Improvements
