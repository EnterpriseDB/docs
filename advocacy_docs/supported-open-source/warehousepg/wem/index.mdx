WarehousePG Enterprise Manager (WEM) is the centralized management and monitoring interface for WarehousePG (WHPG) clusters. By integrating with Prometheus, Grafana, and Loki, WEM provides real-time visibility into host-level metrics, cluster health, and query performance.
