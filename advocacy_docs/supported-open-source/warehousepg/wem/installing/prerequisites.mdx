!!! Note
This guide assumes you have installed Prometheus and Loki. 
!!!

## Prerequisites

- WarehousePG (WHPG) version 6.x running on RH7 or RH8.
- WarehousePG version 7.x running on RH8 or RH9.
- [Loki](https://grafana.com/docs/loki/latest/setup/install/) 3.5 or later.
- [Prometheus](https://prometheus.io/docs/prometheus/latest/installation/) 3.5.0 or later.
- A database user that holds the superuser role and is able to connect to your WHPG cluster from the WHPG Exporter host.

## Network requirements

The following table lists the connection requirements among the different components. Note that the ports listed are default values, you can customize them according to your environment:

| Source | Destination | Use  
| ------ | ----------- | ---
| WHPG Coordinator | Prometheus:9090 | Push host level metrics
| WHPG Coordinator | Loki:3100       | Push log files
| WEM          | Exporter:9187   | Expose live query endpoint
| WEM          | Prometheus:9090 | Display host level metrics
| WEM          | Loki:3100       | Display log metrics
| End user         | WEM:8080    | Access to WEM console
