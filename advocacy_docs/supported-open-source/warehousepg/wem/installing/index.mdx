The installation process consists of the following steps:

1. Verify the [prerequisites](prerequisites).
1. [Install the observability stack](observability-stack) — Prometheus, Loki, and Alertmanager. Skip this step if you already have these services running.
1. [Install WHPG Collector](collector) on your WarehousePG (WHPG) cluster.
1. [Install and configure WEM](wem) on your dedicated host.

If you're upgrading an existing installation, see [Upgrading](upgrading).
