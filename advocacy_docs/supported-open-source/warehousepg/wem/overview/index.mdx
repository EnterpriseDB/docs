WarehousePG Enterprise Manager (WEM) is the centralized management and monitoring hub for your WarehousePG (WHPG) clusters. It replaces fragmented command-line tools with a unified web interface that combines real-time observability, AI-assisted development, and interactive cluster administration.

Whether you are a DBA tuning performance or a developer writing complex distributed queries, WEM provides the insights and safety nets required to run WarehousePG at scale.

## Why WEM?

Managing a distributed architecture across multiple segment nodes is inherently complex. WEM simplifies this by providing:

- **Consolidated visibility:** Stop jumping between nodes. View host metrics, SQL statistics, and system logs in one place.

- **Built-in intelligence:** Use the AI Assistant to explain execution plans and optimize distributed queries.

- **Proactive health:** Identify data skew, monitor resource bottlenecks, and run automated Canary checks.
