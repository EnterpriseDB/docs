## Supported WarehousePG and platform versions

WarehousePG Enterprise Manager 0.5 is compatible with the following versions of WarehousePG:

- WarehousePG (WHPG) version 6.x running on RHEL 8 or RHEL 9.
- WarehousePG version 7.x running on RHEL 8 or RHEL 9.
