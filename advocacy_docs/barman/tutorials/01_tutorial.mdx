[Barman](/barman) is an open-source administration tool for backup and recovery of PostgreSQL databases. It offers multiple options for data transfer and replication, and is designed to encourage responsible planning and maintenance practices. 

Let's walk through a basic Barman setup, providing streaming backup of a PostgreSQL database.

This is an interactive tutorial - you can run the commands below right in your browser.
