You can configure project-specific settings for profiles, notifications, and alerts. Use the **Settings** menu to access these settings.

-   [Profile](profile.mdx) — Select **Profile** to manage your project's profile, including the project's name.

-   [Notifications](notifications.mdx) — Select **Notifications** to configure how the project’s notifications arrive.

-   Security — Select **Security** to configure project security settings. You can add keys that you can use when enabling TDE for a new cluster. The keys you can add are managed by a key management system.

-   [Alerts](alerts.mdx) — Select **Alerts** to see the alert settings of the project and control how and when they're sent.
