To perform a database migration, you must leverage the HM Console. This section provides reference material to explain the options available under the **Migrate** section of the left navigation in a project.

-   [Sources](sources.mdx): view and manage migration database sources that you have added with the DMS Reader.
-   [Destinations](destinations.mdx): view and manage migration database [destinations] that you have added with the DMS Writer.
-   [Migrations](migrations.mdx): view existing migrations or start new database migrations from existing sources to existing destinations.
-   [Credentials](credentials.mdx): Obtain project-scoped credentials required to connect sources and destinations with the DMS Reader and Writer.
