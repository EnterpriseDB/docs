navTitle: Knowledge Base

---

Knowledge Bases in the Asset Library enables you to manage your collections of AI models that can be used in your Hybrid Manager clusters. 

![Knowledge Bases Page](../../images/assetlibrary-knowledge-bases.png)

The Knowledge Bases page consists of a search and filter bar, a sort control and a table of knowledge bases. Each knowledge base is displayed in its own row with the following information:

-   **Name**: The name of the knowledge base.
-   **Source Data**: The source data that the knowledge base is built from.
-   **Database**: The database that the knowledge base is built from.
-   **Cluster**: The cluster that the knowledge base is built from.
-   **Model**: The AI model that the knowledge base is built from.
