---
title: Deleting clusters
navTitle: Delete clusters

---

You can delete the cluster from the **Postgres Clusters** page and from the detailed view of the cluster.

-   From the project page, select **Clusters > Postgres Clusters**. From the three-dot menu next to the cluster you want to delete, select **Delete Cluster**.

-   From the project page, select **Clusters > Postgres Clusters**. Select the cluster you want to delete. From the detailed view of the cluster, select **Quick Actions > Delete Cluster**.

When you're prompted to confirm the deletion, select **Delete**.
