# System specifications

Sovereign AI and Data Factory is available in two primary configurations, with support for node expansion based on workload needs. All systems are rack-mounted and delivered pre-assembled and pre-validated by EDB and Supermicro.

## Configuration options

| Configuration | Description | Total Nodes | GPU-Enabled |
|---------------|-------------|-------------|-------------|
| Core Compute | Designed for high-performance Postgres and hybrid workloads | 7 | ❌ |
| Advanced AI | Includes all Core Compute nodes plus GPU servers for model serving and agentic workloads | 11 | ✅ |

Customers can scale beyond these base configurations by adding more Compute or AI Nodes.

## Node roles

### Administration Node (1x)

Hosts Hybrid Manager, authentication, and orchestration services.

- **System**: 1U Supermicro SYS-111C-NR
- **CPU**: Intel Xeon Gold 4514Y (16-core)
- **Memory**: 128 GB DDR5 ECC
- **Storage**: 2x 3.2TB NVMe
- **Networking**: 1GbE RJ45 + 25GbE SFP28

### Hybrid Manager Nodes (3x)

Form the high-availability control plane. Manage lifecycle and scheduling.

- Same hardware as Compute Nodes (see below), configured for control-plane use.

### Compute Nodes (3x)

Run Postgres, analytics, and vector workloads.

- **System**: 2U Supermicro SYS-221H-TNR
- **CPU**: 2x Intel Xeon Platinum 6538N (32-core each)
- **Memory**: 1024 GB DDR5 ECC
- **Storage**:
- 2x 960GB M.2 NVMe (OS)
- 4x 15.3TB NVMe (data)
- 2x 1.92TB NVMe (journaling / pg_wal)
- **Networking**: 2x 100GbE QSFP28
- **Power**: 2x 2600W redundant

### 2-GPU AI Nodes (2x, Advanced AI only)

Serve mid-scale inference workloads.

- **System**: 2U Supermicro SYS-221H-TNR
- **CPU**: 2x Intel Xeon Platinum 6538N
- **Memory**: 1024 GB DDR5 ECC
- **GPUs**: 2x NVIDIA Ada L40S (48GB GDDR6 each)
- **Storage**: Same as Compute Nodes
- **Networking**: 2x 100GbE QSFP28

### 8-GPU AI Nodes (2x, Advanced AI only)

Designed for large-scale inference, agentic execution, and LLM hosting.

- **System**: Supermicro 10U SYS-A21GE-NBRT-G1
- **CPU**: Intel Xeon Platinum 8570 (56-core)
- **Memory**: 3072 GB DDR5
