title: Connecting to Hybrid Manager

---

To verify that your installation of the EDB Postgres AI Hybrid Manager platform was successful, connect to Hybrid Manager Console using your browser.

```shell
https://portal.foo.network/
```

In this configuration, the https doesn't validate, as the hostname doesn't exist on the wider internet. Accept the certificate warning to proceed to the portal. Accept the certificate warning to proceed to the portal.

At the login screen, you can enter the default username and password generated with the bootstrap installation: `owner@mycompany.com` and `password`.

After you log in, you can [create a database cluster](../../../..).
