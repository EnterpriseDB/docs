You need a system from which you can run the installation process. This system needs the following software installed:

-   [AWS CLI](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html#getting-started-install-instructions) â€” The AWS Command Line Interface (CLI) to manage your AWS services.
-   [kubectl](https://docs.aws.amazon.com/eks/latest/userguide/install-kubectl.html) to manage your Kubernetes cluster.
-   [yq](https://github.com/mikefarah/yq) to prepare the HCP installation files.
-   [Helm](https://helm.sh/docs/intro/install/) to render and apply the Kubernetes manifest.

You can install these tools using the linked instructions or by using the instructions for your particular operating system that follow.

## macOS

Install the AWS CLI, kubectl, yq, and Helm:

```shell
brew install awscli kubectl yq helm
```

## Linux

=== "Ubuntu"

````
!!! Note
    These instructions are for Ubuntu 16.04 or newer, as they assume Snap support is available by default.

```shell
sudo snap install aws-cli --classic
sudo snap install helm --classic
sudo snap install kubectl --classic
sudo snap install yq

```
````

=== "Red Hat"

````
Install the AWS CLI:

```shell
sudo yum install awscli
```

Install kubectl:

```
curl -LO "https://dl.k8s.io/release/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl"
chmod +x kubectl
sudo mv kubectl /usr/local/bin/
kubectl version --client
```

Install Helm:

```
curl -LO https://get.helm.sh/helm-v3.12.3-linux-amd64.tar.gz
tar -xvf helm-v3.12.3-linux-amd64.tar.gz
sudo mv linux-amd64/helm /usr/local/bin/helm
helm version
```            

Install yq:

```
sudo curl -L "https://github.com/mikefarah/yq/releases/latest/download/yq_linux_amd64" -o /usr/bin/yq
sudo chmod +x /usr/bin/yq
yq --version
```
````

=== "Debian"

````
```shell
sudo apt update
sudo apt install snapd
sudo snap install snapd
sudo snap install aws-cli --classic
sudo snap install kubectl --classic
sudo snap install yq
```
````

#### Installing Helm

1.  Download the Helm binary for Windows. On the Helm [GitHub Releases page](https://github.com/helm/helm/releases), locate the latest release and download the helm-v3.x.x-windows-amd64.zip file.

2.  Extract the Zip file. Use a tool like 7-Zip, WinRAR, or the built-in Windows extraction utility to unzip the downloaded file.

3.  Copy `helm.exe` from the extracted folder to a directory in your path. Common directories include `C:\Program Files\`, `C:\Program Files (x86)\`, or you can create a directory, such as `C:\Helm\`, and add it to your path.

4.  Verify the installation:

    1.  Open a new Command Prompt or PowerShell window.

    2.  Run the following command to verify that Helm is installed correctly:

        ````
        ```bash
        helm version
        ```
        ````

    3.  The output displays Helm version information.
