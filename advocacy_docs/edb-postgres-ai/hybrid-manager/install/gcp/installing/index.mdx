There are two methods you can use to install Hybrid Manager on GCP GKE. Select the method that fits your use case:

- Using [Helm](installhelm.mdx). Use a Helm install for testing and experimenting with Hybrid Manager. A Helm install is suitable for nonproduction use cases.

- Using [EDB Postgres AI Operator](installop.mdx). Use the EDB Postgres AI Operator install when you are ready to use Hybrid Manager in production.
