AWS EKS has prerequisites to meet before you can install the EDB Postgres AI Software Deployment. Fulfill the prerequisites to ensure that you have everything you need before you start the installation process.

## Prerequisites

-   [A system from which you can run the installation process](installsystem.mdx)
-   [An AWS account with the necessary permissions](awsaccount.mdx)
-   [An EKS cluster configured to host HCP](ekscluster.mdx)

## Next steps

After you complete the prerequisites, you can move on to [installing the EDB Postgres AI Software Deployment on AWS EKS](../installing.mdx).
