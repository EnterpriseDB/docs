---
title: Install system

---

You need a system from which you can run the installation process. This system needs the following software installed:

-   [OpenShift CLI (oc)](https://docs.redhat.com/en/documentation/openshift_container_platform/4.17/html/cli_tools/openshift-cli-oc#cli-getting-started)
-   [kubectl](https://kubernetes.io/docs/reference/kubectl/) to manage your Kubernetes cluster
-   [yq](https://github.com/mikefarah/yq) to prepare the HCP installation files

You can install these tools using the linked instructions or by using the instructions for your operating system that follow.

## macOS

```shell
brew install kubectl yq
```

## Linux

<TabContainer>
<Tab title="Ubuntu">

```shell
sudo apt install kubectl
sudo snap install yq
```

</Tab>

<Tab title="Red Hat">

```shell
sudo yum install kubectl
sudo wget https://github.com/mikefarah/yq/releases/latest/download/yq_linux_amd64 -O /usr/bin/yq && sudo chmod +x /usr/bin/yq
```

</Tab>

<Tab title="Debian">

```shell
sudo apt-get install kubectl
sudo wget https://github.com/mikefarah/yq/releases/latest/download/yq_linux_amd64 -O /usr/bin/yq && sudo chmod +x /usr/bin/yq
```

</Tab>
</TabContainer>

## Windows

<TabContainer>
<Tab title="Winget">

```shell
winget install kubectl --source winget
winget install yq --source winget
```

</Tab>

<Tab title="Manually">

1.  Install [kubectl](https://kubernetes.io/docs/tasks/tools/install-kubectl-windows/).
2.  Install [yq](https://github.com/mikefarah/yq?tab=readme-ov-file#windows).

</Tab>
</TabContainer>
