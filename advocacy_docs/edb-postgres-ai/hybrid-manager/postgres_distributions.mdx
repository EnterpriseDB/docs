# PostgreSQL

PostgreSQL offers all the features gathered in the [PostgreSQL documentation](https://www.postgresql.org/docs/current/).

# EDB Postgres Extended Server

EDB Postgres Extended Server adds a number of features to PostgreSQL to support [transparent data encryption](/tde/latest).

EDB Postgres Extended Server can also be combined with and fully utilize EDB's powerful high-availability and multi-master replication solution, [EDB Postgres Distributed](/pgd/latest).

# EDB Postgres Advanced Server

EDB Postgres Advanced Server builds on the features of EDB Postgres Extended Server and supports features such as [advanced security](/epas/latest/epas_security_guide), [advanced SQL](/epas/latest/application_programming/15_enhanced_sql_and_other_misc_features/) and, optional, [Oracle compatibility](/epas/latest/working_with_oracle_data).
