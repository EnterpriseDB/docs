title: Migrations
navTitle: Migrations
description: Displaying the status of data migrations for an EDB Postgres AI project
---

The **Migrations** page displays the status of all ongoing and completed data migrations for an EDB Postgres AI project. The page shows the following information:

- **Migration Name**: The name of the migration.
- **Status**: The status of the migration, Starting/Failed/Completed.
- **Source**: The source of the migration. 
- **Destination**: The destination of the migration.

To the right of each migration, you can see the trash can button. Clicking this button deletes the migration.
