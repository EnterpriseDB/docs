---
title: Managing cloud providers
navTitle: Cloud Providers
description: How to manage cloud providers in an EDB Postgres AI project
---

The **Cloud Providers** page displays all the cloud providers configured in the project. By default you can enable, connect, and display the Cloud Service hosting providers - Google Cloud, Microsoft Azure, and AWS. 

Selecting the other **Your Cloud Account** tab displays any cloud providers you have added and connected to your EDB Postgres AI project. Also shown on this tab are instructions on how to connect a new cloud provider.
