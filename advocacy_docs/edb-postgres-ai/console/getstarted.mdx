Getting Started with the EDB Postgres AI Console.

_The EDB Postgres AI Console is a web-based user interface that provides a single pane of glass for managing and monitoring EDB Postgres AI Database Cloud Service and EDB Postgres AI Databases. The EDB Postgres AI Console provides a unified view of the EDB Postgres AI Database Cloud Service and EDB Postgres AI Databases, allowing users to manage and monitor their databases, users, and resources from a single interface._
