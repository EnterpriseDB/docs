HCP Lakehouse is a powerful tool for running analytical queries on your data.
It's built on top of Postgres, so you can use your existing Postgres knowledge and tools.

To get the best experience with Lakehouse, follow the [quick start](../quick_start.mdx) to query benchmarking data.
Then you can try loading your own data and use [external tables](../external_tables.mdx).

This section details some of the important things to know:

-   [Benchmarking datasets](datasets.mdx)
-   [Queries in Lakehouse](queries.mdx)
-   [Delta Table tools](delta_tables.mdx)
-   [DirectScan and fallback modes](directscan.mdx)
-   [Users](users.mdx)
-   [Loading data](loadingdata.mdx)
-   [PGAA functions](pgaa_functions.mdx)

For information on the PGFS functions for managing storage location, see the [AI Accelerator PGFS documentation](https://www.enterprisedb.com/docs/edb-postgres-ai/ai-accelerator/pgfs/).
