The EDB Postgres AI analytics products extend Postgres with high-performance analytical capabilities, from vectorized query execution over open table formats in object storage to massively parallel processing (MPP) for large-scale data warehousing workloads.

## Postgres Analytics Accelerator (PGAA)

[Postgres Analytics Accelerator (PGAA)](/pgaa/latest) is a high-performance Postgres extension that enables querying large-scale data stored in open table formats like Delta Lake, Apache Iceberg, and Parquet directly from Postgres.

By offloading heavy analytical queries to a vectorized execution engine, PGAA bridges the gap between operational databases and data lakes without requiring data movement.

Key capabilities:

- **[Read tables in object storage](/pgaa/latest/read_no_catalog)**: Query Parquet, Delta Lake, and Iceberg files in S3, GCS, or Azure using standard SQL.
- **[Iceberg catalog integration](/pgaa/latest/catalog)**: Connect to external Iceberg REST catalogs to manage table metadata.
- **[Write to object storage](/pgaa/latest/ctas)**: Write Postgres data out to optimized lakehouse formats in your object store using CTAS.
- **[Spark acceleration](/pgaa/latest/spark)**: Offload massive datasets and complex joins to a remote Spark cluster, with optional GPU acceleration via NVIDIA RAPIDS.
- **[Replicate with PGD](/pgaa/latest/pgd)**: Combine with EDB Postgres Distributed for automated partition offloading and real-time replication.

## WarehousePG

[WarehousePG](/supported-open-source/warehousepg/) is an open-source massively parallel processing (MPP) database built on Postgres, designed for large-scale analytical and data warehousing workloads. 

Key capabilities:

- **[Massively parallel processing](/warehousepg/latest)**: Distribute query execution across multiple segments for high-throughput analytical queries.
- **[Support for Greenplum migrations](/supported-open-source/warehousepg/edbpggpsupp)**: Tools and guidance for migrating existing Greenplum workloads to WarehousePG.
- **[WarehousePG Enterprise Manager (WEM)](/supported-open-source/warehousepg/wem)**: Monitor, manage, and optimize your WarehousePG cluster.
- **[FlowServer](/supported-open-source/warehousepg/flowserver)**: High-speed data loading into WarehousePG.
- **[whpg-copy](/supported-open-source/warehousepg/whpg-copy)**: A utility for efficient bulk data copy operations.
