## Postgres options 

Building on decades of Postgres expertise, the EDB Postgres® AI databases are the core of the EDB Postgres AI platform. 

-   [Enterprise Postgres (Oracle Compatible)](/epas/latest) - Also known as EDB Postgres Advanced Server can take on Oracle workloads.
-   [Enterprise Postgres](/pge/latest) - Also known as EDB Postgres Extended Server is designed for large-scale, mission-critical enterprise workloads.

All of them are built on the same Postgres core, so you can use the same tools and skills across all of them.

- [PostgreSQL](/supported-open-source/postgresql) is the world's most advanced open source database, and the foundation for all EDB Postgres databases.

And for distributed high-availability and fault tolerance for mission-critical workloads, you can use Enterprise Postgres (Oracle Compatible) and Enterprise Postgres in combination with EDB Postgres Distributed: 

-   [EDB Postgres Distributed](/pgd/latest)
