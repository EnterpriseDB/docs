## EDB Tools

You can deploy all Postgres distributions (PostgreSQL, EDB Postgres Advanced Server, and EDB Postgres Extended Server) with the following EDB Tools.

- [EDB CloudNativePG Global Cluster](/postgres_distributed_for_kubernetes/latest/)
- [EDB CloudNativePG Cluster](/postgres_for_kubernetes/latest/)

## EDB extensions

EDB also supports a wide range of EDB, and open-source extensions. See [Postgres extensions available by deployment](/pg_extensions) for compatibility information.

To know which plan to get based on your Postgres distribution requirements, see [EDB Subscription Plans Comparison](https://www.enterprisedb.com/products/plans-comparison).
