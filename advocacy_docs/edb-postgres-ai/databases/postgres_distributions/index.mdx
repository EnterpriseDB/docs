EDB builds upon the upstream open-source [PostgreSQL](https://www.postgresql.org/) project, enhancing it with additional capabilities designed to support a wider range of use cases and enterprise requirements.

<br/>

![Postgres distributions comparison by features](/edb-postgres-ai/databases/images/postgres_distros_transparent_500w.svg)


## Postgres distributions

### PostgreSQL

PostgreSQL offers all the features gathered in the [PostgreSQL documentation](https://www.postgresql.org/docs/current/).

You can obtain PostgreSQL from our [EDB repository](https://www.enterprisedb.com/software-downloads-postgres), for which you require an EDB account and license. Alternatively, you can download it from the [official PostgreSQL site](https://www.postgresql.org/download/).

### EDB Postgres Extended Server 

EDB Postgres Extended Server adds a number of features to PostgreSQL to support [transparent data encryption](/tde/latest).

EDB Postgres Extended Server can also be combined with and fully utilize EDB's powerful high-availability and multi-master replication solution, [EDB Postgres Distributed](/pgd/latest).

You can obtain EDB Postgres Extended Server from our [EDB repository](https://www.enterprisedb.com/software-downloads-postgres), for which you require an EDB account and license.

### EDB Postgres Advanced Server 

Additionally to all the features included in PostgreSQL and EDB Postgres Extended Server, EDB Postgres Advanced Server also adds a number of features to support [advanced security](/epas/latest/epas_security_guide), [advanced SQL](/epas/latest/application_programming/15_enhanced_sql_and_other_misc_features/) and, if required, [Oracle compatibility](/epas/latest/tools_utilities_and_components).

You can obtain EDB Postgres Advanced Server from our [EDB repository](https://www.enterprisedb.com/software-downloads-postgres), for which you require an EDB account and license.
