In [Choosing your postgres distribution](/edb-postgres-ai/databases/postgres_distributions), you have learned about the different Postgres distributions available to you, and how they differ.

This detailed view displays compatibility per Postgres distribution.

|                             | PostgreSQL | EDB Postgres Extended Server | EDB Postgres Advanced Server | EDB Postgres Advanced Server in Oracle compatibility mode |
|-----------------------------|------------|------------------------------|------------------------------|-----------------------------------------------------------|
| SQL Compatibility           | PostgreSQL | PostgreSQL                   | PostgreSQL and more          | Mix (PostgreSQL and Oracle)                               |
| Binary compatibility        | Yes        | No                           | No                           | No                                                        |
| Advanced PGD features       | No         | Yes                          | Yes                          | Yes                                                       |
| Transparent Data Encryption | No         | Yes                          | Yes                          | Yes                                                       |
| Advanced Security           | No         | No                           | Yes                          | Yes                                                       |
| Advanced SQL                | No         | No                           | Yes                          | Yes                                                       |
| Advanced performance        | No         | No                           | Yes                          | Yes                                                       |
| Resource Manager            | No         | No                           | Yes                          | Yes                                                       |
| Bulk Data Loader            | No         | No                           | Yes                          | Yes                                                       |
| Oracle Compatibility        | No         | No                           | No                           | Yes                                                       |

To know which plan to get based on your Postgres distribution requirements, see [EDB Subscription Plans Comparison](https://www.enterprisedb.com/products/plans-comparison).
