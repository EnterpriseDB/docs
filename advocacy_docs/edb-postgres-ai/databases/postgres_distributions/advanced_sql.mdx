SQL querying is only supported in EDB Postgres Advanced server deployments.

|                         | PostgreSQL | EDB Postgres Extended Server | EDB Postgres Advanced Server | EDB Postgres Advanced Server in Oracle compatibility mode |
|-------------------------|------------|------------------------------|------------------------------|-----------------------------------------------------------|
| Oracle style packages   | No         | No                           | Yes                          | Yes                                                       |
| Autonomous transactions | No         | No                           | Yes                          | Yes                                                       |
| Interval partitioning   | No         | No                           | Yes                          | Yes                                                       |
| AUTOMATIC partitioning  | No         | No                           | Yes                          | Yes                                                       |
| Optimizer hints         | No         | No                           | Yes                          | Yes                                                       |
| COLLECTIONS             | No         | No                           | Yes                          | Yes                                                       |
| Hierarchical queries    | No         | No                           | Yes                          | Yes                                                       |
