Building on decades of Postgres expertise, EDB provides a range of Postgres distributions to help you manage your databases effectively. EDB builds upon the upstream open-source [PostgreSQL](https://www.postgresql.org/) project, enhancing it with additional capabilities designed to support a wider range of use cases and enterprise requirements.

These are the main differences between community PostgreSQL, EDB Postgres Extended Server and EDB Postgres Advanced Server.

## Postgres distributions

### PostgreSQL

PostgreSQL offers all the features gathered in [PostgreSQL documentation](https://www.postgresql.org/docs/current/). 

You can obtain PostgreSQL from the [official PostgreSQL site](https://www.postgresql.org/download/), or from our [EDB repository](https://www.enterprisedb.com/software-downloads-postgres), for which you require an EDB account.

### EDB Postgres Extended Server 

Additionally to all the features PostgreSQL offers, EDB Postgres Extended Server also adds a number of features to support [transparent data encryption](../../../product_docs/docs/tde/index).

Advanced Replication is also available for EDB Postgres Extended Server with [EDB Postgres Distributed](../../../product_docs/docs/pgd/index).

You can obtain EDB Postgres Extended Server from our [EDB repository](https://www.enterprisedb.com/software-downloads-postgres) for which you require an EDB account and license.

### EDB Postgres Advanced Server 

Additionally to all the features PostgreSQL and EDB Postgres Extended Server offer, EDB Postgres Advanced Server also adds a number of features to support [Advanced Security](../../../product_docs/docs/epas/latest/epas_security_guide), [Advanced SQL](../../../product_docs/docs/epas/latest/application_programming/15_enhanced_sql_and_other_misc_features/) and, if required, [Oracle compatibility](../../../product_docs/docs/epas/latest/tools_utilities_and_components).

You can obtain EDB Postgres Advanced Server from our [EDB repository](https://www.enterprisedb.com/software-downloads-postgres) for which you require an EDB account and license.

Here is a more detailed overview of the supported features:

|-----------------------------|------------|------------------------------|------------------------------|-----------------------------------------------------------|
| SQL Compatibility           | PostgreSQL | PostgreSQL                   | PostgreSQL and more          | Mix (PostgreSQL and Oracle)                               |
| Binary compatibility        | Yes        | No                           | No                           | No                                                        |
| Advanced PGD features       | No         | Yes                          | Yes                          | Yes                                                       |
| Transparent Data Encryption | No         | Yes                          | Yes                          | Yes                                                       |
| Advanced Security           | No         | No                           | Yes                          | Yes                                                       |
| Advanced SQL                | No         | No                           | Yes                          | Yes                                                       |
| Advanced performance        | No         | No                           | Yes                          | Yes                                                       |
| Resource Manager            | No         | No                           | Yes                          | Yes                                                       |
| Bulk Data Loader            | No         | No                           | Yes                          | Yes                                                       |
| Oracle Compatibility        | No         | No                           | No                           | Yes                                                       |
