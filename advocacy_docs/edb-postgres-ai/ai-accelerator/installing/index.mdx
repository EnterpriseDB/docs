Pipelines is delivered as a set of extensions. Depending on how you are deploying Pipelines, these extensions may be installed automatically or you may need to install them manually.

- [Manually installing pipelines packages](packages)

Once the packages are installed, you can [complete the installation](complete) by activating the extensions within Postgres.
